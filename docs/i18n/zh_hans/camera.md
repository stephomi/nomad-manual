# ![](/icons/camera.webp) 相机

此菜单用于创建和修改相机，以及控制你与相机的交互方式。

![](/images/camera_overview2.webp)

Nomad 中的相机有多种用途：

* 为雕刻设置精确角度的视图
* 像照相机一样用于给物体构图
* 作为第一人称视角相机在场景中导航
* 作为正交相机用于等距游戏或工业风格渲染

## 控制相机

### 旋转
在背景上拖动*一根*手指可以旋转相机。  
如果在模型上拖动手指，则会开始雕刻操作。

::: tip 如果我点不到背景，还能旋转相机吗？
可以，你可以先用*两根*手指放在屏幕上——就像要开始平移/缩放手势一样——然后松开*一根*手指。
:::

### 聚焦 / 重置
在模型上*双击*可以将焦点对准所点位置。  
如果在背景上*双击*，相机会对准当前选中的网格。

### 平移
用*两根*手指移动，可以平移相机。

### 缩放
使用捏合手势可以放大/缩小视图。

### 旋转视图（滚动）
用*两根*手指旋转可以*滚动*视图。
::: warning
此手势仅在 `trackball`（轨迹球）旋转模式下可用。
:::

### 桌面端控制

在桌面端，使用 Alt/Opt 键来控制相机：

* tip 在空白处拖动 = 旋转相机
* alt+tip 拖动 = 平移
* alt+tip 拖动，然后松开 alt = 缩放（与 ZBrush 相同）

对于带有 2 个或更多按键的 Wacom 手写笔，可在 Wacom 设置中将笔尖和按键配置为：

* tip = 左键单击
* 下摇杆键 = 中键单击
* 上摇杆键 = 右键单击

使用上述设置，你可以只用手写笔来操作相机：

* 上摇杆键 + 悬停移动 = 旋转相机
* 下摇杆键 + 悬停移动 = 平移

## 相机列表与控制

![](/images/camera_list.webp)

### 视图
你可以通过 `Add View` 保存当前相机视角。  
点击视图名称时，相机会恢复到该视角。

::: tip
保存的视图会同时保存[投影类型](#projection-type)设置以及[参考图像](background.md)。  
如果你想在不同背景的前/左/后参考视图之间切换，这会非常有用。
:::

|   Action   |              Icon              | Description                                                                  |
| :--------: | :----------------------------: | :--------------------------------------------------------------------------: |
| Visibility | ![](/icons/eye_open.webp)    | 切换相机可见性。被隐藏的相机不会出现在上一视图/下一视图的切换中 |
|    Name    |                                | 选择该相机                                                                   |
|   Image    | ![](/icons/image.webp)       | 如果相机关联了参考图像，这里会显示其缩略图                                  |

### ![](/icons/tool_view.webp) Add View
基于当前视角创建一个新相机。

### ![](/icons/camera.webp) Icons

切换视口中是否显示相机图标。若某个相机被选中，其图标始终可见。

### Projection Type
你可以更改相机的 `Field of View`（视野 / 焦距）。  
通常建议在雕刻时使用较小的视野，有助于把握比例。  
你也可以使用 `Orthographic`（正交）模式，它大致相当于视野为 0 的效果。

### First Person
启用后，枢轴点会直接设置在相机上，而不是雕塑上。  
在背景上拖动手指时，相机位置会保持不变，只改变旋转方向，类似第一人称游戏的操作方式。  
在雕刻环境（场景）而非单个物体时非常有用。

![](/images/camera_rotation_ortho_view.webp)

### Rotation Type
默认情况下，相机使用 `Turntable`（转盘）旋转模式。  
这意味着你只有两个自由度，操作更直观，但在某些情况下你可能需要更高的灵活性。  
你可以切换到 `Trackball`（轨迹球）模式，此时可以通过在视口中用*两根*手指旋转来*滚动*视图。  
在桌面端还有一种替代的轨迹球模式，对部分用户来说可能更熟悉。

### Orthographic snap

启用后，如果你有键盘，在旋转视图时按住 Shift，会将相机吸附到最近的前/后/上/下/左/右视图，并切换为正交相机。  
当点击视图立方体以吸附到前/后/左/右/上/下视图时，相机同样会切换为正交模式。

### Reset view

将相机移动到正前方，并将整个场景适配到视图中。

### Snap view
将视图吸附到最近的前/后/左/右/上/下视图。  
如果你已经处于其中一个视图，再次点击会旋转 180 度吸附到相对的一侧。

### Speed

如果你觉得相机移动太慢或太快，可以为 `rotation`（旋转）、`translation`（平移）和 `zooming`（缩放）设置速度倍率。  
在雕塑非常大或非常小时尤其有用。

### 枢轴点概览

当你旋转相机时，可以看到一个小的粉色点，那就是相机的枢轴点。  
理解枢轴点的位置非常重要，否则你可能会在操作相机时迷失方向或感到沮丧。

默认情况下，枢轴点会通过以下操作更新：
- 在模型上双击
- 在背景上双击（新的枢轴点会位于网格中心）
- 放上*两根*手指（平移/缩放/滚动）时，会将枢轴点更新到*两根*手指中心位置

### Update Pivot...

你可以进一步自定义在以下情况下更新枢轴点：

* When camera starts moving（当相机开始移动时）
* Allow air pivot（允许“空中”枢轴点）
* When double tapping（双击时）
* Sculpt (stroke)（雕刻笔划时）

::: tip
熟悉之后，如果你不再需要提示，可以在[设置](settings.md)菜单中隐藏这个粉色点提示。
:::

### Double tap on object
当启用 `Focus` 时，在物体上双击会将枢轴点移动到所点物体上。

### Double tap on background
启用后，在背景上双击会将枢轴点设置为以下之一：Selection（选中对象）、Scene（整个场景），或在两者之间切换。
