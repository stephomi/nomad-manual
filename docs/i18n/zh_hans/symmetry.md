# ![](/icons/symmetry.webp) 对称

此菜单控制笔划如何在镜像平面或径向上重复，以及如何在非对称对象上恢复对称。

![](/images/symmetry_overview.webp) 

## 概览
你可以通过多种方式使用对称：

* 作为镜像，将工作在 X（左/右）、Y（上/下）、Z（后/前）或它们的组合之间翻转。  
* 在 X、Y、Z 轴上设置径向对称和重复次数，例如雕刻海星。  
* 镜像可以围绕原点（称为世界模式）或围绕对象中心（称为本地模式）运作。  
* 从非对称开始的雕刻可以被强制变为对称。

启用/禁用对称的快捷方式也可以在左侧快速面板中找到（“*Sym*”）。小的 “L/W” 表示 Nomad 当前处于本地还是世界对称模式。你也可以长按或向屏幕中央滑动以调出菜单。

![](/images/symmetry_button.webp) 

这是一个全局选项，因此其状态会在不同工具之间保持一致。  
唯一的例外是变换工具（[移动](#translate)、[旋转](#rotate)、[缩放](#scale) 和 [Gizmo](#gizmo)），它们有各自独立的对称状态。

::: tip
对称菜单主要用于控制笔划对称。你也可以通过[场景菜单中的重复器](scene#repeaters)来镜像和重复对象。
:::

## 启用
切换镜像模式，这与左侧快速面板中的 `Sym` 按钮相同。

## 平面

启用对称平面，以及径向对称的重复次数。注意你不必只选择单一平面，你可以启用 1、2 或 3 个平面来实现复杂对称。

径向对称的轴和重复次数。同样地，它们也不局限于单一轴，并且可以与平面对称组合使用，从而非常快速地生成细致的结果。（总重复次数上限为 150）

![](/videos/symmetry_demo.mp4) 

## 模式
镜像可以是“本地”的，随对象一起移动；也可以是“世界”的，不随对象移动。  
如果不确定需要哪种模式，可以观察叠加在对象上的镜像平面和径向指示器。在本地模式下，如果你使用变换 gizmo 移动模型，镜像指示器也会一起移动。在世界模式下，镜像指示器保持固定，对象会从中滑过。

## 镜像处理
![](/images/symmetry_mirroring.webp)

在靠近对称平面雕刻时，一些笔刷的对称行为可能并不完美。本节允许你通过将雕刻的一侧复制到另一侧来恢复对称。

`Direction` - `<<` 和 `>>` 按钮决定哪一侧会被复制到另一侧。Nomad 会根据你当前的视图来计算方向，例如设置为 `<<` 时，总是从屏幕右侧复制到屏幕左侧。

![](/icons/shield.webp) `Mask` - 遮罩按钮可以让你限定会被镜像的区域；遮罩目标侧会保护该区域不被覆盖，遮罩源侧则会阻止该区域被镜像到目标侧。

![](/icons/tool_hide.webp) `Hide` - 启用时，源侧中被隐藏的区域不会被镜像到目标侧。

`Mirror` 会尝试识别镜像两侧的拓扑是否相同，如果相同，则只移动顶点。这是更常见的情况。

`Split & Mirror` 会沿镜像平面切割对象，复制一侧，将其镜像到另一侧，并在镜像平面上焊接顶点。这是一种更具破坏性的操作，会删除多重细分，但在模型两侧差异很大时有时是必须的。

### 翻转对象
![](/images/symmetry_flip.webp)
将左侧变为右侧，右侧变为左侧。效果类似于使用 gizmo 工具并将 X 轴缩放设置为 -1。

## 重置与编辑

![](/images/symmetry_edit.webp)

可以编辑对称的位置和方向（但不推荐！）。如有需要，`World center` 和 `Orientation` 会将对称的位置和方向重置为默认值。

Nomad 通常知道应该把对称平面放在哪里。不建议手动调整，但 `Gizmo (Edit)` 按钮为高级用户提供了这种能力。点击该按钮后，会显示一个 gizmo，用于平移和旋转对称平面。如果你想恢复默认中心和方向，可以使用 `object center` 和 `orientation` 按钮。

这些选项的行为会根据你当前所在的空间（*Local/World*）而变化。  
如果结果与预期不符，请先确认你处于正确的空间中。

::: tip
`Gizmo (Edit)` 按钮被刻意显示为灰色，以提醒你大概率不应该使用它！
:::

## 显示选项
![](/images/symmetry_show.webp)


`Show line` 和 `Show plane` 会切换视口中对称位置的叠加显示。注意关闭这些选项只会在关闭对称菜单后才会生效。
