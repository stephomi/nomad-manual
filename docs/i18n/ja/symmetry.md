# ![](/icons/symmetry.webp) 対称

このメニューでは、ストロークをミラープレーンや放射状に繰り返す方法、および非対称なオブジェクトに対称性を復元する方法を制御します。

![](/images/symmetry_overview.webp) 

## 概要
対称は次のように利用できます:

* ミラーとして使用し、X（左右）、Y（上下）、Z（前後）、またはそれらの組み合わせで作業内容を反転させる。  
* X/Y/Z 軸ごとに繰り返し数を指定した放射状対称（例: ヒトデのような形状のスカルプト）。  
* ミラーを原点（ワールドモード）またはオブジェクトの中心（ローカルモード）を基準に動作させる。  
* 非対称な状態から始まったスカルプトを強制的に対称にする。  

対称のオン/オフ切り替えショートカットは、左側クイックパネルの「*Sym*」にもあります。小さな「L/W」は、Nomad がローカル対称モードかワールド対称モードかを示します。長押し、または画面中央へスワイプすることでメニューを呼び出すこともできます。

![](/images/symmetry_button.webp) 

これはグローバルオプションなので、状態はツールをまたいで引き継がれます。  
例外はトランスフォーム系ツール（[Move](#translate)、[Rotate](#rotate)、[Scale](#scale)、[Gizmo](#gizmo)）で、これらは独自の対称状態を持ちます。

::: tip
対称メニューは主にストロークの対称を制御するためのものです。オブジェクト自体のミラーや繰り返しは、[シーンメニュー内のリピーター](scene#repeaters)から行うこともできます。
:::

## Enabled
ミラーモードのオン/オフを切り替えます。これは左クイックパネルの `Sym` ボタンと同じです。

## Planes

対称の平面を有効化し、放射状対称の繰り返し数を設定します。単一の平面に限定される必要はなく、1・2・3 軸すべてを有効にして複雑な対称を作ることもできます。

放射状対称の軸と繰り返し数を指定します。これも単一軸に制限されず、平面対称と組み合わせて使用することもできるため、非常に短時間でディテールのある結果を生成できます。（総繰り返し数は 150 までに制限されています）

![](/videos/symmetry_demo.mp4) 

## Method
ミラーは「Local」にしてオブジェクトと一緒に動かすか、「World」にして固定するかを選べます。どちらが必要か分からない場合は、オブジェクト上にオーバーレイ表示されるミラープレーンと放射状インジケータを確認してください。ローカルモードでは、トランスフォームギズモでモデルを移動するとミラーインジケータも一緒に動きます。ワールドモードではミラーインジケータは固定され、オブジェクトの方がその中をスライドします。

## Mirroring
![](/images/symmetry_mirroring.webp)

対称平面付近をスカルプトする際、一部のブラシは完全には対称にならない場合があります。このセクションでは、スカルプトの片側をもう一方にコピーすることで対称性を復元できます。

`Direction` - `<<` と `>>` ボタンで、どちら側をどちらにコピーするかを決定します。Nomad は現在のビューポートからこれを計算するため、例えば `<<` に設定すると、常に画面右側から画面左側へコピーされます。

![](/icons/shield.webp) `Mask` - マスクボタンを使うと、どの部分をミラーするかを限定できます。コピー先側をマスクするとその領域は保護され、コピー元側をマスクするとその領域はコピー先にミラーされなくなります。

![](/icons/tool_hide.webp) `Hide` - 有効な場合、コピー元側で非表示になっている領域はコピー先にミラーされません。

`Mirror` は、ミラーの両側でトポロジが同一かどうかを判定し、同一であれば頂点を移動するだけで処理します。こちらがより一般的なシナリオです。

`Split & Mirror` は、オブジェクトをミラー位置で切断し、片側をコピーして反転し、ミラー上で頂点を結合します。より破壊的なオプションであり、マルチレゾリューションは削除されますが、ミラーを挟んでモデルが大きく異なる場合にはこの方法が必要になることがあります。

### Flip object
![](/images/symmetry_flip.webp)
左側を右側に、右側を左側に入れ替えます。Gizmo ツールメニューで X スケールを -1 に設定した場合と見た目が似ています。

## Reset and Edit

![](/images/symmetry_edit.webp)

対称位置と向きを編集することも可能です（ただし推奨はされません）。必要であれば、`World center` と `Orientation` によって対称位置と向きをデフォルト値にリセットできます。

Nomad は通常、対称平面をどこに置くべきかを把握しています。そのため手動調整は推奨されませんが、上級ユーザー向けに `Gizmo (Edit)` ボタンで編集を可能にしています。このボタンをクリックするとギズモが表示され、対称平面を移動・回転できます。デフォルトの中心と向きに戻したい場合は、`object center` と `orientation` ボタンで復元できます。

これらのオプションの挙動は、現在どの空間（*Local/World*）にいるかによって変化します。  
期待通りに動作しない場合は、正しい空間にいるかどうかを確認してください。

::: tip
`Gizmo (Edit)` ボタンが意図的にグレーアウトされているのは、「基本的には使うべきではない」という注意喚起のためです！
:::

## Show options
![](/images/symmetry_show.webp)


`Show line` と `Show plane` は、対称位置のビューポートオーバーレイ表示を切り替えます。これらのオプションをオフにしても、対称メニューを閉じるまでは反映されない点に注意してください。
