# ![](/icons/pencil.webp) ストローク {#stroke}

---
![](/images/stroke_overview.webp) 

## 概要 {#overview}

ほとんどのツールブラシでは、ストロークの挙動をカスタマイズできます。
設定は 2D ペイントアプリケーションにあるものと似ていますが、いくつかのオプションはスカルプトや 3D に特有のものです。

オプションは 5 つのサブメニューに分かれています:

| Name     | Icon                         | Description                                                        |
| :------: | :--------------------------: | :----------------------------------------------------------------: |
| Stroke   | ![](/icons/stroke_dot.webp) | モデルにストロークがどのように適用されるかを制御します            |
| Alpha    | ![](/icons/alpha.webp)      | グレースケールテクスチャをブラシスタンプとしてどのように使うか    |
| Falloff  | ![](/icons/falloff.webp)    | ブラシ強度が中心から端に向かってどのように変化するかを制御します  |
| Filter   | ![](/icons/filter.webp)     | ブラシがモデルのジオメトリにどのような影響を受けるか              |
| Pressure | ![](/icons/pressure.webp)   | スタイラスの筆圧応答を制御します                                  |

::: tip
すべてのストロークオプションがすべてのツールに適用されるわけではありません。現在のツールで使用されないストロークオプションは無効化されるか非表示になります。 
:::


## ストローク {#stroke-1}

### 半径 {#radius}

![](/images/stroke_radius.webp)

#### 半径を共有 {#share-radius}

有効にすると、すべてのツールが同じ半径を使用します。デフォルトでは、各ツールが独自の半径を持ちます。

#### サイズ {#size}

* Screen - 半径はスクリーン単位で設定されます。半径を 100 ピクセル幅にした場合、ズームイン・ズームアウトに関係なく 100 ピクセル幅のままです。
* Constant (3d) - 半径は 3D 単位で設定されます。例えば球体を作成し、その球体と同じサイズの半径にした場合、ズームイン・ズームアウトしても半径は球体と同じサイズのままです。


### ストローク {#stroke-type}

![](/images/stroke_strokemode.webp)

ストロークには複数のモードがあります:

### ![](/icons/stroke_dot.webp) ドット {#dot}
通常のペイントストロークのようにドラッグします。 
![](/videos/stroke_dot.mp4) 

### ![](/icons/stroke_roll.webp) ロール {#roll}
ブラシアルファがストロークの方向に合わせて回転します。布のステッチを作るのに便利です。 
![](/videos/stroke_roll.mp4) 

 ### ![](/icons/radius.webp) Lock + radius
固定された**_高さ_**でブラシストロークをスタンプします。ドラッグでスケールと回転を設定します。
![](/videos/stroke_lock_radius.mp4) 

### ![](/icons/falloff.webp) ロック + 強度 {#lock-intensity}
固定された**_半径_**でブラシストロークをスタンプします。ドラッグで高さと回転を設定します。
![](/videos/stroke_lock_intensity.mp4) 

![](/images/stroke_strokemodemove.webp)

`Move` と `Drag` ツールには独自の 3 つのオプションがあります:

### ![](/icons/snake.webp) ドラッグ {#drag}

ストローク中、ブラシ半径内の内容が継続的に更新されます。速いストロークでは表面が後ろに残り、遅いストロークでは素材を保持してより長い形状を作ります。これは `Drag` ツールのデフォルトモードです。`Dynamic Topology` と組み合わせることで、ヘビのような押し出しを作ることができます。 
![](/videos/stroke_drag.mp4) 


### ![](/icons/grab.webp) グラブ {#grab}
ブラシを開始したときにブラシ半径内にあるものを選択し、その選択を保持します。これは、移動量を慎重に調整し、最初に選択した以上の部分を誤って動かさないようにしたい、より正確な移動操作に便利です。これは `Move` ツールのデフォルトモードです。
![](/videos/stroke_grab.mp4) 


### ![](/icons/radius.webp) ロック + 半径 (ドラッグ) {#lock-radius-drag}
ユーザー指定の半径は無視され、ストロークの開始点からどれだけ離してドラッグしたかに基づいて動的に設定されます。短い距離 = 小さい半径、長い距離 = 大きい半径となります。フォールオフの形状を制御するには強度スライダーを使用します。有機的でゴムのような形状のブロッキングに便利です。
![](/videos/stroke_lockradius_drag.mp4) 



### 間隔強度を調整 {#adjust-spacing-intensity}
![](/images/stroke_space_smooth.webp)

間隔が小さいストローク（50% 未満）はすばやく蓄積し、間隔が大きい値よりも強いストロークになります。このトグルを有効にするとこれを補正し、間隔に関係なくストロークの強度がおおよそ同じになるようにします。

### ストローク間隔 {#stroke-spacing}
ドラッグ操作中に各ブラシストロークをどれだけ離して適用するかを指定します。100% 未満の値ではストロークが重なり、連続したストロークのように見えます。100% を超える値では隙間が生じ始め、ステッチやジッパーのようなディテールをスカルプトするのに便利です。

### レイジーロープ安定化 {#lazy-rope-stabilizer}
ストロークはポインタ位置から一定距離だけ遅れて追従します。これを使って滑らかな線を描くことができます。
![](/videos/stroke_lazy_rope.mp4) 

### ストロークスムージング {#stroke-smoothing}
複数のポインタ位置を平均して、より滑らかなストロークを得ます。
値を高くするとストロークはポインタから遅れますが、最終的には追いつきます。
滑らかな線を描くのに便利です。

### スナップ半径 {#snap-radius}
ストロークの開始位置を前のストロークの終点にスナップします。半径は、前のストロークの終点を探す距離を決定します。頻繁に一時停止しながら長い連続線を描く場合に便利です。

### ![](/icons/silhouette.webp) シルエット {#silhouette}
![](/images/stroke_silhouette.webp)
デフォルトでは、ストロークはブラシ半径内のモデル表面にのみ影響します。シルエットを有効にすると、ストロークはモデル全体を貫通して投影されます。これはモデルの初期ブロッキングや、側面を垂直に保つ必要がある形状に非常に有用です。

投影方向は明示的に設定できます。デフォルトの「Closest」方式では、ビューに対して最適な角度が自動的に検出されます。 

![](/videos/stroke_silhouette.mp4) 

### ![](/icons/dice.webp) ランダマイズ {#randomize}

![](/images/stroke_randomize.webp)

ストロークの強度、平行移動、回転、スケールをそれぞれランダム化できます。これを使ってさまざまな効果を作成できます。例えば、ペイントツールでランダム化するとまだらな色を作れますし、クリースツールでランダム化すると肌のディテールを作るのに使えます。

![](/videos/stroke_randomize.mp4) 

### ストロークオフセット {#stroke-offset}

ストロークに一定のオフセットを適用します。これは、指がストロークを隠してしまうような小さな画面で便利です。 


## ![](/icons/alpha.webp) アルファ {#alpha}
![](/images/stroke_alpha.webp) 

`Alpha` はブラシの挙動を変調するテクスチャです。
これはグレースケール画像で、黒いピクセルは変形なし、白いピクセルは最大変形を意味します。

アルファ画像をクリックして変更します。

マテリアルプレビューをクリックすると、マテリアルプリセットからアルファを読み込めます。ここでマテリアルプリセットを保存し、ツールにテクスチャを埋め込むかどうかを選択することもできます。

::: tip
テクスチャはリサイズされないため、大きなテクスチャはパフォーマンスを低下させる可能性があります。
:::

### ピクセルを反転 {#invert-pixels}
画像の値を反転します。黒いピクセルは白に、白いピクセルは黒になります。

::: tip

Nomad に同梱されているビルトインアルファは反転できません。

:::

### スケーリング {#scaling}

Nomad のブラシサイズは、ユーザー定義の半径を持つ円です。テクスチャは多くの場合正方形または長方形であり、`Scaling` パラメータを使ってテクスチャを円の中にどのように収めるかを決められます。正方形テクスチャの場合、値 0.7 で円の内側に収まります。値 1 ではテクスチャが拡大され、円がテクスチャの内側に収まるようになり、端がクリップされます。

![](/images/alpha_scaling.webp) 

`Scaling - Y` を有効にすると、アルファを縦方向に伸縮できます。

![](/images/alpha_scaling_y.webp) 

### 回転 {#rotation}

アルファテクスチャはストロークの方向に合わせて回転します。回転オフセットを追加でき、ロックアイコンを有効にすると、テクスチャは画面に対してこの回転に固定されたままになります。

### タイリング {#tiling}
![](/images/stroke_tiling.webp) 

ブラシプロファイル内でテクスチャをどれだけ頻繁に繰り返すかを指定します。タイルモードでは、ストローク内を単一テクスチャに制限したり、繰り返しテクスチャにしたり、あるいは 2 枚ごとにテクスチャを反転させてパターンを作成したり、シームレスなテクスチャを作るのに役立てたりできます。

![](/videos/alpha_tiling.mp4) 



### 中間値 {#mid-value}

デフォルトでは、黒いピクセルは変形なし、白いピクセルは完全な正の変形を意味します。例えば、岩のアルファテクスチャを持つクレイブラシでは、アルファが黒でない部分だけが表面を押し出します。

ブラシで表面を押し込む、あるいは押し込みと押し出しの両方を行いたい場合は、ミッド値を変更できます。値を 0.5 に設定すると、アルファの中間のグレーは何もしません。黒は押し込み、白は押し出します。




## フォールオフ {#falloff}

![](/images/falloff_menu.webp) 

これは [Alpha](#alpha) と似ていますが、強度を単一のカーブで制御します。これはアルファと組み合わされるため、ディテールにはテクスチャを使い、フォールオフでツール中心部の強度やエッジのフェードを制御できます。

カーブが上にあるときは最大変形、下にあるときはブラシが何の効果も持たない状態です。

カーブはブラシ先端の断面図と考えることができます。下部のセクションには、ブラシを 1 回「スタンプ」したときにモデル表面がどのように見えるかのプレビューが表示されます。ブラシにアルファテクスチャがある場合は、それも表示され、フォールオフとアルファがどのように相互作用するかをプレビューできます。

### プリセット {#preset}
これを選択した状態でカーブグラフをクリックすると、プリセットメニューが表示されます。丸みを帯びたカーブはソフトな結果を、角ばったカーブはシャープな結果をもたらします。左ツールバーの `Sub` ボタンはフォールオフを事実上反転させ、カーブの上部が表面を押し出す代わりに押し込むようになったり、その逆になったりします。

### Catmull-Rom {#catmull-rom}
選択すると、ユーザーは独自のフォールオフカーブを描くことができます。カーブエディタは Nomad の他のカーブと同じように動作します:

* カーブをクリックして新しいポイントを作成
* ポイントをドラッグして位置を変更
* ポイントをクリックしてシャープとスムーズを切り替え
* ポイントを隣のポイントにドラッグして削除

### B-spline {#b-spline}
選択すると、ユーザーは独自のフォールオフカーブを描くことができます。エディタの操作は Catmull-Rom と同じですが、ポイントはカーブ上ではなくカーブに影響を与える位置にあり、より滑らかなカーブ形状を作るのに役立ちます。

カーブエディタには 3 つのボタンがあります:

| Item     | Icon                        | Description                                  |
| :------: | :-------------------------: | :------------------------------------------: |
| Maximize | ![](/icons/maximize.webp)  | カーブエディタを拡大表示します              |
| Reset    | ![](/icons/reset.webp)     | カーブをデフォルト状態に戻します            |
| Symmetry | ![](/icons/symmetric.webp) | カーブを対称な「ブラシ先端」として表示します |


### 影響度 {#influence}

* Sphere (3d) - 影響は、頂点からブラシ中心までの距離を取ることで計算されます。
* Circle (2d) - 頂点はまずエリア平面に投影され、その後ブラシ中心までの距離が取られます。これはアルファのサンプリング方法と似ています。 
* Cylinder - 影響はエリアを通してシリンダーとして投影され、下記のシルエットモードで使用されます。

### シルエット {#silhouette-1}
デフォルトでは、ストロークはブラシ半径内のモデル表面にのみ影響します。シルエットを有効にすると、ストロークはモデル全体を貫通して投影されます。これはモデルの初期ブロッキングや、側面を垂直に保つ必要がある形状に非常に有用です。



## フィルター {#filter}

![](/images/filter_menu.webp) 

### ストロークを蓄積 {#accumulate-stroke}
1 回のストロークで追加/削除できる量に制限を設けないようにします。例えば `Clay` ツールではこれが有効になっているため、素材を積み重ね続けることができます。一方 `Brush` ツールではこれが無効になっているため、同じストロークをメッシュの同じ領域に動かし続けても、一定以上は素材が追加されません。 

### 手前向き頂点のみ {#front-facing-vertex-only}
背面を向いた頂点を無視します。
薄いジオメトリの一部だけを、反対側に影響を与えずにペイントしたい場合に便利です。
スカルプトにも機能しますが、アーティファクトが発生する場合があります。

### ダイナミックトポロジーを許可 {#allow-dynamic-topology}
このオプションは、メッシュが [Dynamic Topology](topology.md#dynamic-topology) モードの場合にのみ利用できます。ツールごとにダイナミックトポロジーを有効・無効にできます。

### 接続トポロジー {#connected-topology}
ツールで触れた表面に連結している頂点のみをスカルプトするようにします。例えば `Move` ツールと一緒に使用すると、別のパーツと交差していても、その一部だけを動かすことができます。
![](/videos/connected_topology.mp4)

### エリア保護 {#protect-area}
![](/images/filter_protect_area.webp) 

これらのオプションは、さまざまな条件の下でメッシュの一部にツールが影響するのを防ぎます。 

「Auto」オプションは、[Shading](shading) メニューで非表示、マスク、フェイスグループが表示されている場合はそれらが保護されることを意味しますが、そのメニューでオフになっている場合は保護が無効になります。

* `All` - 保護フィルタをグローバルにするか、ツールごとにするかを切り替えます。
* `Hide` - Hide ツールでメッシュの一部が非表示になっている場合、それらを保護するかどうかを設定します。
* `Mask` - メッシュの一部がマスクされている場合、それらを保護するかどうかを設定します。
* `Facegroup` - 最初に触れたフェイスグループ内でのみツールを使用できるようにするかどうかを設定します。


### シャープエッジを保持 {#keep-sharp-edges}
法線が面の法線と大きく異なるポイントを除外します。

これにより、平面エリアの計算方法（エリアサンプリング）が変わります。

このオプションは、フラット化系ツールや、はみ出しなしでほぼ平坦な表面に色を付けたい場合に便利です。

![](/videos/filter_keep_sharp_edges.mp4)

### エリアサンプリング {#area-sampling}
一部のブラシやストロークオプションは、動作するために表面に対する平面法線と平面位置を必要とします。

ツール半径に対する比率としてサンプリングエリアを設定することで、この平均平面の計算方法を制御できます。

100% では、選択円内のすべてのポイントが考慮されます。

0% では、最も近い頂点または三角形のみが考慮されます。これらの値は `Normal radius` と `Position radius` の両方でリンクさせることも、ロックを解除して個別に設定することもできます。


### デプスマスキング {#depth-masking}
![](/images/stroke_depthmask.webp)

計算された平面（エリアサンプリング）から、ある距離より上または下にあるポイントを除外します。

これを使って、凸部だけ、あるいはキャビティ部分だけにペイントできます。

グラフは表面の断面を表しています。水平線が表面位置で、円は表面の上下に対するペイントフォールオフ半径を表します。`Height offset` は、マスキング計算を開始する表面からの上下方向の割合です。`Top area` と `Bottom area` は、オフセットポイントより上と下のフォールオフをスケーリングするためのものです。

#### 例: くぼみにペイント {#example-paint-in-cavities}
キャビティ部分だけにペイントするには、Height offset を -100% に設定し、Top area スライダーを水平線から離れるように調整します。これは、最初のクリックで「ゼロ」深度を設定し、その深度より下の領域だけが影響を受けることを意味します。

![](/videos/stroke_depth_cavity.mp4)

#### 例: 突起にペイント {#example-paint-on-bumps}
高い部分だけにペイントするには、Height offset を +90% に設定し、円の下端が水平線とわずかに交差するようにします。高いゾーンの頂点をクリックすると、その深度が設定され、その深度と少し下、そしてそれより高い部分だけがペイントされます。深いキャビティはスキップされます。
![](/videos/stroke_depth_bump.mp4)


## 圧力 {#pressure}
![](/images/pressure_menu.webp) 

スタイラス/ペンの筆圧がブラシにどのような影響を与えるかを制御します。

デフォルトでは `Use global settings` が有効になっており、すべてのブラシが同じ筆圧設定を共有します。

### 圧力 - 半径 {#pressure-radius}

このカーブは、ブラシ半径が筆圧によってどのように変化するかを制御します。デフォルトは線形関係であり、スタイラスの応答が滑らかであれば、半径の変化も滑らかに感じられるはずです。ただし、多くのスタイラスは非線形な応答を持っており、このカーブで補正できます。例えば、高い筆圧でも半径が最大値に達していないように感じる場合は、「out-pow3」のような上向きに曲がったカーブプリセットを使って、より早い段階で半径を大きくできます。

このダイアログはフォールオフカーブ表示と似ており、カーブウィンドウをタップしてプリセットを使用したり、Catmull-Rom や B-Spline モードで独自のカーブを描いたりできます。

一定の半径にしたい場合は、このセクションを無効にしてください。

### 圧力 - 強度 {#pressure-intensity}

半径スライダーと同様ですが、強度を制御します。一定の強度にしたい場合は、このセクションを無効にしてください。

### 圧力スムージング {#pressure-smoothing}

スタイラスの筆圧イベントを平均して、より滑らかな結果を得ます。
