## 概要 {#overview}

ツールは右側の`Toolbox`から選択し、左側の`Tool Controls`で操作します。追加の設定は右上メニューの一番左のアイコンである`Settings`メニューにあります。

ブラシ系ツールにはサイズと強度のコントロールがあります。選択系ツールには複数の選択スタイルのコントロールがあります。ツールコントロールの一番下には、よく使う操作（Smooth, Mask, Hide, Gizmo, Color, Alpha）へのショートカットがあります。

Nomad のツールはツールボックス内で色分けされています:

* <span class=brush>**ブラシ系ツール**</span>（Clay, Brush, Smooth, Layer, Inflate, Nudge, Stamp, DelLayer）
* <span class=move>**移動系ツール**</span>（Move, Drag）
* <span class=mask>**マスク系ツール**</span>（Mask, SelMask）
* <span class=paint>**ペイント系ツール**</span>（Paint, Smudge）
* <span class=flatten>**フラット化ツール**</span>（Flatten, Planar）
* <span class=pinch>**ピンチ系ツール**</span>（Crease, Pinch）
* <span class=selection>**選択ベースのツール**</span> — まず 2D マスクを描き、その後に処理が行われます（Trim, Split, Project）
* <span class=creation>**作成系ツール**</span>（Tube, Lathe, Insert）
* <span class=transform>**トランスフォーム系ツール**</span>（Transform, Gizmo）
* <span class=misc>**その他のツール**</span>（Facegroup, Hide, Measure, Select）
* <span class=view>**ビュー系ツール**</span>



多くのツールは [Stroke](stroke.md) メニューから、ブラシ挙動、筆圧、テクスチャなどをカスタマイズできます。 


### ブラシコントロール {#brush-controls}

左ツールバーには半径と強度のスライダーがあり、その下にツールカテゴリ固有のコントロールがあります（後述）。

![](/images/tool_left_panel.webp)

::: tip
多くのツールでは強度スライダーは 100% を超えて設定できます。いろいろ試してみてください！
:::

### サブモード {#sub-mode}
強度スライダーのすぐ下のボタンが `Sub` ボタンです。ラベルと機能はツールごとに変化し、押している間は通常そのツールの反対の挙動を呼び出します。例えば [Paint](#paint) では消しゴムモードになり、[Crease](#crease) では溝ではなく盛り上がったエッジを作ります。

デフォルトでは「押している間だけ有効」なスティッキーボタンとして動作します。押している間だけ Sub モードが有効になり、離すとオフになります。タップすると Sub モードが常時有効になります。

### ショートカット {#shortcuts}
左ツールバーの一番下には [Smooth](#smooth), [Mask](#mask), [Hide](#hide), [Gizmo](#gizmo), [Color](painting.md#pbr-sliders), [Alpha](stroke.md#alpha) のショートカットがあります。 

デフォルトではこれらもすべてスティッキーボタンとして動作します。押している間だけそのモードが有効になり、離すとオフになります。タップすると、そのショートカットモードが常時有効になります。

### 選択コントロール {#selection-controls}

[Selection Mask](#selection-mask), [Trim](#trim), [Split](#split), [Project](#project), [Facegroup](#facegroup), [Hide](#hide) ツールはすべて、メッシュの領域を選択するために似たコントロールを使用します。

![](/images/tools_shape_selector_panel.webp)


* `Lasso` - フリーハンドで描く形状
* `Polygon` - 曲線および／または直線の組み合わせで定義される閉じた形状。[Shape editing](#shape-editing) を参照。
* `Curve` -（Project のみ）- 投影用のフリーハンド曲線
* `Path` -（Project のみ）- 点で定義される曲線。[Shape editing](#shape-editing) を参照。
* `Line` - 線をドラッグして平面セグメントを定義します。デフォルトでは即座にメッシュに対して処理を行います。これを避けたい場合は自動バリデートをオフにします（ラインアイコンを長押しまたはスワイプ）。
* `Rect` - 対角線をドラッグすると、長方形のコーナーを定義します。長押しまたはスワイプで、自動バリデート、正方形に固定、最初のクリックを長方形の中心にするオプションを表示します。
* `Ellipse` - 対角線をドラッグすると、楕円のサイズを定義します。長押しまたはスワイプで、自動バリデート、円形に固定、最初のクリックを楕円の中心にするオプションを表示します。
* `Flip` - シェイプマスク、または Project ツールの方向を反転します。

ほとんどのツールには自動バリデートのオプションがあります。これは、シェイプの描画が終わると同時に処理が実行されるという意味です。自動バリデートがオフの場合、シェイプの横に緑色のボタンが表示され、それを押すと処理が実行されます。この場合、処理前にシェイプを編集したりビューを調整したりできます。準備ができたら緑のボタンを押してシェイプを使用します。

### 形状編集 {#shape-editing}
ポリゴン編集とカーブ編集は似た挙動をします:

* まず線をドラッグして 2 点を定義し、その線の中央からドラッグしてポリゴンまたはカーブを定義します。
* 点をクリックするとスムーズ／シャープを切り替えます。 
* カーブや線のセクションをクリック＆ドラッグすると新しい点を作成します。
* 点を削除するには、その点を隣の点にドラッグし、赤くなったら離します。
* ポリゴンまたはパスアイコンの角にあるゴミ箱アイコンでシェイプを削除します。

### ![](/icons/tool_brush.webp) ブラシ {#brush}
標準的なブラシです。`Sub` で素材を削り取ります。

![](/videos/tool_brush.mp4)

### ![](/icons/tool_move.webp) 移動 {#move}
ブラシの下の領域がブラシに「くっつき」、弾性的な変形が行われます。ストローク中は選択が維持されるため、ブラシを離れた場所まで動かしてから元の位置に戻すと、変形は発生していないように見えます。

Sub モードは `Normal` で、ブラシ下の領域をサーフェス法線方向に移動させます。

このブラシは大きな形状変更にも、細かい調整にも適しています。

#### 移動設定 {#move-settings}

* `Radius (Background)` - モデルのエッジからどれくらい離れていてもスカルプトできるかを決めます。シルエットを調整するときに便利です。 
* `Same-side vertex only` - 変形方向と反対を向いている頂点を無視します。


![](/videos/tool_move.mp4)

### ![](/icons/tool_drag.webp) ドラッグ {#drag}
ブラシの下の領域がブラシに「くっつき」、弾性的な変形が行われます。Move ブラシと異なり、ストローク中に選択が継続的に更新されるため、特に Dynamic Topology を有効にしている場合、長く蛇のような形状を作ることができます。

Sub モードは `Normal` で、ブラシ下の領域をサーフェス法線方向に移動させます。

このブラシは、よりラフでジェスチャルな形状変更に向いています。

#### ドラッグ設定 {#drag-settings}

* `Radius (Background)` - モデルのエッジからどれくらい離れていてもスカルプトできるかを決めます。シルエットを調整するときに便利です。 
* `Same-side vertex only` - 変形方向と反対を向いている頂点を無視します。

![](/videos/tool_drag.mp4)

### ![](/icons/tool_smooth.webp) スムーズ {#smooth}
ポイント位置の平均を取ることで領域をスムーズにします。このツールはポリゴン密度に大きく依存します。
ポリゴン数が多いほど、スムージングの効果は弱くなります。

Sub モードは `Relax` で、ワイヤーフレームのみをスムーズにし、形状ディテールは維持しようとします。

#### スムーズ設定 {#smooth-settings}

![](/images/tool_smooth_settings.webp)

##### フェイスグループ {#smooth-facegroup}

* `Relax` - Facegroup の境界をスムーズにします。100% を超える強度で素早く境界をスムーズにできます。`Auto` は Facegroup プレビューが有効なときのみスムーズにし、`Off` は無効、`On` は常に有効です。 

##### 頂点 {#vertex}
* `Sticky vertex on border` - 平面など開いたエッジを持つメッシュでは、角をスムーズに潰してしまうことがあります。このオプションを有効にすると開いたエッジを固定します。
* `Relax` - 左ツールバーの Sub モードである Relax と同じです。
* `Stable smoothing` - トポロジに依存しないスムージングを試みます。トポロジ密度が不均一な場合や高いスムージング強度で最も効果的です。

##### ペイント {#painting}
* `Screen Smoothing` - 高ポリゴン数でもトポロジに依存しないスムージングを行いたい場合に使用します。
* `Screen samples` - スムージングの品質です。値が高いほど滑らかになりますが、処理は遅くなります。

::: tip
ポリゴン密度が高い場合は、強度を 100% 以上に上げる必要があります。非常に高い値（300%, 500% など）は、叩いて一気に平らにするようなスカルプトツールとしても有効で、ブラシの下の領域を素早く平坦にできます。
:::

![](/videos/tool_smooth.mp4)

### ![](/icons/tool_mask.webp) マスク {#mask}

Sub モードは `Unmask` で、マスクを消去します。

2D ペイントソフトの選択と同様に、マスクはブラシで塗ることも、シェイプ選択で作ることもでき、ぼかしやシャープ化も可能です。 

2D ペイントソフトと異なり、Facegroup からマスクを作成したり、マスクを使って押し出し／抽出のような操作で新しいジオメトリを作成することもできます。 

![](/videos/tool_mask1.mp4)

 ビューポート上部に追加コントロールを持つツールバーが表示されます。 

![](/images/tool_mask_toolbar.webp)

バーのタイトルをタップすると展開／折りたたみができ、右上の矢印をタップすると UI の上部／下部に移動できます。

#### マスク設定 {#mask-settings}
![](/images/tool_mask_settings.webp)

![](/videos/tool_mask2.mp4)

* `Preview` - Mask 設定メニューは主にマスクからジオメトリを作成するために使われます。そのためデフォルトでは新しいジオメトリがどのようになるかをプレビューします。プレビューなし、抽出プレビュー、分割プレビュー、さらにそのジオメトリを X-ray モードで表示するかどうかを選択できます。

##### 厚み {#thickness}
* `Height` - 抽出される形状の高さです。プラス／マイナスアイコンで、外側への押し出し、内側への押し出し、中心からの押し出しを切り替えられます。 
* `Height/Height+Mask` - 高さを一定にするか、マスクのぼかし部分で高さを変化させるかを切り替えます。後者では柔らかく高さが変化する形状を作れます。 

##### スムーズさ {#smoothness}
有効時、抽出形状の境界をスムーズにします。ポリゴン数が多いほど効果的です。 
* `Iterations` - 適用するスムージング量です。値が高いほど非常に滑らかな曲線エッジになりますが、マスク形状からのズレが大きくなります。
* `All/Sharp border/Borders only` - スムージングの方向を制御します。すべて（側面と上面）、シャープなエッジを保ちつつ側面と上面、境界のみ（上面はそのまま）から選択できます。

##### エッジループ（側面） {#edge-loop-side}
* `Auto Edge-loop (side)` - 抽出形状の側面に、マスク領域のポリゴンと揃った正方形ポリゴンができるよう分割数を自動計算します。無効時はスライダーでエッジループ数を手動設定します。

----

##### 抽出 {#extract}
* `Extract` - 抽出ジオメトリを作成します。
* `Closing action` - 抽出の挙動を指定します。`None` はマスクされたポリゴンを新しい形状として複製します。`Fill` は同様に複製し、背面を塞ごうとします。`Shell` は `thickness` で設定した量だけ押し出し、デフォルトです。

::: tip

プレビューが `Extract` モードかつ `X-ray` 有効のときに Extract ボタンを押すと、最初は混乱するかもしれません。メニューがアクティブなため、新しい形状に対して再度抽出プレビューを行おうとし、元のサーフェスを X-ray 表示にします。しかし新しいサーフェスにはマスクがないため抽出プレビューができず、「Nothing to Extract!」という警告が表示されます。 

これは正常な挙動です。Mask 設定メニューを閉じて新しい形状と元の形状を確認し、必要であれば元のサーフェスを再選択してマスクをクリアしたり新しいマスクを描いたりしてください。
:::

##### 分割 {#split-mask}
* `Split` - マスクされた領域と非マスク領域の両方を新しい形状として抽出します。 
* `Closing action (masked)` - マスク側抽出の挙動です。`None` はマスクされたポリゴンを新しい形状として複製します。`Fill` は同様に複製し、背面を塞ごうとします。`Shell` は `thickness` で設定した量だけ押し出し、デフォルトです。
* `Closing action (unmasked)` - 非マスク側抽出の挙動です。`None` は非マスクポリゴンを新しい形状として複製します。`Fill` は同様に複製し、背面を塞ごうとします。`Shell` は `thickness` で設定した量だけ押し出し、デフォルトです。
* `Sync border` - マスク側と非マスク側の抽出形状の境界が近い位置に保たれるようにします。無効時は、Shell 操作が各面を法線方向に押し出すため、形状間に隙間が生じることがあります。

##### カーブ {#carve}
* `Carve` - デフォルトモードでは、`thickness` 量だけ表面を削り取ったような挙動をし、オレンジの皮を切り取るような結果になります。 



### ![](/icons/tool_maskSelector.webp) 選択マスク {#selection-mask}
このツールは基本的に [Mask ツール](#mask) とほぼ同じですが、ストロークでマスクを塗るのではなく、[選択コントロール](#selection-controls) を使ってマスクを作成する点が異なります。

Sub モードは `Unmask` で、選択コントロールを使ってマスクを消去します。

Selection Mask は `Mask` ツールと同じツール設定を共有します。

![](/videos/tool_selector_mask.mp4)

### ![](/icons/tool_paint.webp) ペイント {#paint}
色とマテリアルプロパティを適用します。マテリアルについて詳しくは [Painting](painting.md) セクションを参照してください。

Sub モードは `Erase` で、ペイントを削除します。

#### ペイント設定 {#paint-settings}
* `Layer fitering` - Photoshop や Procreate のレイヤーアルファロックのように機能します。レイヤー上でペイントしている場合、これを有効にすると、すでにペイントされている部分のみ変更でき、未ペイント部分は保護されます。

### ![](/icons/tool_flatten.webp) フラット {#flatten}
ポイントを平均平面に投影して領域をフラットにします。

Sub モードは `Fill` で、最も高いポイントを基準に平面を定義し、ポイントを持ち上げる方向に引き寄せます。

#### フラット設定 {#flatten-settings}

* `Lock plane direction` - 最初のクリックで計算された平面方向を使用します。デフォルトでは無効です。
* `Lock plane origin`- 最初のクリック位置を平面の中心として使用します。デフォルトでは無効です。

これらのいずれか、または両方が無効な場合、Flatten は長いストロークで異なる深さや曲率をまたぐことで、徐々に深さを変えたり平面角度を変えたりできます。これはブラシメニューのエリアサンプリングオプションと組み合わせることで、非常に精密なコントロールが可能になります。

::: tip
曲率の高い領域（例えば頬をフラットにしたいのに、ブラシが鼻の側面まで影響してしまう場合）では、Flatten ブラシの影響を受けてほしくない部分をマスクで保護してみてください。
:::

![](/videos/tool_flatten.mp4)


### ![](/icons/tool_planar.webp) プレーナー {#planar}
ポイントを平均平面に投影して平面化しますが、Flatten ブラシよりも蓄積が少なくなります。これにより、よりクリーンなハードエッジ面が作られます。素早いストロークは表面を押し引きし、すでに平面な領域からゆっくり外側に向かってストロークすると、平面を維持しやすくなります。

Sub モードは `Fill` で、最も高いポイントを基準に平面を定義し、ポイントを持ち上げる方向に引き寄せます。

Planar は実際には `Flatten` と同じツールですが、`Lock plane direction` が有効になっているため、より安定したハードエッジ面を作りやすくなっています。一方 Flatten はより彫刻的で、半分フラットな領域を作るのに向いています。

![](/videos/tool_planar.mp4)

### ![](/icons/tool_crease.webp) クリース {#crease}
Crease ツールは小さな切れ込みや溝を彫るのに便利です。

Sub モードは `Invert` で、溝ではなく盛り上がった線を作ります。

#### クリース設定 {#crease-settings}

* `Pinch factor` - 頂点をブラシ中心に向かって横方向にどれだけ引き寄せるかを決めます。Pinch が 1、Offset が 0 の場合、表面の高さは変わらず、エッジだけがストロークに向かって引き寄せられます。
* `Offset factor` - 頂点を奥行き方向にどれだけ押し引きするかを決めます。Pinch が 0、Offset が 1 の場合、深い溝や盛り上がりができますが、溝の側面や底を定義するジオメトリが十分に引き寄せられないため、ギザギザに見えることがあります。

![](/videos/tool_crease.mp4)

### ![](/icons/tool_trim.webp) トリム {#trim}
Trim ツールはメッシュの一部を切り取るツールで、残された穴をどのように処理するかのオプションを提供します。切り取り範囲は [選択コントロール](#selection-controls) で定義します。

::: tip
このツールはカメラからの投影で動作するため、カメラがパースペクティブモードの場合は警告が表示されます。

オルソビューでは、メッシュを切る平面はビューに平行になり、一般的に期待される結果になります。パースビューで行うと、手前側と奥側で切り口の見え方が変わります。
:::

#### トリム設定 {#trim-settings}

* `Stroke painting` - Paint メニューでペイントが有効な場合、塞がれた領域は現在選択中の色で塗られます。
* `Boolean` - Trim の穴を四角ポリゴンで埋めます。埋められた領域はフラットになります。
* `Legacy` - Trim の穴を三角形で埋めます。埋められた領域はフラットになります。
* `Fill` - Trim の穴を三角形で埋めます。埋められた領域は石鹸の膜のような曲面になります。
* `None` - 穴を埋めません。
* `Boolean Detail Shape` - Trim の「形状側」に使われる四角形や三角形の大きさの目安です。
* `Boolean Detail Hole` - Trim で埋められる穴側に使われる三角形やポリゴンの大きさの目安です。 
* `Legacy Detail` - Trim に使われる三角形の大きさの目安です。
* `Legacy Hole smoothing` - 埋められた領域の三角形をどれだけリラックスさせるかを決めます。
* `Legacy Threshold espilon` - Legacy の穴埋めアルゴリズムを改善するために調整できる値です。
* `Fill Detail` - Trim に使われる三角形の大きさの目安です。

### ![](/icons/tool_layer.webp) レイヤー {#layer}
表面を持ち上げますが、高さに上限を設けます。

ペンを離さずに同じ場所を何度もなぞっても、Layer は一定の高さまでしか盛り上がらず、それ以上は増えません。他のツールはストロークを重ねるほど高さが蓄積していきます。

なお、デフォルトではこの上限はストロークごとにリセットされます。新しいストロークを開始すると、その時点の表面高さから再び盛り上がります。

複数ストロークにわたって一定の最大高さを維持したい場合は、Nomad の [Layers](layers.md) システムと Layer ツールを組み合わせて使います。

レイヤーを作成し、このツールを使用します。最大高さはレイヤーに紐づくため、何度ブラシストロークを重ねても高さは常に同じになります。

`Sub` は最小深度を使用し、溝を作ります。

#### レイヤー設定 {#layer-settings}

* `Use layer data` - 有効時、かつレイヤーが選択されている場合、最大高さをレイヤーデータから取得します。
* `Inflate`- 有効時、Layer の作用方向を調整し、より滑らかな結果を得ます。
* `Relax (Normal)` - 法線に適用されるスムージング量です。

### ![](/icons/tool_stamp.webp) スタンプ {#stamp}

クリック＆ドラッグで、選択中の Alpha の形状に沿って造形を盛り上げます。

これは単に Stroke タイプを `Lock + radius` に設定した [Brush ツール](#brush) です。 

`Sub` はスタンプを押し込む方向に作用します。

::: tip
Stamp は通常、高解像度ジオメトリで最も効果的です。オンラインで「wrinkles alpha」「pores alpha」「scales alpha」などを検索すると、これらのアルファテクスチャと Stamp を組み合わせて、キャラクタースカルプトに有機的なディテールを追加するのに非常に役立ちます。

2 つのストロークモードは用途が異なります。 

* `Lock + radius` は*高さ*が一定で、ドラッグでスタンプの幅と回転を調整します。同じ深さ／高さで、回転やスケールだけ変えたい皮膚テクスチャなどに向いています（パターンの繰り返しを隠すのに有効）。
* `Lock + intensity` は*幅*が一定で、ドラッグで回転と高さを調整します。同じサイズで、回転や高さだけ変えたいリベットなどに向いています。 

:::

![](/videos/tool_stamp.mp4)

#### チューブ左ツールバー {#tube-left-toolbar}

![](/images/tool_tube_left_toolbar.webp)

左ツールバーには次のオプションがあります:

* `Sync` - 現在のチューブがインスタンスであり、インスタンス間で子ノードが異なっている場合、それらを再同期します。
* `Snap` - 有効時、Curve と Path モードで描画したカーブやパスが他オブジェクトにスナップします。無効時は最初のポイントのみスナップし、その深度で残りのチューブが描画されます。小さなフライアウトメニューがあります:
    * `Offset` - スナップの深さを設定します。0% はチューブ断面の中心がサーフェスにスナップし、正の値はサーフェスから浮かせ、負の値はサーフェスにめり込ませます。

    * `Closed` - チューブをループ状にします。
    * `Screen` - Auto-validate 無効時のみ利用可能です。有効時、パスは画面に「固定」され、ビューやオブジェクトを動かしてもパスはその場に留まります。無効時、パスは 3D シーンの一部となり、カメラやオブジェクトと一緒に動きます。

* `Path` - クリックでカーブポイントを定義してチューブを作成します。緑のサークルをタップしてパスを確定します。小さなフライアウトメニューがあります:
    * `B-spline` - カーブポイントが必ずしもカーブ上に乗らない代わりに、デフォルトより滑らかな結果を得られる別のカーブ描画方式です。
    * `Closed` - チューブをループ状にします。
    * `Screen` - 有効時、パスは画面に「固定」され、ビューやオブジェクトを動かしてもパスはその場に留まります。無効時、パスは 3D シーンの一部となり、カメラやオブジェクトと一緒に動きます。

![](/images/tool_tube_toolbar.webp)

### ![](/icons/tool_insert.webp) インサート {#insert}
あるオブジェクトの表面上に別のオブジェクトを配置します。使い方は Stamp ツールに似ていますが、フル 3D 形状を扱います。

左ツールバーからプリミティブを選択した状態で、任意のサーフェス上をクリック＆ドラッグすると、その位置にプリミティブが配置され、ドラッグ量でサイズが決まります。ドラッグを終えると、Insert は [Transform](#transform) モードに切り替わります。

Instance モードでは、ドラッグで新しいインスタンスをサーフェス上にスライドさせながら作成します。サイズは最初の形状から複製されるため、同じサイズのオブジェクトを多数配置するのに便利です。

挿入できるのはプリミティブだけではありません。アウトライナーで*任意の*形状を選択し、Insert を Instance または Clone モードにしておけば、そのオブジェクトのコピーを他のサーフェス上にドラッグして配置できます。

オブジェクトにカスタムピボットが設定されている場合、そのピボットがアンカーポイントとして使用されます（下の動画参照）。

### ![](/icons/tool_transform.webp) トランスフォーム {#transform}
1 本指／2 本指で、主に他オブジェクトの表面上でモデルを移動／回転／スケールします。

ツールは左ツールバーで制御し、5 つのボタンがあります:

* `Snap` - モデルを他のサーフェスにスナップします。
* `Group` - 選択オブジェクトが複数オブジェクトやインスタンスの組み合わせで構成されている場合、ツールの挙動を決定します。
* `Move` - 1 本指ドラッグでオブジェクトを移動します。Snap 有効時は、指の下のサーフェスに沿ってオブジェクトをスライドさせます。
* `Rotate` - 1 本指ドラッグでオブジェクトを回転させます。Snap 有効時は、指の下のサーフェス法線を軸に回転します。
* `Scale` - 1 本指ドラッグでオブジェクトをスケールします。

Transform は 2 本指を使うことで、これら 3 モードのうち 2 つを素早く切り替えて操作できます:

* オブジェクトをドラッグして配置します。最初の指を画面から離さずに動きを止めます。
* 2 本目の指を画面に触れたままドラッグすると、オブジェクトがスケールします。
* 2 本目の指を離し、1 本目の指を再びドラッグすると、オブジェクトは再び Move モードになります。

また、2 本目の指のタップジェスチャで 2 つ目のモードを切り替えることもできます:

* オブジェクトをドラッグして配置し、動きを止めますが、指は離しません。
* 1 本目の指を押さえたまま 2 本目の指でタップします。
* ツールが Rotate モードに切り替わります。1 本目の指をドラッグして回転を設定します。
* 再度 2 本目の指でタップすると、ツールは Move モードに戻ります。

これは、例えば地形上に岩を複製して配置するような作業に非常に高速なワークフローを提供します。Clone ボタンも左ツールバーにあるため、簡単にアクセスできます:

* Transform ツールで岩を配置します。
* 指を離し、Clone ボタンを押します。
* この岩を移動し、必要に応じて回転／スケールします。
* 指を離し、再度 Clone ボタンを押します。
* この岩を移動し、繰り返します。

![](/videos/tool_transform.mp4)

### ![](/icons/tool_gizmo.webp) ギズモ {#gizmo}
オブジェクトの移動、回転、スケール、およびピボットの変更を行うツールです。

ビューポートのハンドルには次の機能があります:

* `Move` - 線＋矢印をドラッグして X/Y/Z 軸方向に移動します。Gizmo 中央のピーチ色の点をドラッグすると、スクリーンスペースで自由に移動します。赤／緑／青の四角をクリックすると、それぞれ X/Y/Z 平面上で移動します。
* `Rotate` - 赤／緑／青の円をドラッグして X/Y/Z 軸回りに回転します。円の内側の球体をドラッグすると自由回転します。
* `Scale`- 外側の赤／緑／青の点をドラッグして X/Y/Z 軸方向にスケールします。外側の赤／緑／青のコーンをドラッグすると X/Y/Z 平面上でスケールします。外側のピーチ色の円をドラッグすると一様スケールします。

![](/images/tool_gizmo.webp)

#### ノードと頂点 {#nodes-and-vertices}

Nomad のすべてのオブジェクトはノードと頂点で構成されています:

* `Node` - オブジェクトの「ハンドル」で、平行移動・回転・スケール（変換行列）を保持します。
* `Vertices` - オブジェクトの表面を定義するポイントで、位置とペイント情報を保持します。

8 頂点からなる単純なボックスがあるとします。このボックスは、変換行列を変更して移動することも、頂点位置を変更して移動することもできます。スカルプト時は通常頂点を変更し、Gizmo でオブジェクトを動かすときは通常ノードを変更します。Nomad では両方が可能です。 

#### 左メニューツールバー {#left-menu-toolbar}

左ツールバーでは、Gizmo がノードと頂点のどちらに作用するかなどを制御します:

* `Clone` - 現在のオブジェクトの独立したコピーを作成し、Gizmo でドラッグして移動できます。
* `Instance` - 現在のオブジェクトのインスタンスコピーを作成します。オブジェクト同士はリンクされており、一方をスカルプトするとすべてのインスタンスに反映されます。
* `Group/Object/Vertex/Auto` - Gizmo がオブジェクトのノードに作用するか、頂点に作用するかを設定します。デフォルトの `Auto` モードは最適な挙動を推測します。複数オブジェクトが階層構造で親子関係にある場合、`Object` は現在のオブジェクトのみを移動し、子オブジェクトはその場に留まります。Gizmo はマスクやシンメトリも考慮できます。
* `Pin` - デフォルトでは、Gizmo は選択オブジェクトのピボット位置に移動します。Pin 有効時は、Gizmo は現在位置に固定されます。
* `Align` - ピボットを現在のオブジェクトに揃えるか、ワールドに揃えるかを切り替えます。
* `Snap rotation` - 回転値を一定刻みにスナップさせます。スナップ値は有効時に表示され、編集できます。
* `Snap translation` - 平行移動値を一定刻みにスナップさせます。スナップ値は有効時に表示され、編集できます。
* `Pivot` - 有効時、オブジェクトを動かさずに Gizmo を移動・回転できます。追加メニューは後述します。

#### ピボット {#pivot}
Pivot モードが有効なとき、ピボットを素早く変更するためのメニューが表示されます:

**Reset** 
* `Center` - ピボットをオブジェクトの中心に移動します。
* `Bottom` - ピボットをオブジェクトの底面に移動します。
* `Align` - 回転をリセットし、ワールドに揃えます。
* `Mask` - ピボットを非マスク領域の中心に移動します。

**On Tap**
* `None` - オブジェクトをタップしても何もしません。
* `Normal` - タップしたサーフェス位置にピボットを移動し、その法線に合わせて回転させます。
* `First` - タップしたサーフェス位置にピボットを移動します（回転は変更しません）。
* `Medial` - タップしたサーフェスの真下にあるオブジェクトの中間位置にピボットを移動します。

##### マトリックス {#matrix}
* ![](/icons/target.webp) `Move origin` - オブジェクトのピボットがシーンの中心（原点）に来るようにオブジェクトを移動します。
* ![](/icons/bake.webp)  `Bake` - 現在位置を固定し、平行移動・回転を 0、スケールを 1 にリセットします。
* ![](/icons/bake.webp) -> ![](/icons/tool_gizmo.webp) `Bake Pivot` - Gizmo ハンドルのワールド位置に変換行列の値を合わせます。
* ![](/icons/reset.webp) `Reset` - 平行移動を 0、回転を 0、スケールを 1 に設定し、オブジェクトを移動・回転させます。

::: tip Bake と Bake Pivot の違い
ボックスを作成し、Gizmo ツールを選択して設定メニューを開き、ピン留めします。デフォルトでは平行移動と回転は 0、スケールは 1 です。

Pivot モードを有効にし、ハンドルを片側に移動してから Pivot モードを無効にします。ピボットは変わりましたが、平行移動値はまだ 0 のままです。 

ピボットの「本当の」位置を確認したい場合は `Bake Pivot` をクリックします。すると平行移動値が更新されます。この操作中、ボックスも Pivot モード中も位置は変わりません。

ボックスを移動・回転させてから `Move Origin` を押すと、ピボットがワールド中心に来るようにオブジェクトが移動しますが、回転はそのままです。

`Reset` をクリックすると、回転も 0 にリセットされます。

再度ボックスを移動・回転させ、今度は `Bake` をクリックします。ピボットもボックスもその場に留まりますが、平行移動と回転の値は 0 に設定されます。

これを何度か試してみてください。ピボット値は普段は隠されており、Nomad が面倒を見てくれますが、ピボットを空間上の実際の位置に合わせたい場合は Bake Pivot が役立つことが理解できるはずです。

:::

* `Translation` - 平行移動 X, Y, Z の値
* `Rotation` - 回転 X, Y, Z の値
* `Scale` - 一様スケール有効時は一様スケール値、無効時は X, Y, Z のスケール値
* `Uniform scale` - 一様スケールと各軸独立スケールの切り替え

-----

* `Compact` - Gizmo レイアウトを切り替え、追加ハンドルを回転球の外側／内側に配置します。
* `Widget size` - Gizmo のサイズ
* `Thickness` - Gizmo の線の太さ
* `Opacity` - Gizmo の不透明度
* `Hide on interaction` - ドラッグ中に Gizmo を一時的に非表示にするかどうかを切り替えます。

-----

* `Tangent roll threshold` - X/Y/Z 回転用の円ハンドルをドラッグしたときの UI 挙動を制御します。値が 0 の場合、回転はダイヤルのように円を描いてドラッグします。値が 90 の場合、ヨーヨーの紐を引くように、最初にクリックした位置に向かって直線的にドラッグします。0〜90 の値では両方の挙動を組み合わせ、閾値以下では直線的、閾値以上では円運動になります。
* `Numerical input` - 有効時、Gizmo をシングルタップすると、タップしたウィジェット軸に対して正確な値を入力するウィンドウが表示されます。

[Gizmo](#gizmo), [Translate](#translate), [Rotate](#rotate), [Scale](#scale) はそれぞれ独自のシンメトリチェックボックスを持っています！

デフォルトではこのシンメトリはオフになっています。
:::

### ![](/icons/tool_faceGroup.webp) フェイスグループ {#facegroup}

Facegroup を使うと、オブジェクトを色分けされた面のグループとして整理できます。Nomad では Facegroup を次のように活用できます:

* マスクのクイック選択
* オブジェクトの一部の表示／非表示
* オブジェクトを分割せずに整理する
* UV 領域の定義
* Quad Remesher のガイド
* Smooth などのツールに追加のコントロールを与える

#### フェイスグループ左ツールバー {#facegroup-left-toolbar}

* `Patch ` - 利用可能な Facegroup をパッチとして表示します。未使用パッチは削除できます。パッチをタップすると名前や色を変更できます。プラスアイコンで新しいパッチを作成します。
* `Dot` - オブジェクト上をペイントして Facegroup を定義します。`+ Face Group` 有効時は、ストロークごとに自動的に新しい Facegroup が作成され、領域を素早く定義するのに便利です。タップで選択領域を塗りつぶします。スライダーでドットの半径を設定します。
* `Relax` - Facegroup の境界をスムーズにします。Quad Remeshing 用のクリーンなエッジや、ハードサーフェスモデリング用のパネルを定義するのに非常に有用です。スライダーで半径と強度を制御します。
* `Shape selector` - ブラシではなくシェイプで Facegroup を作成します。`Lock+Radius`, `Lasso`, `Polygon`, `Rect`, `Ellipse` が利用できます。[Shape Selector](#shape-selector) を参照してください。
* `Auto-pick` - 有効時、ストローク開始位置のパッチを自動的に選択し、そのストローク全体に適用します。Facegroup 領域の整理に非常に便利です。Facegroup がはみ出している場合、Auto-pick を有効にし、正しいパッチのある場所からストロークを開始して境界までドラッグすると、正しいパッチに再割り当てできます。

### ![](/icons/tool_hide.webp) 非表示 {#hide}
オブジェクトの一部を非表示または単独表示にします。 

主なモードは左メニューから制御します:

* `Shape selector` - ブラシではなくシェイプで非表示にします。`Lasso`, `Polygon`, `Line`, `Rect`, `Ellipse` が利用できます。[Shape Selector](#shape-selector) を参照してください。

ビューポート上部に追加コントロールを持つツールバーが表示されます:

* `Clear` - オブジェクトを復元し、すべての非表示部分を再表示します。
* `Invert` - 非表示部分と表示部分を入れ替えます。
* `Facegroup` - Facegroup を使って素早く領域を非表示にします。Facegroup をタップすると、その Facegroup 全体が非表示になります。
* `Mask` - マスクが有効な場合、このボタンをタップするとマスクされた部分が非表示になります。
* `Delete` - 非表示部分を削除します。
* `Split` - 非表示部分を新しい形状として分割します。

* `<Count>, Half, Same` - 上部ツールバーの Remesh、Half、Same ボタンと同じです。
* `Target Quads` - 上部ツールバーの `Quads` ボタンと同じです。
* `Adaptive quad count` - 曲率の高い領域では小さなクアッドを、曲率の低い領域では大きなクアッドを使用する自動調整を有効にします。

* `Reproject Vertex` - 新しいレイアウトを入力メッシュへ再投影するかどうかを切り替えます。
* `Face Group` - 上部ツールバーの `Face Group` ボタンと同じです。
* `Relax Slider` - フェイスグループ境界をリラックスさせるためのショートカットスライダーです。

::: tip

アーティファクトを最小限に抑えて良好なクアッドリメッシュを得るためのレシピ:

* メッシュにフェイスグループを設定して、理想的なクアッドフローを定義します。
* フェイスグループをリラックスして、滑らかなフェイスグループ境界を作ります。
* デシメートします。これにより、フェイスグループのエッジに細長い面や歪んだ面が存在しないようにできます。デシメート設定では、フェイスグループが有効になっていることを確認してください。これにより、三角形のエッジがフェイスグループのエッジに正確に沿うようになります。

Quad Remesher のオプションでは、すでにメッシュをリラックスしているため、Relax を無効にしておくと良い結果が得られるはずです。

:::

### ![](/icons/tool_select.webp) 選択 {#select}
シェイプモードを使用してシーン内のオブジェクトを選択します。`Unselect` は選択からオブジェクトを除外します。

### ![](/icons/tool_view.webp) ビュー {#view}
この「ツール」は特に何もしません。これは単に、シーンを変更せずにモデルを閲覧するための手段です。


## ツールボックスコンテキストメニュー {#toolbox-context-menu}

![](/images/tools_context_menu.webp)

ツールボックス内のツールを右クリック、または長押しするとコンテキストメニューが表示されます。このメニューには次のオプションがあります:

* `Save` - ツールに加えた変更を保存します
* `Clone` - ツールを複製して新しいツールショートカットを作成します
* `Last save` - 直前に保存したバージョンのツールに戻します
* `Icon` - ツールのアイコンを変更します
* `Reset` - ツールをデフォルト設定にリセットします
