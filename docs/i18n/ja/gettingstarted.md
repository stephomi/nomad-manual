# はじめに

## Nomad へようこそ！

Nomad は多くのデバイスで動作する 3D スカルプトアプリです。特に、Apple iPad と Apple Pencil、Samsung Galaxy Tab とスタイラスなど、筆圧感知スタイラス付きのタブレットで最も快適に動作します。

デスクトップ向けスカルプトアプリである ZBrush や Blender にインスパイアされており、豊富な機能を維持しつつ、分かりやすい UI に重点を置いています。すでに 3D スカルプトアプリを使ったことがある場合、Nomad はとても馴染みやすく感じられるはずです。

もしこれが初めての 3D スカルプトであれば、いくつか基本を知っておくと良いでしょう。

## 最初のスカルプト

Nomad を起動すると、画面に球体が表示されます。スタイラスで球体をドラッグするだけでスカルプトを開始できます。スカルプトをしやすくするため、対称モードはデフォルトで有効になっています。

![](/videos/gettingstarted_01.mp4)

ブラシのサイズを大きくしたり小さくしたりするには、左側の半径スライダーを使用します。

![](/videos/gettingstarted_02.mp4)

ブラシの強さを強くしたり弱くしたりするには、左側の強度スライダーを使用します。

![](/videos/gettingstarted_03.mp4)

デフォルトのツールは `Clay tool` で、これは表面に盛り上げるように追加します。表面を削り取りたい場合は、左側の `Sub` ボタンをタップします。再度表面に追加したい場合は、もう一度 Sub ボタンをタップします。

![](/videos/gettingstarted_04.mp4)

表面をスムーズにするには、`Smooth` ボタンをタップします。通常のスカルプトに戻るには、もう一度 Smooth ボタンをタップします。

![](/videos/gettingstarted_05.mp4)

モデルの周りを回転させるには、モデルの外側の空いているスペースをドラッグします。

![](/videos/gettingstarted_06.mp4)

ズームするには、2 本指でのピンチイン／ピンチアウトジェスチャーを使用します。

![](/videos/gettingstarted_07.mp4)

カメラをパンするには、2 本指で画面を押さえたままドラッグします。

![](/videos/gettingstarted_08.mp4)

間違えた場合は、2 本指でタップすると取り消しができます。または左下の取り消しボタンを使用します。 

![](/videos/gettingstarted_09.mp4)

::: tip デスクトップ版
デスクトップでは、alt/opt キーを使ってカメラを操作します：

* tip で空いているスペースをドラッグ = カメラ回転
* alt+tip ドラッグ = パン
* alt+tip ドラッグ後に alt を離す = ズーム（ZBrush と同じ）

ペンに 2 つ以上のボタンがある Wacom タブレットでは、Wacom の設定でペン先とボタンを次のように設定してください：

* tip = 左クリック
* 下側ロッカー = 中クリック
* 上側ロッカー = 右クリック

この設定により、ペンだけでカメラ操作が可能になります：

* 上側ロッカー + ホバー移動 = カメラ回転
* 下側ロッカー + ホバー移動 = パン
:::

## 色を付ける

Nomad では、スカルプトした表面にペイントすることができます。右側のツールメニューから `Paint` ツールを探してクリックします。左側のツールバーに色付きの球体が表示されるので、それをクリックするとカラーピッカーが表示されます。色を選んで、モデルの上をペイントしてください。

![](/videos/gettingstarted_10.mp4)

消しゴムとして使うには、左側ツールバーの `Erase` ボタンをタップし、そのまま表面をなぞって消します。再度ペイントモードに戻るには、もう一度 Erase ボタンをタップします。

![](/videos/gettingstarted_11.mp4)

Clay ブラシの加算／減算モード、Smooth、Paint を使って、シンプルなカートゥーン風の頭を作ってみましょう：

![](/images/gettingstarted_head1.webp)

## その他のツール

ツールパレットには、スカルプトを助ける多くのツールがあります。ここまでで使ったのは、Clay ブラシ（デフォルトの開始ツール）、Smooth、Paint です。Smooth は頻繁に使うため、左側ツールバーにショートカットが用意されています。

Nomad のツールは、Move、Crease、Inflate のようなスカルプト関連のツールから、Split や Trim のような木工や金属加工に近いツールまで、さまざまなことができます。詳しくは [Tools](tools.md) ページを参照してください。

Move、Crease、Inflate、Smooth ツールを使って、頭にさらにディテールを加えたり、形を変えたりしてみましょう：

![](/images/gettingstarted_head2.webp)

## インターフェース

![](/images/interface_overview1.webp)

* `Top menus` - Nomad のほとんどの機能にアクセスするためのメニューです。左上のメニューは主にシーンやオブジェクト関連の機能を、右上のメニューはツール関連の機能を扱います。画面が小さい場合、スペース節約のためにこれらのメニューはまとめて折りたたまれます。
* `Stats` - シーン全体、現在のオブジェクト、マスクの状態、メモリ使用量などの情報を表示します。
* `Nav Cube` - 現在スカルプトのどの向きを見ているかを示す補助表示であり、さまざまなビューにジャンプするショートカットでもあります。キューブをタップすると、その面のビューにジャンプします。ドラッグすると回転します。キューブの横にある小さなアイコンをタップすると、現在のオブジェクトをフレームに収めたり、デフォルトのホームビューにリセットしたりできます。
* `Toolbox` - Nomad のツールがこのスクロール可能な領域から利用できます。
* `Left toolbar` - 多くのツールで使う半径と強度のスライダー、その他のツール用のコンテキスト依存ボタン、対称モード、ツールの alt/sub モード、マスク、スムーズ、ギズモ、ペイントオプションなどのショートカットがあります。
* `Bottom toolbar` - よく使う機能へのショートカットで、以下で説明します。

::: tip 左利きですか？
すべてのツールバーの配置と順序を左右反転できます。[Mirror top bar](interface.md#mirror-top-bar) などの関連オプションを参照してください。
:::

## 下部ツールバー

![](/images/interface_bottom_toolbar.webp)

* `Undo` - 直前の操作を元に戻します
* `Redo` - 直前に取り消した操作をやり直します
* `History` - [History](history.md) メニューで説明されている履歴オプションにアクセスします。
* `Solo` - 現在のオブジェクトのみを表示するか、すべてのオブジェクトを表示するかを切り替えます
* `X-Ray` - 他のすべてのオブジェクトを X 線モードで表示し、現在のオブジェクトをソリッド表示にします。長押しまたは上方向へのスワイプで、X 線モードの色と不透明度を設定できます。
* `Voxel` - [Voxel Remesher](topology.md#voxel-remesher) のボクセルリメッシュボタンへのショートカットです。長押しまたは上方向へのスワイプで、ボクセルリメッシュの品質を設定するショートカットが表示されます。
* `Grid` - グリッドの表示を切り替えます。長押しまたは上方向へのスワイプで、グリッドのオプションが表示されます。
* `Wire` - ワイヤーフレームのオーバーレイ表示を切り替えます。長押しまたは上方向へのスワイプで、ワイヤーフレームのオプションが表示されます。
* `Inspect` - 現在のメッシュに関する追加情報の表示を切り替えます。デフォルトでは UV を表示しますが、長押しまたは上方向へのスワイプで、存在する場合は他のプロパティを検査したり、それを背景またはメッシュ上に表示するかを選択できます。

## 次のステップ

次に何を学ぶべきかは、あなた自身と、何に興味があるかによって変わります。いくつか提案を挙げます：

スカルプトツールについてもっと学びたいですか？[Tools](tools.md) セクションへ進んでください。

モデルをエクスポートしたいですか？あるいはスカルプト用にモデルをインポートしたいですか？スカルプトの画像を作成したいですか？[Files](files.md) セクションへ進んでください。

スカルプトのディテールコントロールについてもっと学びたいですか？[Topology](topology.md) セクションへ進み、マルチレゾリューションやデシメーションについて学んでください。

複数のオブジェクトを扱いたいですか？シンプルなオブジェクトやプリミティブを組み合わせて大きなシーンを作りたいですか？[Scene](scene.md) セクションへ進んでください。

Nomad について *すべて* 学びたいですか？良い選択です！このマニュアルは Nomad のすべてをカバーしており、多くのヒントやテクニックが含まれています。また、上部には優れた検索機能があります。左側のナビゲーションを使って、さらに学んでください。

動画の方が好みであれば、Holger Schönischka 氏が YouTube で Nomad に関する膨大な数の Tips & Tricks を公開しています： https://www.youtube.com/@ProcreateFX/videos
