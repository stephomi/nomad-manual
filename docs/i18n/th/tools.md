## ภาพรวม

เลือกเครื่องมือจาก `Toolbox` ทางขวา และควบคุมด้วย `Tool Controls` ทางซ้าย การตั้งค่าเพิ่มเติมอยู่ในเมนู `Settings` ไอคอนแรกในเมนูมุมขวาบน

เครื่องมือแปรงจะมีตัวควบคุมขนาดและความแรง เครื่องมือเลือกจะมีตัวควบคุมรูปแบบการเลือกหลายแบบ ด้านล่างของตัวควบคุมเครื่องมือมีปุ่มลัดสำหรับคำสั่งที่ใช้บ่อย (Smooth, Mask, Hide, Gizmo, Color, Alpha)

เครื่องมือของ Nomad ถูกแยกสีใน toolbox:

* <span class=brush>**เครื่องมือแปรง**</span> (Clay, Brush, Smooth, Layer, Inflate, Nudge, Stamp, DelLayer)
* <span class=move>**เครื่องมือย้าย**</span> (Move, Drag)
* <span class=mask>**เครื่องมือมาสก์**</span> (Mask, SelMask)
* <span class=paint>**เครื่องมือระบายสี**</span> (Paint, Smudge)
* <span class=flatten>**เครื่องมือปรับแบน**</span> (Flatten, Planar)
* <span class=pinch>**เครื่องมือหนีบ**</span> (Crease, Pinch)
* <span class=selection>**เครื่องมือที่อิงการเลือก**</span> ซึ่งจะวาดมาสก์ 2 มิติก่อน แล้วจึงเกิดการทำงาน (Trim, Split, Project)
* <span class=creation>**เครื่องมือสร้างรูปทรง**</span> (Tube, Lathe, Insert)
* <span class=transform>**เครื่องมือแปลงรูป**</span> (Transform, Gizmo)
* <span class=misc>**เครื่องมือเบ็ดเตล็ด**</span> (Facegroup, Hide, Measure, Select)
* <span class=view>**เครื่องมือมุมมอง**</span>



เครื่องมือจำนวนมากสามารถปรับแต่งพฤติกรรมแปรง แรงกด เท็กซ์เจอร์ ฯลฯ ได้ผ่านเมนู [Stroke](stroke.md) 


### ตัวควบคุมแปรง

แถบเครื่องมือด้านซ้ายมีสไลเดอร์สำหรับรัศมีและความแรง และจากนั้นเป็นตัวควบคุมเฉพาะประเภทของเครื่องมือ อธิบายด้านล่าง

![](/images/tool_left_panel.webp)

::: tip
สไลเดอร์ความแรงของหลายเครื่องมือสามารถเกิน 100% ได้ ลองเล่นดู!
:::

### Sub mode
ปุ่มที่อยู่ใต้สไลเดอร์ความแรงคือปุ่ม `Sub` ป้ายและหน้าที่จะเปลี่ยนไปตามแต่ละเครื่องมือ และเมื่อกดจะเรียกใช้พฤติกรรมทางเลือก ซึ่งมักจะตรงข้ามกับโหมดหลัก เช่น สำหรับ [Paint](#paint) จะเข้าสู่โหมดลบสี (Erase) สำหรับ [Crease](#crease) จะสร้างขอบนูนแทนร่อง ฯลฯ

โดยค่าเริ่มต้นจะทำงานแบบปุ่มกดค้างชั่วคราว คือกดค้างเพื่อเรียกใช้ชั่วคราว เมื่อปล่อยจะปิด หากแตะหนึ่งครั้ง โหมด sub จะถูกเปิดค้างถาวร

### ปุ่มลัด
ด้านล่างของแถบเครื่องมือซ้ายมีปุ่มลัดสำหรับ [Smooth](#smooth), [Mask](#mask), [Hide](#hide), [Gizmo](#gizmo), [Color](painting.md#pbr-sliders), [Alpha](stroke.md#alpha)

โดยค่าเริ่มต้นทั้งหมดทำงานแบบปุ่มกดค้างชั่วคราว คือกดค้างเพื่อเรียกใช้ชั่วคราว เมื่อปล่อยจะปิด หากแตะหนึ่งครั้ง โหมดลัดนั้นจะถูกเปิดค้างถาวร

### ตัวควบคุมการเลือก

เครื่องมือ [Selection Mask](#selection-mask), [Trim](#trim), [Split](#split), [Project](#project), [Facegroup](#facegroup) และ [Hide](#hide) ใช้ตัวควบคุมคล้ายกันในการเลือกพื้นที่ของ mesh

![](/images/tools_shape_selector_panel.webp)


* `Lasso` - รูปทรงวาดอิสระ
* `Polygon` - รูปทรงปิดที่กำหนดด้วยเส้นโค้งและ/หรือเส้นตรง ดู [การแก้ไขรูปทรง](#shape-editing) ด้านล่างสำหรับข้อมูลเพิ่มเติม
* `Curve` - (เฉพาะ Project) - เส้นโค้งวาดอิสระสำหรับการ project
* `Path` - (เฉพาะ Project) - เส้นโค้งที่กำหนดด้วยจุด ดู [การแก้ไขรูปทรง](#shape-editing) ด้านล่างสำหรับข้อมูลเพิ่มเติม
* `Line` - ลากเส้นเพื่อกำหนดระนาบตัด โดยค่าเริ่มต้นจะทำงานกับ mesh ทันที ปิด auto validate หากไม่ต้องการ (กดค้างหรือปัดที่ไอคอนเส้น)
* `Rect` - ลากเส้นทแยงเพื่อกำหนดมุมของสี่เหลี่ยมผืนผ้า กดค้างหรือปัดเพื่อแสดงตัวเลือก auto validate, บังคับให้เป็นสี่เหลี่ยมจัตุรัส และให้คลิกแรกเป็นจุดกึ่งกลางของสี่เหลี่ยม
* `Ellipse` - ลากเส้นทแยงเพื่อกำหนดขนาดวงรี กดค้างหรือปัดเพื่อแสดงตัวเลือก auto validate, บังคับให้เป็นวงกลม และให้คลิกแรกเป็นจุดกึ่งกลางของวงรี
* `Flip` - กลับด้าน shape mask หรือทิศทางของเครื่องมือ project

เครื่องมือส่วนใหญ่มีตัวเลือก auto validate หมายความว่าการทำงานจะเกิดขึ้นทันทีเมื่อวาดรูปทรงเสร็จ เมื่อปิด auto validate จะมีปุ่มสีเขียวปรากฏข้างรูปทรงเพื่อใช้ยืนยันการทำงาน วิธีนี้ช่วยให้คุณแก้ไขรูปทรง ปรับมุมมอง และเมื่อพร้อมใช้รูปทรงแล้วให้กดปุ่มสีเขียว

### การแก้ไขรูปทรง
การแก้ไข Polygon และ Curve ทำงานคล้ายกัน:

* เริ่มต้นด้วยการลากเส้นเพื่อสร้าง 2 จุด จากนั้นลากออกจากกึ่งกลางเส้นเพื่อสร้าง polygon หรือ curve
* คลิกที่จุดเพื่อสลับระหว่างมุมโค้ง (smooth) และมุมหัก (sharp)
* คลิกและลากบนส่วนโค้งหรือส่วนเส้นเพื่อสร้างจุดใหม่
* เพื่อลบจุด ให้ลากจุดไปทับเพื่อนบ้านจนกลายเป็นสีแดง
* ไอคอนถังขยะที่มุมของไอคอน polygon หรือ path จะลบรูปทรง

### ![](/icons/tool_move.webp) Move
พื้นที่ใต้แปรงจะติดไปกับแปรง ทำให้เกิดการเปลี่ยนรูปแบบยืดหยุ่น การเลือกจะถูกเก็บไว้ระหว่างการย้าย ดังนั้นหากคุณย้ายแปรงออกไป แล้วเลื่อนกลับมาที่เดิม จะไม่เห็นการเปลี่ยนรูป

โหมด sub คือ `Normal` และจะย้ายพื้นที่ใต้แปรงไปตาม normal ของผิว

แปรงนี้เหมาะทั้งสำหรับการเปลี่ยนรูปขนาดใหญ่และการปรับเล็กน้อยอย่างระมัดระวัง

#### การตั้งค่า Move

* `Radius (Background)` - ระยะห่างจากขอบโมเดลที่ยังสามารถปั้นได้ มีประโยชน์เมื่อทำงานกับเส้นรอบนอก (silhouette) ของวัตถุ
* `Same-side vertex only` - เพิกเฉยต่อจุดยอด (vertex) ที่ชี้ไปในทิศทางตรงข้ามกับการเปลี่ยนรูป


![](/videos/tool_move.mp4)

### ![](/icons/tool_drag.webp) Drag
พื้นที่ใต้แปรงจะติดไปกับแปรง ทำให้เกิดการเปลี่ยนรูปแบบยืดหยุ่น แตกต่างจากแปรง Move ตรงที่การเลือกจะอัปเดตอย่างต่อเนื่องระหว่างจังหวะ ทำให้สามารถสร้างรูปทรงยาวคล้ายงูได้ โดยเฉพาะเมื่อเปิด Dynamic Topology

โหมด sub คือ `Normal` และจะย้ายพื้นที่ใต้แปรงไปตาม normal ของผิว

แปรงนี้เหมาะสำหรับการเปลี่ยนรูปทรงแบบหลวม ๆ เน้นท่าทาง (gestural)

#### การตั้งค่า Drag

* `Radius (Background)` - ระยะห่างจากขอบโมเดลที่ยังสามารถปั้นได้ มีประโยชน์เมื่อทำงานกับเส้นรอบนอกของวัตถุ
* `Same-side vertex only` - เพิกเฉยต่อจุดยอดที่ชี้ไปในทิศทางตรงข้ามกับการเปลี่ยนรูป

![](/videos/tool_drag.mp4)

### ![](/icons/tool_smooth.webp) Smooth
ทำให้พื้นที่เรียบโดยการเฉลี่ยตำแหน่งจุด เครื่องมือนี้ขึ้นกับความหนาแน่นของ polygon อย่างมาก
หากมี polygon จำนวนมาก การทำให้เรียบจะมีผลน้อยลง

โหมด sub คือ `Relax` ซึ่งจะทำให้ wireframe เรียบแต่พยายามรักษารายละเอียดรูปทรง

#### การตั้งค่า Smooth

![](/images/tool_smooth_settings.webp)

##### Facegroup

* `Relax` - ทำให้ขอบของ facegroup เรียบ ใช้ความแรงมากกว่า 100% เพื่อทำให้ขอบเรียบอย่างรวดเร็ว `Auto` จะทำให้เรียบเฉพาะเมื่อเปิด facegroup preview `Off` ปิด `On` เปิด

##### Vertex
* `Sticky vertex on border` - สำหรับ mesh ที่มีขอบเปิด เช่น plane สามารถทำให้มุมเรียบจนหายไปได้ การเปิดตัวเลือกนี้จะล็อกขอบเปิดไว้
* `Relax` - เหมือนโหมด relax ทางแถบเครื่องมือซ้าย
* `Stable smoothing` - พยายามทำให้การทำให้เรียบไม่ขึ้นกับ topology ใช้ได้ดีที่สุดกับ topology ที่ความหนาแน่นแปรผันและค่าความแรงสูง

##### Painting
* `Screen Smoothing` - ใช้ตัวเลือกนี้เพื่อให้การทำให้เรียบไม่ขึ้นกับ topology แม้ที่จำนวน polygon สูง
* `Screen samples` - คุณภาพของการทำให้เรียบ ค่ายิ่งสูงยิ่งเรียบแต่ช้าลง

::: tip
ความหนาแน่น polygon ที่สูงขึ้นอาจต้องเพิ่มความแรงเกิน 100% ค่าสูงมาก (300%, 500%) ก็ใช้เป็นเครื่องมือปั้นได้ดี บังคับให้พื้นที่แบนและเรียบอย่างรวดเร็วใต้แปรง เหมือนใช้ค้อนทุบดินน้ำมัน!
:::

### ![](/icons/tool_mask.webp) Mask
เครื่องมือนี้ใช้มาสก์จุดยอด จุดยอดที่ถูกมาสก์จะถูกป้องกันจากการปั้นหรือการระบายสี

โหมด sub คือ `Unmask` และจะลบมาสก์ในบริเวณที่ระบาย

คล้ายกับการเลือกในโปรแกรมวาดภาพ 2 มิติ มาสก์สามารถระบายด้วยแปรง หรือสร้างด้วยการเลือกแบบรูปทรง ทำให้เบลอหรือคมขึ้นได้

ต่างจากโปรแกรม 2 มิติ มาสก์ยังสามารถสร้างจาก facegroup และใช้สร้าง geometry ใหม่ด้วยการ extrusion/extraction ได้

![](/videos/tool_mask1.mp4)

จะมีแถบเครื่องมือปรากฏด้านบนของ viewport พร้อมตัวควบคุมเพิ่มเติม

![](/images/tool_mask_toolbar.webp)

คุณสามารถใช้ท่าทางแบบ ZBrush ได้ขณะกดปุ่ม quick masking ที่แถบลัดมุมล่างซ้าย:
| Action  | Gesture (hold lower-left shortcut) |
| :-----: | :--------------------------------: |
| Invert  | แตะบนพื้นหลัง                      |
| Clear   | ลากบนพื้นหลัง                      |
| Blur    | แตะบนพื้นที่ที่ถูกมาสก์            |
| Sharpen | แตะบนพื้นที่ที่ไม่ถูกมาสก์         |


#### การตั้งค่า Mask
![](/images/tool_mask_settings.webp)

![](/videos/tool_mask2.mp4)

* `Preview` - เมนูการตั้งค่า Mask ใช้หลัก ๆ เพื่อสร้าง geometry จากมาสก์ ดังนั้นค่าเริ่มต้นจึงแสดงตัวอย่างว่ารูปทรงใหม่จะเป็นอย่างไร คุณสามารถเลือกไม่แสดงตัวอย่าง แสดงตัวอย่างแบบ extract แบบ split และเลือกให้ geometry นี้แสดงในโหมด x-ray หรือไม่

##### Thickness
* `Height` - ความสูงของรูปทรงที่ extract ไอคอน Plus/Minus ให้คุณวนระหว่าง extrusion ออกด้านนอก ด้านใน หรือกึ่งกลาง
* `Height/Height+Mask` - สลับระหว่างให้ความสูงคงที่ หรือให้ส่วนที่เบลอของมาสก์มีผลต่อความสูง ทำให้ได้รูปทรงที่นุ่มและความสูงแปรผัน

##### Smoothness
เมื่อเปิดจะทำให้ขอบของรูปทรงที่ extract เรียบ ทำงานได้ดีกับจำนวน polygon สูง
* `Iterations` - ปริมาณการทำให้เรียบ ค่าสูงจะให้ขอบโค้งเรียบมาก แต่จะเริ่มเบี่ยงจากรูปทรงมาสก์
* `All/Sharp border/Borders only` - การทำให้เรียบสามารถทำได้ทุกทิศทาง (ด้านข้างและด้านบน) หรือทำให้ด้านบนและด้านข้างเรียบแต่คงขอบคม หรือทำเฉพาะขอบโดยไม่แตะพื้นผิวด้านบน

##### Edge loop (side)
* `Auto Edge-loop (side)` - คำนวณจำนวน division ด้านข้างของรูปทรงที่ extract เพื่อให้ polygon เป็นสี่เหลี่ยมและสอดคล้องกับ polygon ของพื้นที่ที่ถูกมาสก์ เมื่อปิด คุณสามารถตั้งจำนวน edge loop เองด้วยสไลเดอร์ edge loop

----

##### Extract
* `Extract` - สร้าง geometry ที่ extract
* `Closing action` - วิธีการปิดด้านหลังของ extract 'None' จะคัดลอก polygon ที่ถูกมาสก์ไปยังรูปทรงใหม่ 'Fill' จะทำเช่นเดียวกันและพยายามปิดผิวด้านหลัง 'Shell' จะ extrude ตามค่า 'thickness' และเป็นค่าเริ่มต้น

::: tip

หาก preview อยู่ในโหมด 'Extract' พร้อม 'X-ray' เปิดอยู่ การกดปุ่ม Extract อาจทำให้สับสนในตอนแรก เพราะเมนูยังเปิดอยู่ มันจะพยายามแสดงตัวอย่างการ extract บนรูปทรงใหม่ และทำ xray ผิวเดิม แต่เนื่องจากไม่มีมาสก์บนผิวใหม่ จึงไม่สามารถแสดงตัวอย่างได้ และ Nomad จะแจ้งเตือน 'Nothing to Extract!'

นี่เป็นเรื่องปกติ ปิดเมนูการตั้งค่า mask เพื่อดูรูปทรงใหม่และผิวเดิม และเลือกผิวเดิมอีกครั้งหากต้องการล้างมาสก์หรือวาดมาสก์ใหม่
:::

##### Split
* `Split` - จะ extract ทั้งส่วนที่ถูกมาสก์และไม่ถูกมาสก์เป็นรูปทรงใหม่
* `Closing action (masked)` - วิธีการปิดด้านหลังของส่วนที่ถูกมาสก์ 'None' จะคัดลอก polygon ที่ถูกมาสก์ไปยังรูปทรงใหม่ 'Fill' จะทำเช่นเดียวกันและพยายามปิดผิวด้านหลัง 'Shell' จะ extrude ตามค่า 'thickness' และเป็นค่าเริ่มต้น
* `Closing action (unmasked)` - วิธีการปิดด้านหลังของส่วนที่ไม่ถูกมาสก์ 'None' จะคัดลอก polygon ที่ถูกมาสก์ไปยังรูปทรงใหม่ 'Fill' จะทำเช่นเดียวกันและพยายามปิดผิวด้านหลัง 'Shell' จะ extrude ตามค่า 'thickness' และเป็นค่าเริ่มต้น
* `Sync border` - ทำให้ขอบระหว่างรูปทรงที่ extract จากส่วนมาสก์และไม่มาสก์อยู่ใกล้กัน เมื่อปิด เนื่องจากการ shell จะ extrude แต่ละหน้าไปตาม normal ของมัน อาจเกิดช่องว่างระหว่างรูปทรงได้

##### Carve
* `Carve` - ในโหมดเริ่มต้น ทำงานเหมือนคุณใช้ Trim เจาะเข้าไปในผิวตามค่า 'thickness' คล้ายการตัดเปลือกส้มออกชิ้นหนึ่ง

### ![](/icons/tool_flatten.webp) Flatten
ทำให้พื้นที่แบนโดย project จุดลงบนระนาบเฉลี่ย

โหมด sub คือ `Fill` และจะกำหนดระนาบจากจุดที่สูงที่สุด และดึงจุดขึ้น

#### การตั้งค่า Flatten

::: tip
เมื่อทำงานในพื้นที่ที่มีความโค้งสูง เช่น พยายามปรับแก้มให้แบนแต่เครื่องมือไปกระทบด้านข้างจมูก ลองสร้างมาสก์เพื่อป้องกันพื้นที่ที่ไม่ต้องการให้ Flatten มีผล

### ![](/icons/tool_trim.webp) Trim
เครื่องมือ Trim ทำงานโดยตัดส่วนหนึ่งของ mesh ออก และมีตัวเลือกในการจัดการช่องว่างที่เหลือ ใช้ [ตัวควบคุมการเลือก](#selection-controls) เพื่อกำหนดพื้นที่ตัด

::: tip
เนื่องจากเครื่องมือนี้ project จากกล้อง คุณจะได้รับคำเตือนหากกล้องอยู่ในโหมด perspective

ในโหมด orthographic รอยตัดผ่าน mesh จะขนานกับมุมมอง ซึ่งมักเป็นสิ่งที่ผู้ใช้คาดหวัง เมื่อใช้กล้อง perspective รอยตัดด้านไกลของวัตถุจะดูต่างจากด้านใกล้
:::

#### การตั้งค่า Trim

* `Stroke painting` - หากเปิด paint ในเมนู paint พื้นที่ที่ปะจะถูกเติมด้วยสีที่เลือกอยู่
* `Boolean` - ปิดรูของการ trim ด้วยบริเวณ quad poly พื้นที่ที่ปะจะเป็นระนาบแบน
* `Legacy` - ปิดรูของการ trim ด้วยบริเวณสามเหลี่ยม พื้นที่ที่ปะจะเป็นระนาบแบน
* `Fill` - ปิดรูด้วยบริเวณสามเหลี่ยม พื้นที่ที่ปะจะเป็นพื้นผิวโค้งคล้ายฟิล์มฟองสบู่
* `None` - ไม่ปิดรู
* `Boolean Detail Shape` - ขนาดโดยประมาณของ quad และสามเหลี่ยมที่ใช้ด้านรูปทรงของการ trim
* `Boolean Detail Hole` - ขนาดโดยประมาณของสามเหลี่ยมหรือ polygon ที่ใช้ปิดรูของการ trim
* `Legacy Detail` - ขนาดโดยประมาณของสามเหลี่ยมที่ใช้ใน trim
* `Legacy Hole smoothing` - ปริมาณการผ่อนคลาย (relax) สามเหลี่ยมในพื้นที่ที่ปะ
* `Legacy Threshold espilon` - ค่าที่สามารถปรับเพื่อปรับปรุงอัลกอริทึมการปิดรูแบบ legacy
* `Fill Detail` - ขนาดโดยประมาณของสามเหลี่ยมที่ใช้ใน trim

### ![](/icons/tool_layer.webp) Layer
ยกพื้นผิวขึ้น แต่จำกัดความสูง

หากคุณไม่ยกปากกาและระบายซ้ำบนพื้นที่เดิม Layer จะยกขึ้นถึงความสูงหนึ่งแล้วไม่สูงไปกว่านั้น ต่างจากเครื่องมืออื่นที่สะสมความสูงต่อไปเรื่อย ๆ

โปรดสังเกตว่าโดยค่าเริ่มต้นขีดจำกัดนี้จะถูกตั้งต่อหนึ่งจังหวะเท่านั้น! หากเริ่มจังหวะใหม่ มันจะสร้างต่อจากความสูงพื้นผิวใหม่

หากต้องการตั้งความสูงสูงสุดคงที่ข้ามหลายจังหวะ ให้ใช้เครื่องมือ Layer ร่วมกับระบบ [Layers](layers.md) ของ Nomad

สร้าง layer แล้วใช้เครื่องมือนี้ ความสูงสูงสุดจะถูกตั้งจาก layer ดังนั้นคุณสามารถปัดหลายจังหวะและความสูงจะเท่าเดิมเสมอ

`Sub` จะใช้ความลึกต่ำสุด สร้างร่อง

#### การตั้งค่า Layer

* `Use layer data` - เมื่อเปิด และมี layer ถูกเลือก ใช้ข้อมูล layer เพื่อกำหนดความสูงสูงสุด

### ![](/icons/tool_stamp.webp) Stamp

#### แถบเครื่องมือซ้ายของ Tube

![](/images/tool_tube_left_toolbar.webp)

แถบเครื่องมือซ้ายมีตัวเลือกดังนี้:

* `Sync` - หาก Tube ปัจจุบันเป็น instance และมี child node ของ Tube ที่ต่างกันระหว่าง instance การกดจะ sync ให้ตรงกัน
* `Snap` - เมื่อเปิด โหมด curve และ path จะ snap ไปบนวัตถุอื่นขณะวาด เมื่อปิด จุดแรกจะ snap จากนั้นท่อที่เหลือจะถูกวาดที่ความลึกนั้น มีเมนูย่อย:
    * `Offset` - ตั้งความลึกของ snap; 0% จะให้กึ่งกลางหน้าตัดท่อ snap กับผิว ค่าเป็นบวกจะยกออกจากผิว ค่าลบจะกดลงไปในผิว
* `Curve` - วาดท่อแบบ freeform มีเมนูย่อย:
    * `Auto-validate` - จะสร้างท่อทันทีเมื่อจังหวะเสร็จ เมื่อปิด จะมีวงกลมสีเขียวข้าง path ให้กดเพื่อยืนยัน หรือใช้ลิงก์ `Reset` ในเมนูนี้เพื่อลบ path
    * `Closed` - ทำให้ท่อเป็นวงปิด
    * `Screen` - ใช้ได้เมื่อปิด Auto-validate เท่านั้น เมื่อเปิด path จะ 'ปัก' กับหน้าจอ ทำให้คุณขยับมุมมองและวัตถุได้โดย path อยู่กับที่ เมื่อปิด path จะเป็นส่วนหนึ่งของฉาก 3 มิติ และจะขยับตามกล้องและวัตถุ
    * `Accuracy` - จำนวนจุดโค้งที่ใช้แปลงเส้นที่วาดเป็นท่อ 0% จะใช้จุดน้อยสุดแต่พลาดความโค้งเล็ก ๆ 100% จะละเอียดมากและใช้จุดจำนวนมาก
* `Path` - สร้างท่อโดยคลิกกำหนดจุดโค้ง แตะวงกลมสีเขียวเพื่อยืนยัน path มีเมนูย่อย:
    * `B-spline` - วิธีวาดเส้นโค้งแบบอื่นที่จุดโค้งมักไม่อยู่บนเส้นตรง ๆ แต่ให้ผลลัพธ์ที่เรียบกว่าวิธีปกติ
    * `Closed` - ทำให้ท่อเป็นวงปิด
    * `Screen` - เมื่อเปิด path จะ 'ปัก' กับหน้าจอ ทำให้คุณขยับมุมมองและวัตถุได้โดย path อยู่กับที่ เมื่อปิด path จะเป็นส่วนหนึ่งของฉาก 3 มิติ และจะขยับตามกล้องและวัตถุ

หากคุณเลือก primitive จากแถบเครื่องมือซ้าย การคลิก-ลากบนพื้นผิวใด ๆ จะวาง primitive ที่ตำแหน่งคลิก การลากจะกำหนดขนาด เมื่อปล่อยลาก Insert จะสลับไปโหมด [Transform](#transform)

ในโหมด Instance การลากจะสร้าง instance ใหม่และเลื่อนไปตามพื้นผิว ขนาดจะถูกคัดลอกจากรูปทรงแรก ทำให้คุณวาง instance ขนาดเท่ากันจำนวนมากบนพื้นผิวอื่นได้

คุณไม่จำเป็นต้อง insert แค่ primitive! เลือก *วัตถุใด ๆ* ใน outliner หาก Insert อยู่ในโหมด Instance หรือ Clone คุณสามารถลากสำเนาของวัตถุที่เลือกไปบนพื้นผิวอื่นได้

หากวัตถุมี pivot แบบกำหนดเอง จะใช้เป็นจุดยึด ดูวิดีโอด้านล่าง

### ![](/icons/tool_transform.webp) Transform
ย้าย/หมุน/สเกลโมเดลโดยตรงด้วยนิ้ว 1 และ 2 นิ้ว มักใช้บนพื้นผิวของวัตถุอื่น

เครื่องมือนี้ควบคุมด้วยแถบเครื่องมือซ้าย และมี 5 ปุ่ม:

* `Snap` - snap โมเดลไปบนพื้นผิวอื่น
* `Group` - หากวัตถุที่เลือกมีทั้งวัตถุและ instance ผสมกัน ตัวเลือกนี้จะกำหนดพฤติกรรมของเครื่องมือ
* `Move` - ลากด้วยนิ้วเดียวเพื่อย้ายวัตถุ เมื่อเปิด snap จะเลื่อนวัตถุไปตามพื้นผิวใต้ปลายนิ้ว
* `Rotate` - ลากด้วยนิ้วเดียวเพื่อหมุนวัตถุ เมื่อเปิด snap จะหมุนรอบ normal ของพื้นผิวใต้ปลายนิ้ว
* `Scale` - ลากด้วยนิ้วเดียวเพื่อสเกลวัตถุ

Transform สามารถใช้สองโหมดได้อย่างรวดเร็วด้วยนิ้วสองนิ้ว:

* ลากวัตถุเพื่อวางตำแหน่ง หยุดลากแตอย่าชูนิ้วแรกออกจากหน้าจอ
* แตะนิ้วที่สองบนหน้าจอขณะนิ้วแรกยังแตะอยู่ เมื่อลากนิ้วที่สอง วัตถุจะสเกล
* ยกนิ้วที่สอง แล้วลากนิ้วแรกต่อ วัตถุจะกลับไปโหมด Move อีกครั้ง

คุณยังสามารถเปลี่ยนโหมดที่สองด้วยท่าทางแตะนิ้วที่สอง:

* ลากวัตถุเพื่อวางตำแหน่ง หยุดลากแต่ยังไม่ยกนิ้วแรก
* แตะนิ้วที่สองขณะกดนิ้วแรกค้าง
* เครื่องมือจะสลับไปโหมดหมุน ลากนิ้วแรกเพื่อกำหนดการหมุน
* แตะนิ้วที่สองอีกครั้ง เครื่องมือจะสลับกลับไปโหมด Move

นี่เป็น workflow ที่รวดเร็วสำหรับการ clone วัตถุบนอีกวัตถุหนึ่ง เช่น หินบนภูมิประเทศ สังเกตว่าปุ่ม clone อยู่ในแถบเครื่องมือซ้ายเพื่อเข้าถึงง่าย:

* ใช้เครื่องมือ Transform เพื่อย้ายหินก้อนหนึ่งไปตำแหน่ง
* ปล่อย แล้วกดปุ่ม clone
* ย้ายหินก้อนนี้ หมุน/สเกลตามต้องการ
* ปล่อย แล้วกดปุ่ม clone
* ย้ายหินก้อนนี้ ทำซ้ำ

### ![](/icons/tool_gizmo.webp) Gizmo
เครื่องมือนี้ให้คุณย้าย หมุน และสเกลวัตถุ รวมถึงเปลี่ยน pivot ของวัตถุ

ตัวจับ (handle) ใน viewport มีคุณสมบัติดังนี้:

* `Move` - ลากบนเส้น+ลูกศรเพื่อย้ายตามแกน X/Y/Z ลากบนจุดสีพีชตรงกลาง gizmo เพื่อย้ายอิสระใน screen-space คลิกที่สี่เหลี่ยมสีแดง เขียว น้ำเงิน เพื่อย้ายบนระนาบ X/Y/Z
* `Rotate` - ลากบนวงกลมสีแดง/เขียว/น้ำเงินเพื่อหมุนตามแกน X/Y/Z ลากบนทรงกลมด้านในวงกลมเพื่อหมุนอิสระ
* `Scale`- ลากบนจุดด้านนอกสีแดง/เขียว/น้ำเงินเพื่อสเกลตามแกน X/Y/Z ลากบนกรวยด้านนอกสีแดง/เขียว/น้ำเงินเพื่อสเกลบนระนาบ X/Y/Z ลากบนวงกลมสีพีชด้านนอกเพื่อสเกลแบบ uniform

![](/images/tool_gizmo.webp)

#### Node และ vertex 

ทุกวัตถุใน Nomad ประกอบด้วย node และ vertex:

* `Node` - 'ด้ามจับ' ของวัตถุ ซึ่งเก็บค่าการย้าย หมุน สเกล เรียกว่า transformation matrix
* `Vertices` - จุดที่กำหนดพื้นผิวของวัตถุ เก็บข้อมูลตำแหน่งและสี

หากคุณมีกล่องง่าย ๆ ที่ประกอบด้วย 8 vertex คุณสามารถย้ายมันได้โดยแก้ transformation matrix หรือแก้ตำแหน่ง vertex เมื่อปั้นคุณมักต้องการแก้ vertex เมื่อย้ายวัตถุด้วย gizmo คุณมักต้องการแก้ node Nomad ให้คุณทำได้ทั้งสองแบบ

#### แถบเครื่องมือเมนูซ้าย

แถบเครื่องมือซ้ายควบคุมว่า gizmo จะทำงานกับ node หรือ vertex รวมถึงฟังก์ชันอื่น ๆ:

* `Clone` - สร้างสำเนาอิสระของวัตถุปัจจุบัน ซึ่งสามารถลากออกไปด้วย gizmo
* `Instance` - สร้าง instance ของวัตถุปัจจุบัน วัตถุจะเชื่อมกัน ดังนั้นการปั้นบนวัตถุหนึ่งจะปรากฏบน instance ทั้งหมด
* `Group/Object/Vertex/Auto` - กำหนดว่า gizmo จะมีผลกับ node หรือ vertex ของวัตถุ ค่าเริ่มต้น 'auto' จะพยายามเดาให้เหมาะสม หากมีวัตถุหลายชิ้นเป็นลูกกันในลำดับชั้น 'Object' จะย้ายเฉพาะวัตถุปัจจุบัน วัตถุลูกจะไม่ขยับ Gizmo ยังสามารถคำนึงถึงมาสก์และสมมาตรได้
* `Pin` - โดยค่าเริ่มต้น gizmo จะย้ายไปที่ pivot ของวัตถุที่เลือก เมื่อเปิด pin gizmo จะอยู่ที่ตำแหน่งเดิม
* `Align` - สลับระหว่างให้ pivot จัดแนวกับวัตถุปัจจุบัน หรือจัดแนวกับ world
* `Snap rotation` - เปิดการ snap ค่าการหมุนเป็นขั้น ๆ ค่า snap จะแสดงและแก้ไขได้เมื่อเปิด snap
* `Snap translation` - เปิดการ snap ค่าการย้ายเป็นขั้น ๆ ค่า snap จะแสดงและแก้ไขได้เมื่อเปิด snap
* `Pivot` - เมื่อเปิด คุณสามารถย้ายและหมุน gizmo โดยไม่ย้ายวัตถุ มีเมนูเพิ่มเติมอธิบายด้านล่าง

#### Pivot
เมื่อเปิดโหมด pivot จะมีเมนูแสดงเพื่อให้เปลี่ยน pivot ได้อย่างรวดเร็ว:

**Reset** 
* `Center` - ย้าย pivot ไปกึ่งกลางวัตถุ
* `Bottom` - ย้าย pivot ไปด้านล่างของวัตถุ
* `Align` - รีเซ็ตการหมุนให้จัดแนวกับ world
* `Mask` - ย้าย pivot ไปกึ่งกลางของบริเวณที่ไม่ถูกมาสก์

**On Tap**
* `None` - ไม่ทำอะไรเมื่อแตะวัตถุ
* `Normal` - ย้ายและหมุน pivot ให้จัดแนวกับจุดที่แตะบนผิว
* `First` - ย้าย (แต่ไม่หมุน) pivot ไปยังจุดที่แตะบนผิว
* `Medial` - ย้าย pivot ไปกึ่งกลางของวัตถุใต้บริเวณที่แตะบนผิว

##### Matrix 
* ![](/icons/target.webp) `Move origin` - ย้ายวัตถุเพื่อให้ pivot อยู่ที่จุดศูนย์กลางของฉาก (origin)
* ![](/icons/bake.webp)  `Bake` - แช่แข็งวัตถุไว้ที่ตำแหน่งปัจจุบัน และตั้งค่า translate/rotate เป็น 0 scale เป็น 1
* ![](/icons/bake.webp) -> ![](/icons/tool_gizmo.webp) `Bake Pivot` - ทำให้ค่า matrix สอดคล้องกับตำแหน่งของ handle gizmo ในโลกจริง
* ![](/icons/reset.webp) `Reset` - ทางลัดที่ตั้งค่า translate เป็น 0 rotate เป็น 0 scale เป็น 1 และย้าย/หมุนวัตถุให้ตรงตามนั้น

::: tip Bake vs bake to pivot
สร้างกล่อง เลือกเครื่องมือ Gizmo เปิดและปักหมุดเมนูการตั้งค่า โดยค่าเริ่มต้นค่า translate และ rotate เป็น 0 scale เป็น 1

เปิดโหมด pivot ย้าย handle ไปด้านหนึ่ง ปิดโหมด pivot Pivot เปลี่ยนแล้ว แต่สังเกตว่า translate ยังเป็น 0

หากต้องการเห็นว่าจริง ๆ pivot อยู่ที่ไหน ให้คลิก `Bake Pivot` ตอนนี้ค่า translate จะอัปเดต สังเกตว่ากล่องไม่ขยับระหว่างการทำงานนี้ เช่นเดียวกับในโหมด pivot

ย้ายและหมุนกล่องไปที่อื่น จากนั้นแตะ `Move Origin` กล่องจะย้ายเพื่อให้ pivot อยู่ที่ศูนย์กลางโลก แต่คงการหมุนไว้

คลิก `Reset` การหมุนจะถูกตั้งเป็น 0 ด้วย

ย้ายและหมุนกล่องอีกครั้ง คราวนี้คลิก `Bake` Pivot จะอยู่ที่เดิม กล่องอยู่ที่เดิม แต่ค่า translate และ rotate ถูกตั้งเป็น 0

ลองฝึกหลาย ๆ ครั้ง! ทำความเข้าใจว่าค่า pivot ถูกซ่อน Nomad จัดการให้คุณ แต่หากต้องการตั้ง pivot ให้ตรงกับตำแหน่งจริงในอวกาศ `Bake Pivot` จะทำให้
:::

* `Translation` - ค่า translate แกน X, Y, Z
* `Rotation` - ค่า rotate แกน X, Y, Z
* `Scale` - ค่า scale แบบ uniform หากเปิด หรือค่า scale แกน X, Y, Z หากปิด
* `Uniform scale` - สลับความสามารถในการสเกลแบบ uniform หรือแยกตามแกน

-----

* `Compact` - สลับ layout ของ gizmo ให้ handle เสริมอยู่ด้านนอกหรือด้านในทรงกลมหมุน
* `Widget size` - ขนาดของ gizmo
* `Thickness` - ความหนาของเส้นบน gizmo
* `Opacity` - ความทึบของ gizmo
* `Hide on interaction` - สลับว่าควรซ่อน gizmo ชั่วคราวเมื่อถูกลากหรือไม่

-----

* `Tangent roll threshold` - ควบคุมพฤติกรรม UI การหมุนเมื่อลากบนวงกลมเพื่อหมุนแกน X/Y/Z หากค่านี้เป็น 0 การหมุนจะเหมือนหมุนปุ่มหมุน (dial) ลาก gizmo เป็นวงกลม หากเป็น 90 การหมุนจะเหมือนดึงเชือกโยโย่ ลากเป็นเส้นตรงเข้าหาหรือออกจากจุดที่คลิกครั้งแรก ค่าระหว่าง 0 ถึง 90 จะผสมทั้งสองแบบ ต่ำกว่าค่าจะเป็นการลากเส้นตรง สูงกว่าจะเป็นการลากเป็นวงกลม
* `Numerical input` - เมื่อเปิด การแตะครั้งเดียวบน gizmo จะเปิดหน้าต่างให้กรอกค่าตัวเลขที่แน่นอนสำหรับแกนของ widget ที่แตะ

### ![](/icons/tool_faceGroup.webp) Facegroup

Facegroup ช่วยให้คุณจัดระเบียบวัตถุให้เป็นพื้นผิวคนละสี คุณสามารถใช้กลุ่มเหล่านี้ได้หลายวิธีใน Nomad:

* วิธีเลือกมาสก์อย่างรวดเร็ว
* ซ่อน/แสดงส่วนต่าง ๆ ของวัตถุ
* จัดระเบียบวัตถุโดยไม่ต้องแยกเป็นชิ้นย่อย
* กำหนดบริเวณ UV
* ช่วยกำหนดทิศทางให้ quad remesher
* ให้การควบคุมเพิ่มเติมสำหรับเครื่องมืออย่าง Smooth

#### แถบเครื่องมือซ้ายของ Facegroup

* `Patch ` - แสดง Facegroup ที่มีอยู่เป็น patch Patch ที่ไม่ได้ใช้สามารถลบได้ แตะ patch เพื่อเปลี่ยนชื่อหรือสี ไอคอนเครื่องหมายบวกใช้สร้าง patch ใหม่
* `Dot` - ระบายบนวัตถุเพื่อกำหนด Facegroup เมื่อเปิด '+ Face Group' ทุกจังหวะใหม่จะสร้าง Facegroup ใหม่โดยอัตโนมัติ เหมาะสำหรับกำหนดบริเวณอย่างรวดเร็ว การแตะหนึ่งครั้งจะ flood fill บริเวณที่เลือก สไลเดอร์กำหนดรัศมีของจุด
* `Relax` - ทำให้ขอบของ Facegroup เรียบ มีประโยชน์มากสำหรับกำหนดขอบที่สะอาดเพื่อ quad remeshing หรือกำหนดแผงสำหรับงาน hard surface สไลเดอร์ควบคุมรัศมีและความแรงของการ Relax
* `Shape selector` - สร้าง Facegroup ด้วยรูปทรงแทนแปรง ผ่าน `Lock+Radius`, `Lasso`, `Polygon`, `Rect` และ `Ellipse` ดู [Shape Selector](#shape-selector) สำหรับข้อมูลเพิ่มเติม
* `Auto-pick` - เมื่อเปิด จะเลือก patch ที่จังหวะเริ่มต้น และใช้ patch นั้นสำหรับส่วนที่เหลือของจังหวะ มีประโยชน์มากสำหรับเก็บขอบเขต Facegroup หาก Facegroup ล้ำออกไปไกลเกินไป ให้เปิด Auto-pick เริ่มจังหวะจากบริเวณที่ patch ถูกต้อง แล้วลากไปถึงขอบเพื่อกำหนด patch ที่ถูกต้องใหม่

### ![](/icons/tool_hide.webp) Hide
ซ่อนหรือแยกส่วนของวัตถุ

โหมดหลักควบคุมจากเมนูซ้าย:

* `Dot` - ระบายบนวัตถุเพื่อซ่อนส่วนของวัตถุ
* `Shape selector` - ซ่อนด้วยรูปทรงแทนแปรง ผ่าน `Lasso`, `Polygon`, `Line`, `Rect` และ `Ellipse` ดู [Shape Selector](#shape-selector) สำหรับข้อมูลเพิ่มเติม
* `Show` - กลับด้านการทำงาน ทำให้โหมดที่เลือกแสดงแทนที่จะซ่อนส่วนของวัตถุ

#### Quad remesher top toolbar
![](/images/tool_quadremesher_toolbar.webp)

A toolbar will appear at the top of the viewport with extra controls:


* `<Count>` - คลิกเพื่อเริ่มกระบวนการ quad remesher ตัวเลขนี้บอกจำนวน quad เป้าหมายที่จะถูกสร้าง

#### Quad remesher settings menu

การตั้งค่าส่วนใหญ่มีให้ในทูลบาร์ด้านบนแล้ว

* `<Count>, Half, Same` - เหมือนกับปุ่ม Remesh, Half, Same ในทูลบาร์ด้านบน
* `Target Quads` - เหมือนกับปุ่ม `Quads` ในทูลบาร์ด้านบน
* `Adaptive quad count` - สลับเพื่อเปิดการใช้ quad ขนาดเล็กในบริเวณที่มีความโค้งสูง และ quad ขนาดใหญ่ในบริเวณที่มีความโค้งต่ำ
* `Adaptive size` - ตั้งค่าปริมาณของความเป็น adaptive ที่ 100% จะอนุญาตให้มีขนาด adaptive สูงสุด ที่ 0% quad จะมีขนาดสม่ำเสมอ
* `Auto-Detect Hard Edges` - สลับเพื่อปรับ layout ของ quad remesh ให้ตามพื้นผิวที่คมชัดได้ดีขึ้น
* `Density (painting)` - เหมือนกับปุ่ม `Density (painting)` ในทูลบาร์ด้านบน
* `Reproject Vertex` - สลับเพื่อฉาย (reproject) layout ใหม่กลับไปยัง mesh ต้นฉบับ
* `Face Group` - เหมือนกับปุ่ม `Face Group` ในทูลบาร์ด้านบน
* `Relax Slider` - แถบเลื่อนทางลัดเพื่อผ่อนคลายขอบของ facegroup

::: tip

สูตรสำหรับได้ quad remesh ที่ดีและมีอาร์ติแฟกต์น้อยที่สุด:

* ทำ Facegroup ให้กับ mesh เพื่อกำหนดทิศทางการไหลของ quad ที่คุณต้องการ
* ใช้ Facegroup relax เพื่อให้ขอบของ facegroup เรียบเนียน
* Decimate สิ่งนี้จะช่วยให้คุณไม่มีหน้าโพลิกอนที่บางหรือบิดเบี้ยวบนขอบของ facegroup ในการตั้งค่า decimate ให้แน่ใจว่าได้เปิดใช้ facegroup ไว้ ซึ่งจะทำให้ขอบสามเหลี่ยมตามขอบของ facegroup อย่างแม่นยำ 

ในตัวเลือก quad remesh ให้แน่ใจว่าได้ปิด relax (เพราะคุณได้ relax mesh ไปแล้ว) และคุณควรได้ผลลัพธ์ที่ดี

:::
