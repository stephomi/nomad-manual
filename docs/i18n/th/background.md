# ![](/icons/image.webp) พื้นหลัง {#background}

เมนูนี้ใช้ควบคุมสีพื้นหลังของ Nomad รวมถึงรูปภาพอ้างอิงที่ต้องการใช้ด้วย

![](/images/background_overview.webp)

## พื้นหลัง {#type}
![](/images/background_selector.webp)

มีตัวเลือกพื้นหลังของวิวพอร์ตอยู่สามแบบ

* `Environment` - แสดงแผนที่สภาพแวดล้อม (environment map) ที่เลือกไว้ใน [Shading](shading) และยังสามารถควบคุมเพิ่มเติมได้ด้วยตัวเลื่อน Blur และ Exposure (ความสว่าง)
* `Color` - สีเรียบหนึ่งสีที่คุณเลือกได้
* `Gradient` - ไล่ระดับสีจากด้านบนลงด้านล่าง ตัวเลื่อน Factor ใช้กำหนดจุดกึ่งกลางของเกรเดียนต์  

## รูปภาพอ้างอิง {#reference}

![](/images/background_reference.webp)

คุณสามารถเพิ่มรูปภาพที่ต้องการลงบนพื้นหลังเพื่อใช้เป็นภาพอ้างอิงได้
คุณสามารถเปลี่ยนตำแหน่งและสเกลได้ เช่น หากต้องการย้ายไปไว้ที่มุมหน้าจอ

### ![](/icons/tool_transform.webp) ทรานส์ฟอร์ม {#transform}

ปุ่มนี้จะให้คุณวางตำแหน่งรูปภาพอ้างอิงแบบโต้ตอบ ใช้นิ้ว 2 นิ้วในการเลื่อน (pan), ปรับสเกล และหมุนรูปภาพอ้างอิงให้อยู่ในตำแหน่งที่ต้องการ ตำแหน่งสุดท้ายจะสะท้อนให้เห็นในตัวเลื่อนด้านล่าง:

* `Overlay` - ที่ 0% รูปภาพอ้างอิงจะอยู่ด้านหลังวัตถุของคุณเสมอ ที่ 100% จะอยู่ด้านหน้า
* `Opacity` - ความโปร่งใสของรูปภาพ
* `Position` - ตำแหน่งแกน X และ Y ของรูปภาพ
* `Rotation` - การหมุนของรูปภาพ
* `Scale` - สเกลของรูปภาพ


::: tip

กล้องและรูปภาพอ้างอิงเชื่อมโยงกัน

หมายความว่า หากคุณตั้งค่ารูปภาพอ้างอิงให้ตรงกับงานปั้นของคุณ แล้วสร้างกล้องจาก [เมนูกล้อง](camera) ตำแหน่ง การหมุน และสเกลของรูปภาพอ้างอิงจะถูกบันทึกไปกับกล้องด้วย คุณสามารถปิดรูปภาพอ้างอิง หมุนไปมุมมองอื่นได้ และเมื่อคุณมองผ่านกล้องอีกครั้ง รูปภาพอ้างอิงจะถูกเปิดใช้งานพร้อมการตั้งค่าที่คุณใช้ไว้

สิ่งนี้ยังรวมถึงการเลือกใช้รูปภาพคนละรูปสำหรับกล้องแต่ละตัวได้ด้วย!
