# ![](/icons/pencil.webp) จังหวะการปัด (Stroke)    

---
![](/images/stroke_overview.webp) 

## ภาพรวม 

คุณสามารถปรับแต่งพฤติกรรมของจังหวะการปัดของแปรงเครื่องมือส่วนใหญ่ได้
การตั้งค่าจะคล้ายกับที่มีอยู่ในแอปวาดภาพ 2 มิติ แต่จะมีบางตัวเลือกที่เฉพาะสำหรับการปั้น (sculpt) และงาน 3 มิติ

ตัวเลือกต่าง ๆ ถูกแบ่งออกเป็น 5 เมนูย่อย:

| Name     | Icon                         | Description                                                        |
| :------: | :--------------------------: | :----------------------------------------------------------------: |
| Stroke   | ![](/icons/stroke_dot.webp) | ควบคุมว่าจังหวะการปัดถูกนำไปใช้กับโมเดลอย่างไร                   |
| Alpha    | ![](/icons/alpha.webp)      | วิธีที่ใช้เท็กซ์เจอร์ระดับสีเทากับรอยปั๊มของแปรง                 |
| Falloff  | ![](/icons/falloff.webp)    | ควบคุมความแรงของแปรงจากกึ่งกลางไปยังขอบอย่างไร                  |
| Filter   | ![](/icons/filter.webp)     | วิธีที่แปรงได้รับผลกระทบจากเรขาคณิตของโมเดล                      |
| Pressure | ![](/icons/pressure.webp)   | ควบคุมการตอบสนองต่อแรงกดของปากกา/สไตลัส                          |

::: tip
ไม่ใช่ทุกตัวเลือกของ Stroke จะใช้ได้กับทุกเครื่องมือ ตัวเลือกที่ไม่ได้ใช้กับเครื่องมือปัจจุบันจะถูกปิดใช้งานหรือซ่อน
:::


## Stroke

### Radius

![](/images/stroke_radius.webp)

#### Share radius

เมื่อเปิดใช้ เครื่องมือทั้งหมดจะใช้รัศมีเดียวกัน ค่าเริ่มต้นคือแต่ละเครื่องมือจะมีรัศมีของตัวเอง

#### Size

* Screen - รัศมีถูกกำหนดในหน่วยของหน้าจอ หากคุณตั้งรัศมีให้กว้าง 100 พิกเซล มันจะคงที่ 100 พิกเซลไม่ว่าคุณจะซูมเข้าออกแค่ไหน
* Constant (3d) - รัศมีถูกกำหนดในหน่วย 3 มิติ ตัวอย่างเช่น หากคุณสร้างทรงกลมและตั้งรัศมีให้มีขนาดเท่าทรงกลม รัศมีจะคงขนาดเท่าทรงกลมเมื่อคุณซูมเข้าออก


### Stroke

![](/images/stroke_strokemode.webp)

จังหวะการปัดสามารถทำงานได้หลายโหมด:

### ![](/icons/stroke_dot.webp) Dot
ลากเหมือนการปัดแปรงวาดภาพปกติ 
![](/videos/stroke_dot.mp4) 

### ![](/icons/stroke_roll.webp) Roll
อัลฟาของแปรงจะหมุนตามทิศทางของจังหวะการปัด มีประโยชน์สำหรับทำลายเย็บผ้า/ด้าย 
![](/videos/stroke_roll.mp4) 

 ### ![](/icons/radius.webp) Lock + radius
ปั๊มรอยแปรงด้วย **_ความสูง_** คงที่ การลากจะกำหนดสเกลและการหมุน
![](/videos/stroke_lock_radius.mp4) 

### ![](/icons/falloff.webp) Lock + intensity 
ปั๊มรอยแปรงด้วย **_รัศมี_** คงที่ การลากจะกำหนดความสูงและการหมุน
![](/videos/stroke_lock_intensity.mp4) 

![](/images/stroke_strokemodemove.webp)

เครื่องมือ `Move` และ `Drag` มีตัวเลือกของตัวเอง 3 แบบ:

### ![](/icons/snake.webp) Drag

จะอัปเดตสิ่งที่อยู่ในรัศมีแปรงตลอดระยะจังหวะการปัด จังหวะที่เร็วจะทิ้งพื้นผิวไว้ข้างหลัง ในขณะที่จังหวะที่ช้าจะดึงวัสดุไว้ ทำให้เกิดรูปทรงที่ยาวขึ้น นี่คือโหมดเริ่มต้นของเครื่องมือ `Drag` เมื่อใช้ร่วมกับ `Dynamic Topology` สามารถใช้สร้างส่วนยื่นคล้ายงูได้ 
![](/videos/stroke_drag.mp4) 


### ![](/icons/grab.webp) Grab
จะเลือกสิ่งที่อยู่ในรัศมีแปรงเมื่อเริ่มปัด และคงการเลือกนั้นไว้ เหมาะสำหรับการย้าย (move) ที่ต้องการความแม่นยำ เพราะคุณสามารถปรับระยะการย้ายได้อย่างระมัดระวังโดยไม่เผลอย้ายเกินกว่าที่เลือกไว้ตอนแรก นี่คือโหมดเริ่มต้นของเครื่องมือ `Move`
![](/videos/stroke_grab.mp4) 


### ![](/icons/radius.webp) Lock + radius (drag) 
รัศมีที่ผู้ใช้ตั้งจะถูกละเลย และจะถูกกำหนดแบบไดนามิกตามระยะที่ลากออกจากจุดเริ่มต้น ระยะสั้น = รัศมีเล็ก ระยะยาว = รัศมีใหญ่ ใช้ตัวเลื่อน intensity เพื่อควบคุมรูปทรงของ falloff มีประโยชน์สำหรับการบล็อกทรงอินทรีย์ที่ยืดหยุ่นคล้ายยาง
![](/videos/stroke_lockradius_drag.mp4) 



### Adjust spacing intensity
![](/images/stroke_space_smooth.webp)

จังหวะที่มีค่า spacing ต่ำ (ต่ำกว่า 50%) สามารถสะสมได้เร็ว ทำให้จังหวะเข้มกว่าค่า spacing ที่สูง ตัวสวิตช์นี้จะชดเชยให้ เพื่อให้จังหวะมีความเข้มใกล้เคียงกันไม่ว่าค่า spacing จะเท่าใด

### Stroke spacing
กำหนดระยะห่างระหว่างการปั๊มแปรงแต่ละครั้งระหว่างการลาก ค่า < 100% จะซ้อนทับกัน ทำให้ดูเหมือนจังหวะต่อเนื่อง ค่า > 100% จะเริ่มมีช่องว่าง เหมาะสำหรับปั้นรายละเอียดอย่างรอยเย็บหรือซิป

### Lazy rope stabilizer
จังหวะจะหน่วงตามตำแหน่งตัวชี้ด้วยระยะหนึ่ง ใช้สำหรับวาดเส้นให้เรียบ
![](/videos/stroke_lazy_rope.mp4) 

### Stroke smoothing
เฉลี่ยตำแหน่งตัวชี้หลาย ๆ จุดเพื่อให้จังหวะเรียบขึ้น
เมื่อใช้ค่าที่สูง จังหวะจะหน่วงตามตัวชี้แต่จะตามทันในที่สุด
มีประโยชน์สำหรับการวาดเส้นเรียบ

### Snap radius
ดูดจุดเริ่มต้นของจังหวะใหม่ให้ไปยังจุดสิ้นสุดของจังหวะก่อนหน้า รัศมีจะกำหนดระยะที่ใช้ค้นหาจุดสิ้นสุดของจังหวะก่อนหน้า มีประโยชน์เมื่อวาดเส้นยาวต่อเนื่องแต่ต้องหยุดบ่อย ๆ

### ![](/icons/silhouette.webp) Silhouette
![](/images/stroke_silhouette.webp)
โดยค่าเริ่มต้น จังหวะจะมีผลเฉพาะพื้นผิวโมเดลภายในรัศมีแปรง เมื่อเปิดใช้ silhouette จังหวะจะถูกฉายทะลุโมเดลทั้งชิ้น มีประโยชน์มากสำหรับการบล็อกทรงเริ่มต้นของโมเดล หรือสำหรับรูปทรงที่ต้องการให้ด้านข้างตั้งฉาก

ทิศทางการฉายสามารถตั้งค่าได้โดยตรง ค่าเริ่มต้น 'Closest' จะตรวจจับมุมที่ดีที่สุดสัมพันธ์กับมุมมอง 

![](/videos/stroke_silhouette.mp4) 

### ![](/icons/dice.webp)Randomize

![](/images/stroke_randomize.webp)

ความเข้ม การเลื่อนตำแหน่ง การหมุน และสเกลของจังหวะสามารถสุ่มได้แต่ละค่า ใช้สร้างเอฟเฟกต์ได้หลากหลาย เช่น สุ่มร่วมกับเครื่องมือ paint เพื่อสร้างสีด่าง หรือสุ่มร่วมกับเครื่องมือ crease เพื่อสร้างรายละเอียดผิวหนัง

![](/videos/stroke_randomize.mp4) 

### Stroke Offset

ใส่ค่า offset คงที่ให้กับจังหวะ มีประโยชน์สำหรับหน้าจอเล็กที่นิ้วของคุณบังจังหวะการปัด 


## ![](/icons/alpha.webp) Alpha
![](/images/stroke_alpha.webp) 

`Alpha` คือเท็กซ์เจอร์ที่ใช้ปรับพฤติกรรมของแปรง
เป็นภาพระดับสีเทา โดยพิกเซลสีดำหมายถึงไม่มีการเปลี่ยนรูป และพิกเซลสีขาวหมายถึงการเปลี่ยนรูปเต็มที่

คลิกที่ภาพ alpha เพื่อเปลี่ยน

คลิกที่ตัวอย่างวัสดุ (material preview) เพื่อโหลด alpha จากพรีเซ็ตวัสดุ คุณยังสามารถบันทึกพรีเซ็ตวัสดุที่นี่ และเลือกฝังเท็กซ์เจอร์ไปกับเครื่องมือได้

::: tip
เท็กซ์เจอร์จะไม่ถูกปรับขนาด ดังนั้นเท็กซ์เจอร์ขนาดใหญ่สามารถทำให้ประสิทธิภาพช้าลงได้
:::

### Invert pixels
กลับค่าของภาพ ทำให้พิกเซลสีดำกลายเป็นสีขาว และสีขาวกลายเป็นสีดำ

::: tip

อัลฟาที่มากับ Nomad ในตัวไม่สามารถกลับค่าสีได้

:::

### Scaling

ขนาดแปรงใน Nomad เป็นวงกลมที่มีรัศมีที่ผู้ใช้กำหนด เท็กซ์เจอร์มักเป็นสี่เหลี่ยมจัตุรัสหรือสี่เหลี่ยมผืนผ้า พารามิเตอร์ `Scaling` ช่วยให้คุณกำหนดได้ว่าเท็กซ์เจอร์ควรพอดีกับวงกลมอย่างไร สำหรับเท็กซ์เจอร์สี่เหลี่ยม ค่า 0.7 จะทำให้เท็กซ์เจอร์พอดีในวงกลม ค่า 1 จะขยายเท็กซ์เจอร์ให้ใหญ่ขึ้นเพื่อให้วงกลมอยู่ด้านใน ทำให้ขอบถูกตัด

![](/images/alpha_scaling.webp) 

การเปิด `Scaling - Y` จะทำให้คุณยืด alpha ในแนวตั้งได้

![](/images/alpha_scaling_y.webp) 

### Rotation

เท็กซ์เจอร์ alpha จะหมุนตามทิศทางของจังหวะการปัด คุณสามารถเพิ่มค่า offset การหมุน และหากเปิดไอคอนล็อก เท็กซ์เจอร์จะล็อกมุมหมุนนี้สัมพันธ์กับหน้าจอ

### Tiling
![](/images/stroke_tiling.webp) 

กำหนดความถี่ที่เท็กซ์เจอร์ถูกทำซ้ำภายในโปรไฟล์แปรง โหมด tiling ช่วยให้คุณจำกัดให้มีเท็กซ์เจอร์เดียวในจังหวะ หรือทำซ้ำหลายครั้ง หรือแบบ mirrored ที่ทุกเท็กซ์เจอร์ลำดับที่สองจะถูกกลับด้านเพื่อสร้างแพทเทิร์นหรือช่วยให้เท็กซ์เจอร์ต่อเนื่องไร้รอยต่อ

![](/videos/alpha_tiling.mp4) 



### Mid value

โดยค่าเริ่มต้น พิกเซลสีดำหมายถึงไม่มีการเปลี่ยนรูป และสีขาวหมายถึงการเปลี่ยนรูปเชิงบวกเต็มที่ เช่น แปรง clay ที่มีเท็กซ์เจอร์ alpha ของหินจะดึงพื้นผิวออกเฉพาะบริเวณที่ alpha ไม่เป็นสีดำ

หากคุณต้องการให้แปรงดันพื้นผิวเข้า หรือทั้งดันเข้าและดึงออก คุณสามารถปรับค่า mid value ได้ หากตั้งค่าเป็น 0.5 สีเทากลางใน alpha จะไม่ทำอะไร สีดำจะดันเข้า สีขาวจะดึงออก




## Falloff

![](/images/falloff_menu.webp) 

คล้ายกับ [Alpha](#alpha) แต่คุณควบคุมความเข้มด้วยกราฟเส้นโค้งเดียว ค่านี้จะถูกรวมกับ alpha เพื่อให้คุณใช้เท็กซ์เจอร์สำหรับรายละเอียด และใช้ falloff เพื่อควบคุมการเฟดขอบและความเข้มที่กึ่งกลางของเครื่องมือ

เมื่อกราฟอยู่ด้านบน หมายถึงการเปลี่ยนรูปเต็มที่ เมื่ออยู่ด้านล่าง แปรงจะไม่มีผล

คุณสามารถมองกราฟนี้เป็นหน้าตัดผ่านปลายแปรง ส่วนล่างจะแสดงตัวอย่างว่าการปั๊มแปรงหนึ่งครั้งจะมีหน้าตาอย่างไรบนพื้นผิวโมเดล และหากมีเท็กซ์เจอร์ alpha สำหรับแปรง ก็จะแสดงด้วยเพื่อพรีวิวว่าการทำงานร่วมกันระหว่าง falloff และ alpha เป็นอย่างไร

### Preset
เมื่อเลือกโหมดนี้ การคลิกบนกราฟเส้นโค้งจะเปิดเมนูพรีเซ็ต เส้นโค้งมนจะให้ผลลัพธ์ที่นุ่มนวล เส้นโค้งเหลี่ยมจะให้ผลลัพธ์ที่คมชัด ปุ่ม `Sub` ในทูลบาร์ด้านซ้ายจะกลับทิศทางของ falloff ทำให้ส่วนบนของกราฟดันเข้าแทนที่จะดึงออก หรือกลับกัน

### Catmull-Rom
เมื่อเลือกแล้ว ผู้ใช้สามารถวาดกราฟ falloff ของตัวเองได้ ตัวแก้ไขกราฟทำงานเหมือนกราฟในส่วนอื่นของ Nomad:

* คลิกบนกราฟเพื่อสร้างจุดใหม่
* ลากจุดเพื่อย้ายตำแหน่ง
* คลิกบนจุดเพื่อสลับระหว่างมุมคมและมุมโค้ง
* ลากจุดไปทับจุดข้างเคียงเพื่อลบจุดนั้น

### B-spline
เมื่อเลือกแล้ว ผู้ใช้สามารถวาดกราฟ falloff ของตัวเองได้เช่นกัน ตัวแก้ไขทำงานเหมือน Catmull-Rom แต่จุดจะมีอิทธิพลต่อเส้นโค้งแทนที่จะอยู่บนเส้นโค้งโดยตรง ซึ่งช่วยให้สร้างรูปทรงเส้นโค้งที่นุ่มนวลได้ง่ายขึ้น

ตัวแก้ไขกราฟมีปุ่ม 3 ปุ่ม:

| Item     | Icon                        | Description                                  |
| :------: | :-------------------------: | :------------------------------------------: |
| Maximize | ![](/icons/maximize.webp)  | ขยายตัวแก้ไขกราฟ                            |
| Reset    | ![](/icons/reset.webp)     | คืนค่ากราฟเป็นสถานะเริ่มต้น                |
| Symmetry | ![](/icons/symmetric.webp) | แสดงกราฟเป็น 'ปลายแปรง' แบบสมมาตร         |


### Influence

* Sphere (3d) - คำนวณอิทธิพลจากระยะระหว่างจุดยอด (vertex) กับกึ่งกลางแปรง
* Circle (2d) - โปรเจกต์จุดยอดลงบนระนาบพื้นที่ก่อน แล้วจึงคำนวณระยะจากกึ่งกลางแปรง คล้ายกับวิธีการ sample alpha 
* Cylinder - โปรเจกต์อิทธิพลผ่านพื้นที่เป็นทรงกระบอก ใช้โดยโหมด Silhouette ด้านล่าง

### Silhouette
โดยค่าเริ่มต้น จังหวะจะมีผลเฉพาะพื้นผิวโมเดลภายในรัศมีแปรง เมื่อเปิดใช้ silhouette จังหวะจะถูกฉายทะลุโมเดลทั้งชิ้น มีประโยชน์มากสำหรับการบล็อกทรงเริ่มต้นของโมเดล หรือสำหรับรูปทรงที่ต้องการให้ด้านข้างตั้งฉาก

### Accumulate stroke
เปิดเพื่อไม่จำกัดปริมาณวัสดุที่เพิ่ม/ลบต่อหนึ่งจังหวะ เช่น เครื่องมือ `Clay` เปิดตัวเลือกนี้ ทำให้วัสดุสามารถกองทับกันได้เรื่อย ๆ ในขณะที่เครื่องมือ `Brush` ปิดตัวเลือกนี้ ทำให้จังหวะหยุดเพิ่มวัสดุหากคุณลากซ้ำบนบริเวณเดิมของเมช 

### Front-facing vertex only
ตัวเลือกนี้จะเมินจุดยอดที่หันหลังให้กล้อง
มีประโยชน์หากคุณต้องการทาสีบางส่วนของเรขาคณิตที่บางโดยไม่กระทบอีกด้าน
ใช้กับการปั้นได้เช่นกัน แต่คุณอาจพบอาร์ติแฟกต์บางอย่าง

### Allow dynamic topology
ตัวเลือกนี้ใช้ได้เฉพาะเมื่อเมชของคุณอยู่ในโหมด [Dynamic Topology](topology.md#dynamic-topology) คุณสามารถเปิดหรือปิด dynamic topology แยกตามเครื่องมือได้

### Connected topology
เปิดเพื่อให้ปั้นเฉพาะจุดยอดที่เชื่อมต่อกับพื้นผิวที่คุณแตะด้วยเครื่องมือ ตัวอย่างเช่น เมื่อใช้ร่วมกับเครื่องมือ `Move` จะช่วยให้คุณย้ายส่วนหนึ่งได้แม้ว่าจะตัดกับอีกส่วนหนึ่งอยู่

### Protect Area
![](/images/filter_protect_area.webp) 

ตัวเลือกเหล่านี้จะหยุดไม่ให้เครื่องมือมีผลกับบางส่วนของเมชภายใต้เงื่อนไขต่าง ๆ 

ตัวเลือก 'Auto' หมายความว่าหากคุณเปิด hide, mask หรือ facegroup ในเมนู [Shading](shading) ส่วนเหล่านั้นจะถูกป้องกัน แต่หากปิดในเมนูนั้น การป้องกันจะถูกปิดด้วย

* `All` - สวิตช์กำหนดว่าฟิลเตอร์ป้องกันจะเป็นแบบทั่วทั้งโปรเจกต์หรือแยกตามเครื่องมือ
* `Hide` - หากบางส่วนของเมชถูกซ่อนด้วยเครื่องมือ hide ให้กำหนดว่าจะป้องกันหรือไม่
* `Mask` - หากบางส่วนของเมชถูกมาสก์ ให้กำหนดว่าจะป้องกันหรือไม่
* `Facegroup` - กำหนดให้ใช้เครื่องมือได้เฉพาะภายใน facegroup แรกที่แตะเท่านั้น


### Keep sharp edges
ไม่รวมจุดที่ normal ต่างจาก normal ของพื้นผิวมากเกินไป

จะเปลี่ยนวิธีคำนวณระนาบพื้นที่ (Area sampling)

ตัวเลือกนี้มีประโยชน์สำหรับเครื่องมือที่อิงการแบน (flatten-based) หรือหากคุณต้องการลงสีบนพื้นผิวที่ส่วนใหญ่แบนโดยไม่ให้สีล้นออกนอกขอบ

![](/videos/filter_keep_sharp_edges.mp4)

### Area sampling
แปรงหรือโหมด stroke บางประเภทต้องการ normal ของระนาบและตำแหน่งระนาบบนพื้นผิวเพื่อทำงาน

คุณสามารถควบคุมวิธีคำนวณระนาบเฉลี่ยนี้ได้โดยตั้งค่าพื้นที่ sampling เป็นสัดส่วนของรัศมีเครื่องมือ

ที่ 100% ทุกจุดภายในวงกลมการเลือกจะถูกนำมาคำนวณ

ที่ 0% จะใช้เฉพาะจุดยอดหรือสามเหลี่ยมที่ใกล้ที่สุด ค่านี้สามารถลิงก์ร่วมกันสำหรับทั้ง `Normal radius` และ `Position radius` หรือปลดลิงก์และตั้งแยกกันได้


### Depth masking
![](/images/stroke_depthmask.webp)

ไม่รวมจุดที่อยู่เหนือหรือต่ำกว่าระยะที่กำหนดจากระนาบที่คำนวณได้ (Area sampling)

ใช้เพื่อทาสีเฉพาะบริเวณนูน หรือเฉพาะบริเวณหลุม/ร่องได้

กราฟแสดงหน้าตัดของพื้นผิว เส้นแนวนอนคือระดับพื้นผิว วงกลมแทนรัศมี falloff ของการทาสีเหนือและใต้พื้นผิว `Height offset` คือเปอร์เซ็นต์เหนือหรือต่ำกว่าพื้นผิวที่เริ่มคำนวณการมาสก์ `Top area` และ `Bottom area` ช่วยให้คุณสเกล falloff เหนือและใต้จุด offset

#### ตัวอย่าง: ทาสีในร่องลึก (cavities)
หากต้องการทาสีเฉพาะบริเวณร่องลึก ตั้งค่า height offset เป็น -100% และปรับตัวเลื่อน top area ให้ห่างจากเส้นแนวนอน นั่นหมายความว่าคลิกแรกจะตั้งค่าความลึก 'ศูนย์' และเฉพาะบริเวณที่ต่ำกว่าความลึกนี้เท่านั้นที่จะได้รับผล

![](/videos/stroke_depth_cavity.mp4)

#### ตัวอย่าง: ทาสีบนปุ่มนูน (bumps)
หากต้องการทาสีเฉพาะบริเวณที่สูง ตั้งค่า height offset เป็น +90% เพื่อให้ด้านล่างของวงกลมตัดกับเส้นแนวนอนเพียงเล็กน้อย เมื่อคุณคลิกบนยอดของบริเวณที่สูง ค่าความลึกนี้จะถูกตั้ง และเฉพาะบริเวณที่ระดับนั้น บวกกับเล็กน้อยด้านล่าง และทุกอย่างที่สูงกว่านั้นจะถูกทาสี ส่วนร่องลึกจะถูกข้าม
![](/videos/stroke_depth_bump.mp4)


## Pressure 
![](/images/pressure_menu.webp) 

ควบคุมว่าความดันของปากกา/สไตลัสมีผลต่อแปรงอย่างไร

โดยค่าเริ่มต้น `Use global settings` จะเปิดอยู่ หมายความว่าแปรงทั้งหมดจะแชร์ค่าความดันเดียวกัน

### Pressure - Radius

กราฟนี้ควบคุมว่ารัศมีแปรงได้รับผลจากความดันอย่างไร ค่าเริ่มต้นคือความสัมพันธ์เชิงเส้น ดังนั้นหากสไตลัสของคุณตอบสนองได้เรียบเนียน การเปลี่ยนแปลงรัศมีก็ควรรู้สึกลื่นไหลเช่นกัน อย่างไรก็ตาม สไตลัสหลายรุ่นมีการตอบสนองไม่เชิงเส้น ซึ่งคุณสามารถชดเชยได้ด้วยกราฟนี้ ตัวอย่างเช่น หากรู้สึกว่ารัศมีไม่ถึงค่าสูงสุดแม้ใช้แรงกดสูง ให้ใช้พรีเซ็ตกราฟอย่าง 'out-pow3' ที่โค้งขึ้นเพื่อเพิ่มรัศมีให้เร็วขึ้น

ไดอะล็อกนี้คล้ายกับหน้าจอกราฟ falloff คุณสามารถใช้พรีเซ็ตโดยแตะที่หน้าต่างกราฟ หรือวาดกราฟของคุณเองด้วยโหมด Catmull-Rom และ B-Spline

หากต้องการให้รัศมีคงที่ ให้ปิดส่วนนี้

### Pressure - Intensity

คล้ายกับตัวเลื่อนรัศมี แต่ใช้ควบคุมความเข้ม หากต้องการให้ความเข้มคงที่ ให้ปิดส่วนนี้

### Pressure smoothing

เฉลี่ยค่าความดันของสไตลัสเพื่อให้ผลลัพธ์เรียบเนียนขึ้น
