# ![](/icons/symmetry.webp) สมมาตร (Symmetry)

เมนูนี้ใช้ควบคุมว่าจังหวะแปรง (strokes) จะถูกทำซ้ำข้ามระนาบกระจกหรือแบบรัศมีอย่างไร และใช้สำหรับกู้คืนความสมมาตรให้กับวัตถุที่ไม่สมมาตร

![](/images/symmetry_overview.webp) 

## ภาพรวม 
คุณสามารถใช้สมมาตรได้หลายวิธี:

* เป็นกระจกเงา สะท้อนงานข้ามแกน X (ซ้าย/ขวา), Y (บน/ล่าง), Z (หลัง/หน้า) หรือผสมกัน
* สมมาตรรัศมีสามารถตั้งได้บนแกน X Y Z พร้อมจำนวนการทำซ้ำ เช่น การปั้นปลาดาว
* กระจกสามารถทำงานรอบจุดกำเนิด (เรียกว่าโหมด world) หรือรอบจุดศูนย์กลางของวัตถุ (เรียกว่าโหมด local)
* งานปั้นที่เริ่มจากการไม่สมมาตรสามารถบังคับให้กลับมาเป็นสมมาตรได้

คีย์ลัดสำหรับเปิด/ปิดสมมาตรสามารถพบได้ที่แผงด่วนด้านซ้าย (ปุ่ม *"Sym"*). ตัวอักษรเล็ก 'L/W' แสดงว่า Nomad อยู่ในโหมดสมมาตร Local หรือ World คุณยังสามารถกดค้างหรือปัดไปที่กึ่งกลางหน้าจอเพื่อเรียกเมนูออกมาได้

![](/images/symmetry_button.webp) 

นี่เป็นตัวเลือกแบบ global ดังนั้นสถานะจะถูกใช้ร่วมกันข้ามเครื่องมือต่าง ๆ  
ข้อยกเว้นเพียงอย่างเดียวคือเครื่องมือแปลงรูป ([Move](#translate), [Rotate](#rotate), [Scale](#scale) และ [Gizmo](#gizmo)) ซึ่งมีสถานะสมมาตรของตัวเอง

::: tip
เมนูสมมาตรนี้มีไว้สำหรับควบคุมสมมาตรของจังหวะแปรงเป็นหลัก คุณยังสามารถสะท้อนและทำซ้ำวัตถุผ่าน [repeaters ในเมนูฉาก](scene#repeaters) ได้ด้วย
:::

## Enabled
สลับโหมดกระจกเงา นี่เหมือนกับปุ่ม `Sym` ในแผงด่วนด้านซ้ายทุกประการ 

## Planes

เปิดใช้งานระนาบสมมาตร และจำนวนการทำซ้ำสำหรับสมมาตรรัศมี โปรดทราบว่าคุณไม่จำเป็นต้องเลือกเพียงระนาบเดียว คุณสามารถเปิดได้ 1, 2 หรือทั้ง 3 ระนาบพร้อมกันเพื่อสร้างสมมาตรที่ซับซ้อน

แกนและจำนวนการทำซ้ำสำหรับสมมาตรรัศมี โปรดทราบว่าสิ่งเหล่านี้ก็ไม่จำกัดแค่แกนเดียวเช่นกัน และยังสามารถทำงานร่วมกับสมมาตรแบบระนาบเพื่อสร้างผลลัพธ์ที่ละเอียดได้อย่างรวดเร็ว (จำนวนการทำซ้ำรวมถูกจำกัดไว้ที่ 150)

![](/videos/symmetry_demo.mp4) 

## Method
กระจกสามารถเป็นแบบ 'Local' ซึ่งจะเคลื่อนที่ไปพร้อมกับวัตถุ หรือแบบ 'World' ซึ่งจะไม่เคลื่อนที่ หากคุณไม่แน่ใจว่าต้องใช้โหมดใด ให้สังเกตระนาบกระจกและตัวบ่งชี้รัศมีที่ซ้อนทับอยู่บนวัตถุ เมื่ออยู่ในโหมด local หากคุณใช้ gizmo แปลงรูปและย้ายโมเดล ตัวบ่งชี้กระจกก็จะเคลื่อนที่ตามไปด้วย เมื่ออยู่ในโหมด world ตัวบ่งชี้กระจกจะอยู่กับที่ และวัตถุจะเลื่อนผ่านมันไป

## Mirroring
![](/images/symmetry_mirroring.webp)

เมื่อปั้นใกล้ระนาบสมมาตร แปรงบางชนิดอาจมีพฤติกรรมสมมาตรที่ไม่สมบูรณ์ ส่วนนี้ช่วยให้คุณกู้คืนสมมาตรได้โดยการคัดลอกด้านหนึ่งของงานปั้นไปยังอีกด้านหนึ่ง 

`Direction` - ปุ่ม `<<` และ `>>` กำหนดว่าด้านใดจะถูกคัดลอกไปยังอีกด้าน Nomad คำนวณจากมุมมองปัจจุบันของคุณ ดังนั้นการตั้งค่าเป็น `<<` ตัวอย่างเช่น จะคัดลอกจากด้านขวาของหน้าจอไปยังด้านซ้ายของหน้าจอเสมอ

![](/icons/shield.webp) `Mask` - ปุ่มมาสก์ช่วยให้คุณจำกัดส่วนที่จะถูกสะท้อนกลับ; การมาสก์ด้านปลายทางจะปกป้องบริเวณนั้น การมาสก์ด้านต้นทางจะหยุดไม่ให้บริเวณนั้นถูกสะท้อนไปยังปลายทาง 

![](/icons/tool_hide.webp) `Hide` - เมื่อเปิดใช้งาน พื้นที่ที่ถูกซ่อนอยู่ด้านต้นทางจะไม่ถูกสะท้อนไปยังด้านปลายทาง 

`Mirror` จะพยายามตรวจสอบว่าทอพอโลยีทั้งสองด้านของกระจกเหมือนกันหรือไม่ ถ้าเหมือนกันก็จะเพียงแค่ย้ายจุดยอด (vertices) วิธีนี้เป็นกรณีที่พบได้บ่อยกว่า

`Split & Mirror` จะตัดวัตถุตามแนวกระจก คัดลอกด้านหนึ่ง สะท้อนมันไปยังอีกด้าน และเชื่อมจุดยอดตามแนวกระจก เป็นตัวเลือกที่ทำลายโครงสร้างมากกว่า และจะลบ multiresolution แต่บางครั้งจำเป็นต้องใช้วิธีนี้หากโมเดลแตกต่างกันมากทั้งสองด้านของกระจก

### Flip object
![](/images/symmetry_flip.webp)
สลับด้านซ้ายให้เป็นด้านขวา และในทางกลับกัน ลักษณะคล้ายกับการใช้เมนูเครื่องมือ gizmo แล้วตั้งค่า scale เป็น -1 บนแกน X

## Reset and Edit

![](/images/symmetry_edit.webp)

สามารถแก้ไขตำแหน่งและทิศทางของสมมาตรได้ (แต่ไม่แนะนำ!) หากจำเป็น `World center` และ `Orientation` จะรีเซ็ตตำแหน่งและทิศทางของสมมาตรกลับเป็นค่าเริ่มต้น

โดยปกติ Nomad จะรู้เองว่าควรวางระนาบสมมาตรไว้ที่ไหน จึงไม่แนะนำให้ปรับสิ่งนี้ด้วยตนเอง แต่ปุ่ม `Gizmo (Edit)` เปิดโอกาสให้ผู้ใช้ขั้นสูงทำได้ เมื่อคลิกปุ่มนี้ จะมี gizmo แสดงขึ้นมาเพื่อให้คุณเลื่อนและหมุนระนาบสมมาตร หากต้องการคืนค่าจุดศูนย์กลางและทิศทางเริ่มต้น ปุ่ม `object center` และ `orientation` จะทำหน้าที่นี้

พฤติกรรมของตัวเลือกเหล่านี้จะเปลี่ยนไปตาม space (*Local/World*) ที่คุณอยู่  
ดังนั้นหากมันทำงานไม่ตรงตามที่คุณคาดไว้ ให้ตรวจสอบก่อนว่าคุณอยู่ใน space ที่ถูกต้องหรือไม่

::: tip
ปุ่ม `Gizmo (Edit)` ถูกทำให้เป็นสีเทาโดยตั้งใจ เพื่อเตือนว่าคุณอาจไม่ควรใช้มัน!
:::

## Show options
![](/images/symmetry_show.webp)


`Show line` และ `Show plane` จะสลับการแสดง overlay ใน viewport ของตำแหน่งสมมาตร โปรดทราบว่าการปิดตัวเลือกเหล่านี้จะมีผลก็ต่อเมื่อเมนูสมมาตรถูกปิดแล้วเท่านั้น
