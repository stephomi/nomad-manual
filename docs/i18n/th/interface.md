# ![](/icons/interface.webp) เมนูอินเทอร์เฟซ {#interface-menu}

เมนูนี้ควบคุมตัวเลือกมากมายสำหรับปรับแต่งหน้าตาอินเทอร์เฟซของ Nomad 

![](/images/interface_overview2.webp)

Nomad สามารถปรับแต่งได้ค่อนข้างลึก เมนูนี้แบ่งออกเป็น 4 ส่วน: Interface, Gesture, Bindings, Debug

![](/images/interface_menu.webp)


::: tip
หน้านี้อธิบายเมนู interface ไม่ใช่ตัวอินเทอร์เฟซเอง! ภาพรวมของอินเทอร์เฟซอธิบายไว้ใน [Getting Started](gettingstarted.md)
:::

## อินเทอร์เฟซ {#interface}

ส่วน Interface ให้คุณเพิ่มคีย์ลัด สร้างทูลบาร์ลอย และควบคุมสี ขนาด และลักษณะของอินเทอร์เฟซของ Nomad

![](/images/interface_overview3.webp)

### เพิ่มทางลัด (ด้านล่าง)... {#add-shortcuts-bottom}
![](/images/interface_shortcuts.webp)

ทูลบาร์ด้านล่างจะมีคีย์ลัดเหล่านี้เปิดใช้งานโดยค่าเริ่มต้น:
* `Undo` - ยกเลิกการทำงานครั้งก่อนหน้า
* `Redo` - เรียกคืนการทำงานที่ถูกยกเลิกครั้งล่าสุด
* `Solo` - ซ่อนอ็อบเจ็กต์อื่นทั้งหมดชั่วคราว เหลือให้เห็นเฉพาะอ็อบเจ็กต์ที่เลือก กดอีกครั้งเพื่อแสดงอ็อบเจ็กต์ทั้งหมด
* `X-ray` - ทำให้อ็อบเจ็กต์อื่นทั้งหมดโปร่งแสงบางส่วนชั่วคราว เหลือให้เห็นเฉพาะอ็อบเจ็กต์ที่เลือกเป็นทึบ กดอีกครั้งเพื่อคืนค่าวัสดุเริ่มต้น
* `Voxel remesh` - รีเมชอ็อบเจ็กต์ปัจจุบันโดยใช้ความละเอียด voxel ครั้งล่าสุด การกดค้างหรือปัดขึ้นจะเปิดสไลเดอร์ความละเอียดและตัวเลือกขอบคม
* `Grid` - เปิด/ปิดกริดมุมมอง การกดค้างหรือปัดขึ้นจะให้คุณเปลี่ยนสีและความทึบของกริด
* `Wireframe` - เปิด/ปิดการซ้อนทับ wireframe การกดค้างหรือปัดขึ้นจะให้คุณเปลี่ยนสีและความทึบของ wireframe
* `Inspector` - ให้คุณดูคุณสมบัติของเมช เช่น uv เท็กซ์เจอร์ที่ bake แล้ว และคุณสมบัติอื่น ๆ แสดงทับบนพื้นหลังของ Nomad
* `Face Group` - เปิด/ปิดการซ้อนทับ facegroup ดูข้อมูลเพิ่มเติมที่ [Tools->FaceGroup](tools.md#facegroup) 

คีย์ลัดที่ใช้บ่อยอื่น ๆ มีให้เลือกจากเมนูนี้ และยังมีอีกมากในปุ่ม bindings

#### ![](/icons/more.webp) การผูกปุ่ม {#bindings-list}

แทบทุกฟังก์ชันของ Nomad สามารถเพิ่มลงในทูลบาร์คีย์ลัดผ่านปุ่ม bindings เมื่อคลิกปุ่มจะมีเมนู bindings แสดงขึ้น:

![](/images/interface_bindings_search.webp)

คุณสามารถค้นหาตามหมวดหมู่ผ่านไอคอนด้านบน หรือใช้ช่องค้นหาเพื่อหาฟังก์ชันตามชื่อ คลิกที่ฟังก์ชันเพื่อเพิ่มลงในทูลบาร์ คลิกอีกครั้งเพื่อลบออก

#### ![](/icons/list.webp) ลำดับ {#order}

จะแสดงรายการคีย์ลัด กดค้างแล้วลากเพื่อจัดลำดับใหม่

#### ![](/icons/reset.webp) รีเซ็ต {#reset}

Reset จะคืนค่าทูลบาร์ด้านล่างเป็นการตั้งค่าเริ่มต้น

### เพิ่มทางลัด (หน้าต่าง)... + {#add-shortcuts-window}
![](/images/interface_add_shortcuts_window.webp)

การคลิกเครื่องหมาย + จะเพิ่มทูลบาร์ลอย ทูลบาร์จะยังไม่แสดงจนกว่าคุณจะคลิกปุ่ม bindings และเพิ่มคีย์ลัดลงไป จากนั้นจึงค่อยทำให้มองเห็นได้ 

คุณสามารถสร้างทูลบาร์ได้หลายอัน แต่ละทูลบาร์มีตัวเลือกต่อไปนี้ในเมนูนี้:

* ![](/icons/checked.webp) `Visible` - เปิด/ปิดการมองเห็นของทูลบาร์
* ![](/icons/more.webp)`Bindings` - แสดงหน้าต่าง binding เพื่อเลือกฟังก์ชันที่จะเพิ่มหรือลบออกจากทูลบาร์
* ![](/icons/list.webp)`Order` - แสดงเมนูสำหรับจัดลำดับทูลบาร์
* ![](/icons/reset.webp) `Reset` - รีเซ็ตทูลบาร์เป็นสถานะเริ่มต้น
* ![](/icons/resize_bottom_right.webp) `Resize corner` - เปิด/ปิดตัวจับปรับขนาดที่มุมขวาล่างของทูลบาร์
* ![](/icons/sort_down.webp) `Collapsable` - เปิด/ปิดตัวจับยุบ/ขยายที่มุมขวาบน
* ![](/icons/trash.webp) `Delete` - ลบทูลบาร์

### กล่องเครื่องมือ {#toolbox}

ตัวเลือกสำหรับเมนูเครื่องมือด้านขวาของอินเทอร์เฟซ Nomad

![](/images/interface_toolbox.webp)

#### ![](/icons/resize_bottom_right.webp) มุมปรับขนาด UI {#ui-resize-corner}

เปิด/ปิดตัวจับปรับขนาดที่มุมล่างของทูลบาร์

#### ซ่อน {#hidden}
ปกติแล้วไอคอน toolbox ในแถบด้านบนจะสลับระหว่างคอลัมน์เดียวยาว หรือรายการเครื่องมือหลายคอลัมน์ ตัวเลือกนี้จะสลับระหว่างรายการหลายคอลัมน์ หรือซ่อนทั้งหมด

#### มีสี {#colored}
ใส่รหัสสีให้ไอคอนตามหมวดหมู่ เช่น เครื่องมือ mask ทั้งหมดเป็นสีน้ำตาล เครื่องมือ split เป็นสีแดง เครื่องมือ flatten เป็นสีเขียว เป็นต้น

#### แถว: อัตโนมัติ (>1) {#rows-auto-1}

#### รีเซ็ตลำดับ {#reset-order}
รีเซ็ตเครื่องมือเริ่มต้นใน toolbox ให้กลับเป็นลำดับเริ่มต้น ไอคอนที่กำหนดเองจะยังคงอยู่ใน toolbox ที่ท้ายรายการ


### พรีเซ็ต {#presets}

![](/images/interface_presets.webp)

ชุดสีสำเร็จรูปสำหรับอินเทอร์เฟซ

#### ปุ่มคอนทราสต์สูง {#high-contrast-button}
สไตล์ปุ่มทางเลือกที่ทำให้มองเห็นได้ชัดขึ้นเมื่อเปิดใช้งาน หากตั้งเป็น Auto Nomad จะใช้โหมดนี้เมื่อความต่างสีของ UI ระหว่างสถานะเปิด/ปิดต่ำ

#### วิดเจ็ตสี/สีพื้นฐาน {#color-widgetcolor-base}
สีหลักที่ใช้ในอินเทอร์เฟซ

#### แผงโปร่งใส, แผงสี, ความแรงของเบลอ {#transparent-panel-color-panel-blur-strength}
![](/images/interface_transparent.webp)
เมื่อเปิดใช้งาน จะมีตัวเลือกเพิ่มเติมสำหรับควบคุมลักษณะของเมนูและพาเนลใน Nomad สามารถปรับสี ความโปร่งใส และระดับการเบลอได้

::: tip
บนอุปกรณ์ขนาดเล็ก อาจเป็นประโยชน์ที่จะตั้ง color panel ให้เกือบขาว โปร่งใส และเบลอน้อย เพื่อไม่ให้เมนูบังฉาก
:::

## ท่าทาง {#gesture}
เมนู gesture ควบคุมว่าการกดปากกาและนิ้วจะควบคุม Nomad อย่างไร

![](/images/interface_gesture.webp)

### ตัวเลือกท่าทาง {#gesture-options}
![](/images/interface_gesture_matrix.webp)

Nomad สามารถจำกัดการทำงานตามอุปกรณ์อินพุตได้ เช่น การลากด้วยนิ้วอาจใช้ได้แค่ขยับกล้อง ในขณะที่การลากด้วยปากกาใช้ได้แค่ปั้น หากคุณมีเมาส์หรือทัชแพด ก็สามารถกำหนดให้ควบคุมการทำงานเฉพาะได้เช่นกัน

ปัจจุบัน Nomad ให้คุณตั้งโหมดเหล่านี้ให้ควบคุมด้วยนิ้ว ปากกา หรือเมาส์ ในแบบผสมใดก็ได้:

* Camera
* Sculpt
* Gizmo
* Material picking (ผ่านการกดค้าง)
* Select object


`Finger always smooths` - สามารถตั้งให้ Smooth ทำงานเฉพาะเมื่อมีการลากด้วยนิ้ว

### ![](/icons/tool_mask.webp) มาสก์ {#mask}

![](/images/interface_gesture_mask.webp)

`One tap shortcuts` - เปิดใช้งานคีย์ลัดแบบแตะครั้งเดียวต่อไปนี้โดยไม่ต้องกดปุ่ม mask ค้าง จะอนุญาตท่าทางดังนี้:
* แตะบนพื้นหลังเพื่อกลับด้าน mask
* แตะบนพื้นที่ที่ถูก mask เพื่อเบลอ mask
* แตะบนพื้นที่ที่ไม่ถูก mask เพื่อทำให้ mask คมขึ้น

### สลับ มาสก์ <-> SelMask {#toggle-mask-selmask}
![](/images/interface_gesture_togglemask.webp)
* `Stroke` - กดค้างจะสลับระหว่าง Mask และ SelMask และเริ่มเส้นใหม่ เมื่อจบเส้น เครื่องมือก่อนหน้าจะถูกเลือกกลับมา 
* `Tool` - กดค้างแล้วปล่อยโดยไม่ขยับเพื่อสลับระหว่าง Mask และ SelMask 

### ![](/icons/tool_hide.webp) ซ่อน {#hide}
![](/images/interface_gesture_hide.webp)

`One tap shortcuts` จะเปิดใช้งานคีย์ลัดต่อไปนี้กับเครื่องมือ hide:
* แตะบน face group เพื่อซ่อน
* แตะในพื้นที่ว่างเพื่อกลับด้านโพลิกอนที่ถูกซ่อน

### ![](/icons/finger3.webp) สามนิ้ว {#three-fingers}
![](/images/interface_gesture_threefingers.webp)

หากอุปกรณ์ของคุณรองรับท่าทาง 3 นิ้ว ให้กำหนดคีย์ลัดสำหรับท่าทางนั้นได้ 

ตารางตัวเลือกช่วยให้คุณกำหนดการลากแนวตั้งและแนวนอนเป็นคีย์ลัดแยกกันได้ โปรดทราบว่าหากเลือกท่าทางเดียวกันให้กับ 2 ตัวเลือก ตัวใดตัวหนึ่งจะถูกปิดใช้งาน

* `Rotate lighting` - หมุน environment ไฟ และ Matcap
* `Tool Radius` - ปรับรัศมีเครื่องมือ
* `Tool Intensity` - ปรับความแรงของเครื่องมือ 

### ![](/icons/fingerprint.webp) ประวัติ 2/3 {#history-23}
`History shortcuts` - เมื่อเปิดใช้งาน ท่าทางต่อไปนี้จะทำงาน:
* Undo - แตะด้วย 2 นิ้ว
* Redo - แตะด้วย 3 นิ้ว

`Long press` - เมื่อเปิดใช้งาน การกด 2/3 นิ้วค้างจะทำการ undo/redo อย่างรวดเร็ว

### การช่วยการเข้าถึง {#accessibility}

![](/images/interface_gesture_accessibility.webp)

`Assistive window` จะเปิดทูลบาร์ลอยสำหรับควบคุมการลาก การ pinch การหมุน และการควบคุมกล้อง

### กล้อง {#camera}
คีย์ลัดไปยังเมนู `Camera` (ตัวเลือกกล้องเคยอยู่ที่นี่ แต่ถูกย้ายไปเมนูกล้องแล้ว)

### ดับเบิลแทปปากกา -> การผูกปุ่ม {#pencil-tap}

คีย์ลัดไปยังเมนู `Bindings` (ตัวเลือก Pencil tap และ double tap เคยอยู่ที่นี่ แต่ถูกย้ายไปเมนู bindings แล้ว)


## การผูกปุ่ม {#bindings}
คีย์ลัดคีย์บอร์ดและปุ่มต่าง ๆ สามารถกำหนดได้จากเมนู bindings:

![](/images/interface_bindings.webp)
แทบทุกฟังก์ชันใน Nomad สามารถผูกกับคีย์ลัดคีย์บอร์ดได้หากอุปกรณ์ของคุณมีคีย์บอร์ด หรือผูกกับปุ่มเสริมบนปากกา หรือแม้แต่คอนโทรลเลอร์เกมแพด

หากต้องการสร้าง binding ให้คลิกสี่เหลี่ยมข้างฟังก์ชัน แล้วกดปุ่ม/คีย์ 

ค้นหาฟังก์ชันผ่านไอคอนหมวดหมู่ด้านบน หรือผ่านช่องค้นหาเพื่อหาตามชื่อ 

สามารถปิดใช้งาน binding รายตัวได้ผ่านช่องทำเครื่องหมายข้างชื่อ binding

### ![](/icons/more.webp) เมนูบริบท {#context-menu}
ไอคอน ![](/icons/more.webp) หลังแต่ละ binding จะเปิด context menu:

![](/images/interface_bindings_context.webp)

* `Clone` - โคลน binding
* `Reset` - รีเซ็ต binding
* `Delete` - ลบ binding
* `Toggle shortcut on key tap` - กำหนดว่าการแตะสั้นกับการกดค้างจะถูกมองว่าแตกต่างกันหรือไม่ เมื่อเปิดใช้งาน การแตะจะเปิดใช้เครื่องมือ การกดค้างจะเปิดใช้เครื่องมือเฉพาะตอนกดคีย์อยู่ เมื่อปล่อยจะกลับไปยังเครื่องมือก่อนหน้า บางครั้งเรียกว่า 'sticky keys' ในแอป 3D อื่น ๆ

### ขั้นสูง {#advanced}
ด้านล่างของเมนู bindings จะมีเมนูเฟืองสำหรับตัวเลือกขั้นสูง:

![](/images/interface_bindings_advanced.webp)

* `Toggle shortcut on key tap` - การแตะคีย์ลัดมาตรฐานสำหรับ mask, smooth, gizmo, hide, sub จะสลับไปโหมดนั้น แต่การกดค้างจะสลับไปโหมดนั้นชั่วคราว และเมื่อปล่อยคีย์ โหมดจะกลับไปเป็นโหมดก่อนหน้า บางครั้งเรียกว่า 'sticky keys' ในแอป 3D อื่น ๆ
* `Toggle tool shortcuts` - เมื่อใช้คีย์ลัดเครื่องมือ หากกดคีย์ลัดเดิมซ้ำ จะสลับกลับไปยังเครื่องมือก่อนหน้า ทำให้คุณสลับไปมาระหว่างสองเครื่องมือด้วยปุ่มลัดเดียวกันได้
* `Invert Y (Zooming)` - กลับทิศการซูมแกน Y
* `Reset bindings` - รีเซ็ต bindings ทั้งหมดเป็นค่าเริ่มต้น

## ดีบัก {#debug}
ตัวเลือกทดลองและ debug บางส่วนจะอยู่ในเมนูนี้ ตัวเลือกเหล่านี้หลายอย่างควรปล่อยไว้เป็นค่าเริ่มต้น และเปลี่ยนเฉพาะหลังจากติดต่อฝ่ายสนับสนุนของ Nomad แล้ว

![](/images/interface_debug.webp)
### ยูวี {#uv}
* `Normalize Uvs` - Nomad จะ normalize UV ให้อยู่ภายใน tile ช่วง [0-1]

### Quad Remesh {#quad-remesh}
* `Instant Mesh` - เปิดใช้งานอัลกอริทึม instant remesh
* `Quadriflow` - วิธี quad remesh ทางเลือก

### เรนเดอร์ {#render}
* `Heightmap` - เปลี่ยน viewport ให้เรนเดอร์ความลึกของฉาก ใช้สร้าง alpha map สำหรับแปรงได้
* `Refraction write depth (back)` - ผิวด้านหลังของเมชหักเหแสงจะถูกเขียนลง depth buffer
* `Flip Y (normal map)` - กลับช่อง y ตอน bake normal map ซึ่งจำเป็นสำหรับเอนจินเกมและเรนเดอร์บางตัว
* `Angle weighted smooth normals` - การปรับวิธี smooth shading ที่ช่วยหลีกเลี่ยงรอยพับในบางกรณี

### เป้าหมาย FPS {#target-fps}
เมื่อปิดใช้งาน Nomad จะซิงก์จำนวนเฟรมต่อวินาทีกับอัตรารีเฟรชของจอแสดงผล

เมื่อเปิดใช้งาน คุณสามารถตั้งค่าเฟรมต่อวินาทีที่ Nomad จะเรนเดอร์ได้

### อินเทอร์เฟซ {#debug-interface}
* `Top: layout` 
  * Collapse: บนอุปกรณ์ขนาดเล็ก แถบด้านบนจะยุบเป็นเมนูย่อยมากขึ้น 
  * Scroll: หากคุณคุ้นเคยกับ Nomad บนจอใหญ่และชอบเลย์เอาต์ปกติ การเปิดใช้งานตัวเลือกนี้จะใช้แถบด้านบนแบบกว้างมาตรฐาน และสามารถเลื่อนดูได้
  * Multiline: แสดงเมนูด้านบนทั้งหมดบนหลายบรรทัด
* `Bottom: draggable popup` - ทูลบาร์ด้านล่างมีปุ่มหลายปุ่มที่เปิด dialog หากเปิดใช้งานตัวเลือกนี้ dialog เหล่านั้นสามารถย้ายไปตำแหน่งอื่นบนหน้าจอได้  
* `Toolbox: show all` - Nomad จะซ่อนเครื่องมือที่ไม่เกี่ยวข้องกับการเลือกปัจจุบัน เช่น แปรงปั้นทั้งหมดจะถูกซ่อนบน primitive ที่ยังไม่ validate ตัวเลือกนี้จะทำให้เครื่องมือที่ใช้ไม่ได้จางลงแทนที่จะซ่อน
* `Toolbox: colored` - ใส่รหัสสีให้ไอคอน toolbox ตามประเภท
* `Panel: Drop shadows` - วาดเงาตกกระทบรอบเมนูและพาเนล บนอุปกรณ์เก่าบางรุ่นอาจกระทบประสิทธิภาพ
* `Panel: Blending` - ตัวเลือก debug
* `Pointer: hovering dot` - สำหรับอุปกรณ์ที่รองรับการ hover ของปากกา แสดงจุดเมื่อปากกาลอยเหนือเมนูและพาเนล

### GIF Turntable {#gif-turntable}
Nomad สามารถส่งออก gif แบบ turntable เคลื่อนไหวได้ โปรดทราบว่าด้วยข้อจำกัดของฟอร์แมต gif คุณภาพจะต่ำ การอัดวิดีโอหน้าจอมักเป็นวิธีที่ดีกว่า

* `Duration` - ระยะเวลาของ turntable เป็นวินาที
* `Rotation center` - จุดหมุนของกล้อง ดังนั้นคือส่วนของฉากที่กล้องจะหมุนรอบ
* `Transparent background` - ใช้ตัวเลือกพื้นหลังโปร่งใสสำหรับ gif โปรดทราบว่า gif รองรับความโปร่งใสแบบ 1 บิตเท่านั้น ขอบจึงอาจหยักมาก
* `Max frame sampling` - เอฟเฟกต์เรนเดอร์คุณภาพสูงหลายอย่างของ Nomad มาจากการรวมหลายเฟรมเข้าด้วยกัน สไลเดอร์นี้กำหนดจำนวนเฟรมที่จะรวม
* `Export (GIF)` - เริ่มกระบวนการส่งออก gif
