# เริ่มต้นใช้งาน {#getting-started}

## ยินดีต้อนรับสู่ Nomad! {#welcome-to-nomad}

Nomad เป็นแอปปั้นโมเดล 3 มิติที่ทำงานได้บนอุปกรณ์หลากหลายชนิด และทำงานได้ดีที่สุดบนแท็บเล็ตที่รองรับปากกากดน้ำหนักได้ เช่น Apple iPad กับ Apple Pencil หรือ Samsung Galaxy Tab พร้อมปากกา

แอปนี้ได้รับแรงบันดาลใจมาจากโปรแกรมปั้นโมเดลบนเดสก์ท็อปอย่าง ZBrush และ Blender โดยเน้นที่ส่วนติดต่อผู้ใช้ (UI) ที่เข้าใจง่าย โดยไม่ลดทอนฟีเจอร์ หากคุณเคยใช้แอปปั้นโมเดล 3 มิติมาก่อน Nomad จะให้ความรู้สึกคุ้นเคยมาก

ถ้านี่เป็นครั้งแรกที่คุณลองปั้นโมเดล 3 มิติ ก็ควรรู้พื้นฐานบางอย่างไว้ก่อน

::: tip ชอบดูวิดีโอมากกว่าหรือเปล่า?
ตอนนี้ใน Youtube มีวิดีโอสอนสำหรับผู้เริ่มต้นเยอะมาก นี่คือลิงก์ดี ๆ บางส่วน:

* [Nomad Sculpt Crash Course โดย Dave Reed](https://www.youtube.com/watch?v=69m86ICy2Q4)
* [Nomad Sculpt Beginner tutorial โดย Small Robot Studio](https://www.youtube.com/watch?v=J644wXoTiww)
* [NOMAD FOR BEGINNERS ซีรีส์โดย SouthernGFX](https://www.youtube.com/watch?v=bEh0WxRoOmg)

แนะนำให้ลองเข้าไปดูช่องหลักของครีเอเตอร์เหล่านี้ พวกเขามักจะลงบทเรียนใหม่ ๆ อยู่เสมอ
:::

## งานปั้นชิ้นแรกของคุณ {#your-first-sculpt}

เมื่อคุณเปิด Nomad ครั้งแรก คุณจะเห็นทรงกลมอยู่บนหน้าจอ เพียงลากปากกาบนทรงกลมนั้นเพื่อเริ่มปั้น โดยค่าเริ่มต้นจะเปิดใช้งานสมมาตร (Symmetry) เพื่อช่วยให้ปั้นได้ง่ายขึ้น

![](/videos/gettingstarted_01.mp4)

หากต้องการทำให้ขนาดแปรงใหญ่หรือเล็กลง ให้ใช้ตัวเลื่อนรัศมี (radius) ทางด้านซ้าย

![](/videos/gettingstarted_02.mp4)

หากต้องการทำให้แปรงแรงขึ้นหรือเบาลง ให้ใช้ตัวเลื่อนความเข้ม (intensity) ทางด้านซ้าย

![](/videos/gettingstarted_03.mp4)

เครื่องมือเริ่มต้นคือ `Clay tool` ซึ่งใช้เพิ่มปริมาตรบนผิวโมเดล หากต้องการลดปริมาตรออกจากผิว ให้แตะปุ่ม `Sub` ทางด้านซ้าย หากต้องการกลับมาเพิ่มปริมาตร ให้แตะปุ่ม Sub อีกครั้ง

![](/videos/gettingstarted_04.mp4)

หากต้องการเกลี่ยผิวให้เรียบ ให้แตะปุ่ม `Smooth` หากต้องการกลับไปปั้นตามปกติ ให้แตะปุ่ม Smooth อีกครั้ง

![](/videos/gettingstarted_05.mp4)

หากต้องการหมุนรอบโมเดล ให้ลากในพื้นที่ว่างนอกตัวโมเดล

![](/videos/gettingstarted_06.mp4)

หากต้องการซูม ให้ใช้ท่าทางสองนิ้วซูม/จีบนิ้ว

![](/videos/gettingstarted_07.mp4)

หากต้องการเลื่อนมุมกล้อง (pan) ให้แตะสองนิ้วบนหน้าจอแล้วลาก

![](/videos/gettingstarted_08.mp4)

หากคุณทำพลาด การแตะสองนิ้วหนึ่งครั้งจะเป็นการย้อนกลับ (undo) หรือใช้ปุ่ม undo ที่มุมล่างซ้ายก็ได้ 

![](/videos/gettingstarted_09.mp4)

::: tip เวอร์ชันเดสก์ท็อป
บนเดสก์ท็อป ปุ่ม alt/opt จะใช้ควบคุมกล้อง:

* tip ลากในพื้นที่ว่าง = หมุนกล้อง
* alt+tip ลาก = เลื่อนมุมกล้อง (pan)
* alt+tip ลาก แล้วปล่อยปุ่ม alt = ซูม (เหมือนใน zbrush)

สำหรับแท็บเล็ต Wacom ที่มีปุ่มบนปากกา 2 ปุ่มขึ้นไป ให้ตั้งค่าใน Wacom settings สำหรับปลายปากกาและปุ่มต่าง ๆ ดังนี้:

* tip = คลิกซ้าย
* rocker ล่าง = คลิกกลาง
* rocker บน = คลิกขวา

เมื่อใช้การตั้งค่าเหล่านี้ คุณจะควบคุมกล้องได้ด้วยปากกาเพียงอย่างเดียว:

* rocker บน + เลื่อนปากกาแบบ hover = หมุนกล้อง
* rocker ล่าง + เลื่อนปากกาแบบ hover = เลื่อนมุมกล้อง (pan)
:::

## เพิ่มสี {#add-color}

Nomad ให้คุณระบายสีบนผิวของงานปั้นได้ จากเมนูเครื่องมือทางด้านขวา ให้หาเครื่องมือ `Paint` แล้วคลิก เมื่อเลือกแล้ว จะมีทรงกลมสีปรากฏบนแถบเครื่องมือด้านซ้าย ให้คลิกที่มัน จะมีตัวเลือกสีปรากฏขึ้น เลือกสี แล้วระบายลงบนโมเดลของคุณ

![](/videos/gettingstarted_10.mp4)

หากต้องการลบสี ให้แตะปุ่ม `Erase` บนแถบเครื่องมือด้านซ้าย จากนั้นลบบนผิวโมเดล แตะปุ่ม erase อีกครั้งเพื่อกลับเข้าสู่โหมดระบายสี

![](/videos/gettingstarted_11.mp4)

ลองใช้แปรง clay ในโหมดเพิ่ม/ลด เกลี่ย (smooth) และระบายสี ดูว่าคุณจะสามารถทำหัวการ์ตูนง่าย ๆ ได้ไหม:

![](/images/gettingstarted_head1.webp)

## เครื่องมืออื่น ๆ {#other-tools}

ชุดเครื่องมือมีเครื่องมือมากมายเพื่อช่วยในการปั้น จนถึงตอนนี้คุณได้ใช้แปรง clay (เครื่องมือเริ่มต้น), smooth และ paint แล้ว เนื่องจาก smooth ถูกใช้บ่อย จึงมีปุ่มลัดพิเศษอยู่บนแถบด้านซ้าย

เครื่องมือใน Nomad สามารถทำได้หลากหลาย ตั้งแต่เครื่องมือปั้นอย่าง move, crease, inflate ไปจนถึงเครื่องมืออย่าง split และ trim ที่มีลักษณะคล้ายเครื่องมือช่างไม้หรือช่างโลหะ หน้า [Tools](tools.md) มีข้อมูลเพิ่มเติม

ลองใช้เครื่องมือ move, crease, inflate และ smooth เพื่อเพิ่มรายละเอียดให้หัวของคุณ และเปลี่ยนรูปทรงดู:

![](/images/gettingstarted_head2.webp)

ตอนนี้คุณรู้พื้นฐานของ Nomad แล้ว มาดูส่วนอื่น ๆ ของอินเทอร์เฟซกันต่อ

## อินเทอร์เฟซ {#interface}

![](/images/interface_overview1.webp)

* `เมนูด้านบน` - เมนูสำหรับเข้าถึงฟีเจอร์ส่วนใหญ่ของ Nomad เมนูมุมซ้ายบนจะเกี่ยวกับซีนและอ็อบเจ็กต์เป็นหลัก ส่วนเมนูมุมขวาบนจะเกี่ยวกับเครื่องมือ บนหน้าจอที่เล็กกว่านี้ เมนูเหล่านี้จะถูกรวมกันเพื่อประหยัดพื้นที่
* `Stats` - ข้อมูลเกี่ยวกับซีน อ็อบเจ็กต์ปัจจุบัน สถานะมาสก์ และการใช้หน่วยความจำ
* `Nav Cube` - ตัวช่วยบอกว่าคุณกำลังมองด้านไหนของงานปั้นอยู่ และเป็นทางลัดสำหรับกระโดดไปยังมุมมองต่าง ๆ การแตะที่ลูกบาศก์จะเปลี่ยนมุมมองไปยังด้านที่แตะ การลากลูกบาศก์จะหมุนมุมมอง แตะไอคอนเล็ก ๆ ข้างลูกบาศก์เพื่อจัดเฟรมอ็อบเจ็กต์ปัจจุบัน หรือรีเซ็ตกลับไปยังมุมมองเริ่มต้น
* `Toolbox` - เครื่องมือของ Nomad ทั้งหมดจะอยู่ในพื้นที่เลื่อนดูได้ส่วนนี้
* `แถบเครื่องมือด้านซ้าย` - ตัวเลื่อนสำหรับรัศมีและความเข้มของเครื่องมือส่วนใหญ่ ปุ่มเฉพาะตามบริบทสำหรับเครื่องมืออื่น ๆ และปุ่มลัดสำหรับ symmetry โหมด alt/sub ของเครื่องมือ การมาสก์ การเกลี่ย (smoothing) gizmo และตัวเลือกการระบายสี
* `แถบเครื่องมือด้านล่าง` - ปุ่มลัดสำหรับฟีเจอร์ที่ใช้บ่อย อธิบายเพิ่มเติมด้านล่าง

::: tip ถนัดซ้ายหรือเปล่า?
คุณสามารถสลับด้านและลำดับของแถบเครื่องมือทั้งหมดได้ ดูที่ [Mirror top bar](interface.md#mirror-top-bar) และตัวเลือกอื่น ๆ ที่เกี่ยวข้อง
:::

## แถบเครื่องมือด้านล่าง {#bottom-toolbar}

![](/images/interface_bottom_toolbar.webp)

* `Undo` - ย้อนกลับการทำงานล่าสุด
* `Redo` - ทำซ้ำการย้อนกลับครั้งล่าสุด
* `History` - เข้าถึงตัวเลือกประวัติการทำงาน อธิบายในเมนู [History](history.md)
* `Solo` - สลับระหว่างการแสดงเฉพาะอ็อบเจ็กต์ปัจจุบัน หรือแสดงทุกอ็อบเจ็กต์
* `X-Ray` - ทำให้อ็อบเจ็กต์อื่นทั้งหมดแสดงผลแบบเอกซเรย์ และทำให้อ็อบเจ็กต์ปัจจุบันทึบ การกดค้างหรือปัดขึ้นจะให้คุณตั้งค่าสีและความทึบของโหมดเอกซเรย์ได้
* `Voxel` - ปุ่มลัดสำหรับปุ่ม voxel remesh ของ [Voxel Remesher](topology.md#voxel-remesher) การกดค้างหรือปัดขึ้นจะเผยปุ่มลัดสำหรับตั้งค่าคุณภาพของ voxel remesh
* `Grid` - สลับการแสดงกริด การกดค้างหรือปัดขึ้นจะเผยตัวเลือกสำหรับกริด
* `Wire` - สลับการแสดงเส้นโครง (wireframe) ซ้อนทับ การกดค้างหรือปัดขึ้นจะเผยตัวเลือกสำหรับ wireframe
* `Inspect` - สลับการแสดงข้อมูลเพิ่มเติมเกี่ยวกับ mesh ปัจจุบัน โดยค่าเริ่มต้นจะแสดง UV แต่การกดค้างหรือปัดขึ้นจะให้คุณตรวจสอบคุณสมบัติอื่น ๆ ได้หากมีอยู่ และเลือกได้ว่าจะแสดงในพื้นหลังหรือบนตัว mesh

## ขั้นตอนถัดไป {#next-steps}

สิ่งที่คุณควรเรียนรู้ต่อไปขึ้นอยู่กับคุณ และสิ่งที่คุณสนใจ! นี่คือคำแนะนำบางอย่าง:

อยากเรียนรู้เพิ่มเติมเกี่ยวกับเครื่องมือปั้นหรือไม่? ไปที่ส่วน [Tools](tools.md)

อยากส่งออกโมเดลของคุณ? หรืออยากนำเข้าโมเดลมาปั้นต่อ? หรืออยากสร้างภาพจากงานปั้นของคุณ? ไปที่ส่วน [Files](files.md)

อยากเรียนรู้เพิ่มเติมเกี่ยวกับการควบคุมรายละเอียดบนงานปั้นของคุณหรือไม่? ไปที่ส่วน [Topology](topology.md) และเรียนรู้เกี่ยวกับ multires และ decimation

อยากทำงานกับอ็อบเจ็กต์หลายชิ้นไหม? รวมอ็อบเจ็กต์ง่าย ๆ และทรงพื้นฐานให้กลายเป็นซีนที่ใหญ่ขึ้น? ไปที่ส่วน [Scene](scene.md)

อยากเรียนรู้ *ทุกอย่าง* เกี่ยวกับ Nomad หรือเปล่า? เลือกได้ดีมาก! คู่มือนี้ครอบคลุมทุกอย่างของ Nomad มีทิปและเทคนิคมากมาย และมีฟังก์ชันค้นหาที่ดีมากอยู่ด้านบน ใช้เมนูนำทางทางด้านซ้ายเพื่อเรียนรู้เพิ่มเติม

ถ้าคุณชอบวิดีโอมากกว่า Holger Schönischka ได้ทำคลังทิปและเทคนิคสำหรับ Nomad ขนาดใหญ่บน Youtube: https://www.youtube.com/@ProcreateFX/videos
