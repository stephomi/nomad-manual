# ![](/icons/camera.webp) กล้อง

เมนูนี้ให้คุณสร้างและแก้ไขกล้อง รวมถึงควบคุมวิธีที่คุณโต้ตอบกับกล้อง

![](/images/camera_overview2.webp)

กล้องใน Nomad มีการใช้งานหลายอย่าง:

* ตั้งค่ามุมมองสำหรับปั้นจากมุมที่แม่นยำ
* ใช้เหมือนกล้องถ่ายรูปเพื่อจัดเฟรมวัตถุของคุณ
* ใช้เป็นกล้องมุมมองบุคคลที่หนึ่งเพื่อเดินสำรวจฉาก
* ใช้เป็นกล้องแบบฉายภาพขนาน (orthographic) สำหรับเกมมุมมองไอโซเมตริกหรือเรนเดอร์สไตล์งานอุตสาหกรรม

## การควบคุมกล้อง

### การหมุน
คุณหมุนกล้องได้โดยลาก *นิ้วเดียว* บนพื้นหลัง  
ถ้าคุณลากนิ้วบนโมเดล แทนที่จะหมุนกล้อง จะเริ่มการปั้นแทน

::: tip ฉันจะหมุนกล้องได้อย่างไรถ้าแตะพื้นหลังไม่ได้?
คุณสามารถวาง *สอง* นิ้วบนหน้าจอ – เหมือนกับจะเริ่มท่าทางแพน/ซูม – แล้วปล่อย *หนึ่ง* นิ้วออก
:::

### โฟกัส / รีเซ็ต
*แตะสองครั้ง* บนโมเดลเพื่อโฟกัสไปยังจุดที่เลือก  
ถ้าคุณ *แตะสองครั้ง* บนพื้นหลัง กล้องจะโฟกัสไปที่เมชที่ถูกเลือกแทน

### การเลื่อน (Translation)
โดยการขยับ *สอง* นิ้ว คุณสามารถแพนกล้องได้

### การซูม
โดยใช้ท่าทางหนีบนิ้ว (pinch) คุณสามารถซูมเข้า/ออกได้

### การหมุนเอียง (Rolling)
คุณสามารถ *หมุนเอียง* มุมมองได้โดยหมุน *สอง* นิ้ว
::: warning
ท่าทางนี้ใช้ได้เฉพาะกับโหมดหมุน `trackball` เท่านั้น
:::

### การควบคุมบนเดสก์ท็อป

บนเดสก์ท็อป ปุ่ม alt/opt ใช้สำหรับควบคุมกล้อง:

* ลากด้วย tip ในพื้นที่ว่าง = หมุนกล้อง
* alt+ลากด้วย tip = แพน
* alt+ลากด้วย tip แล้วปล่อย alt = ซูม (เหมือนใน zbrush)

สำหรับแท็บเล็ต Wacom ที่มีปุ่มบนปากกา 2 ปุ่มขึ้นไป ให้ใช้การตั้งค่า Wacom เพื่อกำหนด tip และปุ่มต่าง ๆ ดังนี้:

* tip = คลิกซ้าย
* rocker ล่าง = คลิกกลาง
* rocker บน = คลิกขวา

ด้วยการตั้งค่าเหล่านี้ คุณสามารถควบคุมกล้องด้วยปากกาเพียงอย่างเดียว:

* rocker บน + เลื่อนขณะลอยปากกา = หมุนกล้อง
* rocker ล่าง + เลื่อนขณะลอยปากกา = แพน

### ![](/icons/tool_view.webp) Add View
สร้างกล้องใหม่จากมุมมองปัจจุบัน

### ![](/icons/camera.webp) Icons

สลับการแสดงไอคอนกล้องในวิวพอร์ต ถ้ามีกล้องถูกเลือก ไอคอนของกล้องนั้นจะแสดงเสมอ

### Projection Type
คุณสามารถเปลี่ยน `Field of View` (FOV / ระยะโฟกัสเลนส์) ของกล้องได้  
โดยทั่วไปแนะนำให้ใช้ FOV ต่ำสำหรับการปั้น เพราะช่วยเรื่องสัดส่วนได้ดี  
คุณยังสามารถใช้โหมด `Orthographic` ซึ่งคล้ายกับการใช้ FOV เท่ากับ 0

### First Person
เปิดใช้การตั้งค่าจุดหมุน (pivot) ให้อยู่ที่ตัวกล้องโดยตรง แทนที่จะอยู่ที่ประติมากรรม  
การลากนิ้วบนพื้นหลังจะทำให้ตำแหน่งกล้องคงที่ แต่เปลี่ยนการหมุน แทนคล้ายกับเกมมุมมองบุคคลที่หนึ่ง  
มีประโยชน์เมื่อปั้นสภาพแวดล้อมมากกว่าวัตถุเดี่ยว

![](/images/camera_rotation_ortho_view.webp)

### Rotation Type
โดยค่าเริ่มต้น กล้องใช้โหมดหมุนแบบ `Turntable`  
หมายความว่าคุณมีอิสระในการหมุนสองแกน ใช้งานได้เข้าใจง่าย แต่ในบางกรณีคุณอาจต้องการความยืดหยุ่นมากขึ้น  
คุณสามารถสลับไปใช้ `Trackball` ซึ่งจะทำให้คุณ *หมุนเอียง* มุมมองได้โดยหมุน *สอง* นิ้วบนวิวพอร์ต  
บนเดสก์ท็อปมีโหมด trackball แบบทางเลือกที่อาจคุ้นเคยกับผู้ใช้บางคนมากกว่า

### Orthographic snap

เมื่อเปิดใช้แล้ว ถ้าคุณมีคีย์บอร์ด การกดปุ่ม Shift ค้างไว้ขณะหมุนมุมมองจะทำให้กล้องสแน็ปไปยังมุมมองด้านหน้า/หลัง/บน/ล่าง/ซ้าย/ขวาที่ใกล้ที่สุด และทำให้กล้องเป็นแบบ orthographic  
กล้องจะถูกตั้งเป็น orthographic ด้วยเมื่อคลิกที่ view cube เพื่อสแน็ปไปยังมุมมองหน้า/หลัง/ซ้าย/ขวา/บน/ล่าง

### Reset view

ย้ายกล้องไปด้านหน้า และจัดให้ฉากพอดีกับมุมมอง

### Snap view
สแน็ปไปยังมุมมองด้านหน้า/หลัง/ซ้าย/ขวา/บน/ล่างที่ใกล้ที่สุด  
ถ้าคุณอยู่ในหนึ่งในมุมมองเหล่านี้อยู่แล้ว การคลิกอีกครั้งจะหมุน 180 องศาไปยังด้านตรงข้าม

### Speed

ถ้าคุณรู้สึกว่ากล้องเคลื่อนที่ช้าหรือเร็วเกินไป คุณสามารถตั้งตัวคูณความเร็วสำหรับ `rotation`, `translation` และ `zooming` ได้  
มีประโยชน์ถ้าประติมากรรมของคุณมีขนาดใหญ่มากหรือเล็กมาก

### ภาพรวมของ Pivot

เมื่อคุณหมุนกล้อง คุณจะเห็นจุดสีชมพูเล็ก ๆ นี่คือจุดหมุน (pivot) ของกล้อง  
การเข้าใจว่าจุดหมุนอยู่ที่ไหนเป็นเรื่องสำคัญมาก เพื่อไม่ให้คุณหลงหรือหงุดหงิดกับกล้อง

โดยค่าเริ่มต้น pivot จะถูกอัปเดตผ่านการกระทำเหล่านี้:
- แตะสองครั้งบนโมเดล
- แตะสองครั้งบนพื้นหลัง (pivot ใหม่จะอยู่ที่กึ่งกลางของเมช)
- วาง *สอง* นิ้วบนหน้าจอ (แพน/ซูม/หมุนเอียง) จะอัปเดต pivot ไปที่กึ่งกลางของ *สอง* นิ้ว

### Update Pivot...

คุณสามารถปรับแต่งให้ pivot ถูกอัปเดตเพิ่มเติมด้วยตัวเลือกเหล่านี้:

* เมื่อกล้องเริ่มเคลื่อนที่
* อนุญาตให้ pivot อยู่ในอากาศ (air pivot)
* เมื่อแตะสองครั้ง
* ขณะปั้น (stroke)

::: tip
เมื่อคุณเริ่มชินแล้ว คุณสามารถซ่อนจุดสีชมพู (ตัวช่วย) ได้ในเมนู [Settings](settings.md)
:::

### แตะสองครั้งบนวัตถุ
เมื่อเปิดใช้ `Focus` การแตะสองครั้งจะย้าย pivot ไปยังวัตถุที่แตะ

### แตะสองครั้งบนพื้นหลัง
เมื่อเปิดใช้ จะตั้ง pivot ให้เป็นหนึ่งในตัวเลือก Selection, Scene หรือสลับไปมาระหว่างสองแบบนี้
