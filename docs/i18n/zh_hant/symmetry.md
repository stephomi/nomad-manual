# ![](/icons/symmetry.webp) 對稱

此選單用來控制筆劃如何在鏡像平面或放射狀方式重複，以及在非對稱物件上恢復對稱的方式。

![](/images/symmetry_overview.webp) 

## 概述
你可以用多種方式使用對稱：

* 作為鏡像，將工作沿著 X（左/右）、Y（上/下）、Z（後/前）或其組合翻轉。  
* 在 X、Y、Z 軸上設定放射對稱與重複次數，例如雕刻海星。  
* 鏡像可以繞著原點運作（稱為世界模式 World mode），或繞著物件中心運作（稱為本地模式 Local mode）。  
* 從非對稱開始的雕刻可以被強制變成對稱。

啟用/停用對稱的快捷方式也可以在左側快速面板中找到（「*Sym*」）。小小的「L/W」表示 Nomad 目前是本地（Local）或世界（World）對稱模式。你也可以長按或往螢幕中央滑動來叫出選單。

![](/images/symmetry_button.webp) 

這是一個全域選項，因此其狀態會在不同工具之間延續。  
唯一的例外是變形工具（[Move](#translate)、[Rotate](#rotate)、[Scale](#scale) 和 [Gizmo](#gizmo)），它們各自有自己的對稱狀態。

::: tip
對稱選單主要是用來控制筆劃對稱。你也可以透過[場景選單中的重複器](scene#repeaters)來鏡像與重複物件。
:::

## Enabled
切換鏡像模式，這與左側快速面板中的 `Sym` 按鈕相同。

## Planes

啟用對稱平面，以及設定放射對稱的重複次數。注意你不必只選擇單一平面，你可以啟用 1、2 或 3 個平面來達成複雜的對稱。

放射對稱的軸向與重複數量。注意這些同樣不受限於單一軸向，甚至可以與平面對稱結合使用，以非常快速地產生細緻結果。（總重複數量上限為 150）

![](/videos/symmetry_demo.mp4) 

## Method
鏡像可以是「Local」（本地），會隨物件一起移動；也可以是「World」（世界），不會移動。若你不確定需要哪一種模式，請觀察覆蓋在物件上的鏡像平面與放射指示器。在本地模式下，如果你使用變形 gizmo 移動模型，鏡像指示器也會跟著移動。在世界模式下，鏡像指示器會保持固定，而物件會在其中滑動。

## Mirroring
![](/images/symmetry_mirroring.webp)

在靠近對稱平面雕刻時，有些筆刷的對稱行為可能不夠完美。此區可以讓你透過將雕刻的一側複製到另一側來恢復對稱。

`Direction` - `<<` 與 `>>` 按鈕決定哪一側會被複製到另一側。Nomad 會依照你目前的視角來計算，因此例如設定為 `<<` 時，會永遠從螢幕右側複製到螢幕左側。

![](/icons/shield.webp) `Mask` - 遮罩按鈕可讓你限定要被鏡像的區域；遮罩目標側會保護該區域不被覆蓋，遮罩來源側則會阻止該區域被鏡像到目標側。 

![](/icons/tool_hide.webp) `Hide` - 啟用時，來源側被隱藏的區域將不會被鏡像到目標側。 

`Mirror` 會嘗試判斷鏡像兩側的拓樸是否相同，若相同，就只移動頂點。這是較常見的情況。

`Split & Mirror` 基本上會沿著鏡像平面切開物件，複製一側，將其鏡像到另一側，並在鏡像平面上焊接頂點。這是較具破壞性的選項，會刪除多重細分（multiresolution），但在模型兩側差異很大時，有時必須使用此方法。

### Flip object
![](/images/symmetry_flip.webp)
將左側變成右側，右側變成左側。外觀上類似使用 gizmo 工具並將 X 軸縮放設為 -1 的效果。

## Reset and Edit

![](/images/symmetry_edit.webp)

可以編輯對稱的位置與方向（但不建議這麼做！）。若有需要，`World center` 與 `Orientation` 會將對稱的位置與方向重設為預設值。

Nomad 通常知道該把對稱平面放在哪裡。不建議手動調整，但 `Gizmo (Edit)` 按鈕允許進階使用者這麼做。點擊此按鈕後，會顯示一個 gizmo，讓你平移與旋轉對稱平面。若你想恢復預設中心與方向，可以使用 `object center` 與 `orientation` 按鈕。

這些選項的行為會依你所在的空間（*Local/World*）而改變。  
如果結果與預期不同，請先確認你是否在正確的空間中。

::: tip
`Gizmo (Edit)` 按鈕被刻意顯示為灰色，以提醒你大概不應該使用它！
:::

## Show options
![](/images/symmetry_show.webp)


`Show line` 與 `Show plane` 會切換顯示對稱位置的視窗疊加圖層。注意關閉這些選項後，只有在關閉對稱選單時才會真正生效。
