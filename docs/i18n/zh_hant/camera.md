# ![](/icons/camera.webp) 相機

此選單可讓你建立與修改相機，以及控制你如何與相機互動。

![](/images/camera_overview2.webp)

Nomad 中的相機有多種用途：

* 設定精準角度的雕刻視角
* 像照相機一樣用來構圖你的物件
* 作為第一人稱視角相機來在場景中導航
* 作為正交相機，用於等角遊戲或工業風格的渲染

## 控制相機

### 旋轉
在背景上拖曳「一根」手指即可旋轉相機。  
如果你在模型上拖曳手指，則會開始雕刻操作。

::: tip 我如果碰不到背景，還能旋轉相機嗎？
可以，你可以先在螢幕上放「兩根」手指——就像要開始平移/縮放手勢一樣——然後放開「一根」手指。
:::

### 對焦 / 重設
在模型上「連點兩下」即可對焦到點選的位置。  
如果你在背景上「連點兩下」，相機會對焦到被選取的網格。

### 平移
用「兩根」手指移動，即可平移相機。

### 縮放
使用雙指捏合手勢即可放大/縮小。

### 滾動
用「兩根」手指旋轉即可「滾動」視角。
::: warning
此手勢僅在 `trackball` 旋轉模式下可用。
:::

### 桌面版控制

在桌面版中，使用 Alt/Opt 鍵來控制相機：

* tip 在空白處拖曳 = 旋轉相機
* alt+tip 拖曳 = 平移
* alt+tip 拖曳，然後放開 alt = 縮放（與 ZBrush 相同）

若使用有兩個或以上按鍵的 Wacom 手寫筆，可在 Wacom 設定中將筆尖與按鍵設定如下：

* tip = 左鍵點擊
* 下方搖桿 = 中鍵點擊
* 上方搖桿 = 右鍵點擊

使用上述設定後，你可以只用手寫筆來操作相機：

* 上方搖桿 + 懸浮移動 = 旋轉相機
* 下方搖桿 + 懸浮移動 = 平移

## 相機控制

![](/images/camera_list.webp)

### 視角
你可以使用 `Add View` 來儲存相機視角。  
如果你點擊視角名稱，相機就會還原到該視角。

::: tip
儲存的視角會一併儲存 [Projection Type](#projection-type) 設定以及 [Reference image](background.md)。  
如果你想在不同背景的前/左/後參考視圖之間切換，這會很有用。
:::

|   Action   |              Icon              | Description                                                                  |
| :--------: | :----------------------------: | :--------------------------------------------------------------------------: |
| Visibility | ![](/icons/eye_open.webp)    | 切換相機顯示。被隱藏的相機會在上一個/下一個按鈕中被略過                         |
|    Name    |                                | 選取相機                                                                     |
|   Image    | ![](/icons/image.webp)       | 若有連結參考圖，則顯示其縮圖                                                 |
| Update View| ![](/icons/update_view.webp) | 以目前視角更新已儲存的視角                                                   |
| Edit Name  | ![](/icons/pencil.webp)      | 編輯相機名稱                                                                 |
|  Delete    | ![](/icons/trash.webp)       | 刪除相機                                                                     |

### ![](/icons/tool_view.webp) Add View
以目前視角建立一個新的相機。

### ![](/icons/camera.webp) Icons

切換是否在視窗中顯示相機圖示。若某個相機被選取，其圖示會始終顯示。

### Projection Type
你可以調整相機的 `Field of View`（視角 / 焦距）。  
一般建議在雕刻時使用較小的視角，有助於掌握比例。  
你也可以使用 `Orthographic` 模式，大致相當於視角為 0。

### First Person
啟用後，樞紐點會直接設在相機上，而不是雕塑上。  
在背景上拖曳手指時，相機位置會被鎖定，只改變旋轉，就像第一人稱遊戲一樣。  
在雕刻環境而非單一物件時特別實用。

![](/images/camera_rotation_ortho_view.webp)

### Rotation Type
預設情況下，相機使用 `Turntable` 旋轉模式。  
這表示你只有兩個自由度，較直覺，但在某些情況下你可能需要更大的彈性。  
你可以切換到 `Trackball`，此時你可以在視窗上用「兩根」手指旋轉來「滾動」視角。  
在桌面版上還有另一種 trackball 模式，對某些使用者來說可能更熟悉。

### Orthographic snap

啟用後，如果你有鍵盤，在旋轉視角時按住 Shift，會將相機吸附到最近的前/後/上/下/左/右視角，並將相機設為正交。  
當點擊視角方塊以吸附到前/後/左/右/上/下視角時，相機也會被設為正交。

### Reset view

將相機移到正面，並讓場景適配到視窗中。

### Snap view
吸附到最近的前/後/左/右/上/下視角。  
如果你已經在其中一個視角，再次點擊會旋轉 180 度到相反的一側。

### Speed

如果你覺得相機移動太慢或太快，可以為 `rotation`、`translation` 和 `zooming` 設定速度倍率。  
在你的雕塑非常大或非常小時特別有用。

### Pivot overview

當你旋轉相機時，可以看到一個小小的粉紅點，那就是相機的樞紐點。  
了解樞紐點的位置非常重要，否則你可能會因為相機而迷路或感到挫折。

預設情況下，樞紐點會透過以下操作更新：
- 在模型上連點兩下
- 在背景上連點兩下（新的樞紐點會在網格中心）
- 放「兩根」手指在螢幕上（平移/縮放/滾動）會將樞紐點更新到「兩根」手指中心

### Update Pivot...

你可以進一步自訂在以下情況更新樞紐點：

* When camera starts moving 
* Allow air pivot
* When double tapping
* Sculpt (stroke)

::: tip
當你習慣之後，可以到 [Settings](settings.md) 選單中隱藏這個提示用的粉紅點。
:::

### Double tap on object
當 `Focus` 啟用時，在物件上連點兩下會將樞紐點移到點擊的物件上。

### Double tap on background
啟用後，會將樞紐點設為「Selection」、「Scene」，或在兩者之間切換。
