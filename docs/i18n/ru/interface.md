# ![](/icons/interface.webp) Меню интерфейса {#interface-menu}

Это меню управляет множеством параметров для настройки интерфейса Nomad. 

![](/images/interface_overview2.webp)

Nomad можно настраивать довольно глубоко, это меню разделено на 4 секции: Interface, Gesture, Bindings, Debug.

![](/images/interface_menu.webp)


::: tip
Эта страница относится к меню интерфейса, а не к самому интерфейсу! Общий интерфейс описан в разделе [Getting Started](gettingstarted.md).
:::

## Интерфейс {#interface}

Раздел Interface позволяет добавлять ярлыки, создавать плавающие панели инструментов и управлять цветом, размером и внешним видом пользовательского интерфейса Nomad.

![](/images/interface_overview3.webp)

### Добавить ярлыки (снизу)... {#add-shortcuts-bottom}
![](/images/interface_shortcuts.webp)

В нижней панели инструментов по умолчанию включены следующие ярлыки:
* `Undo` - отменить предыдущее действие
* `Redo` - восстановить ранее отменённое действие
* `Solo` - Временно скрыть все остальные объекты, оставив видимым только выбранный. Нажмите ещё раз, чтобы восстановить все объекты.
* `X-ray` - Временно сделать все остальные объекты полупрозрачными, оставив выбранный объект непрозрачным. Нажмите ещё раз, чтобы восстановить стандартные материалы.
* `Voxel remesh` - Перестроить сетку текущего объекта, используя последнее использованное воксельное разрешение. Долгое нажатие или свайп вверх откроет ползунок разрешения и переключатель резких краёв.
* `Grid` - Переключить отображение сетки. Долгое нажатие или свайп вверх позволит изменить цвет и непрозрачность сетки.
* `Wireframe` - Переключить наложение каркаса (wireframe). Долгое нажатие или свайп вверх позволит изменить цвет и непрозрачность каркаса.
* `Inspector` - позволяет просматривать свойства вашей сетки, такие как UV, запечённые текстуры и другие параметры, отображаемые на фоне Nomad.
* `Face Group` - Переключить наложение facegroup, подробнее в [Tools->FaceGroup](tools.md#facegroup) 

Другие часто используемые ярлыки доступны из этого меню, ещё больше можно найти в разделе bindings.

#### ![](/icons/more.webp) Привязки {#bindings-list}

Практически любую функцию Nomad можно добавить на панель ярлыков через кнопку bindings. При нажатии на кнопку откроется меню привязок:

![](/images/interface_bindings_search.webp)

Вы можете искать по категориям с помощью значков вверху или использовать поле поиска для поиска функций по имени. Нажмите на функцию, чтобы добавить её на панель. Нажмите ещё раз, чтобы удалить.

#### ![](/icons/list.webp) Порядок {#order}

Отображает список ярлыков. Долгое нажатие и перетаскивание позволяют изменить порядок ярлыков.

#### ![](/icons/reset.webp) Сброс {#reset}

Reset восстановит нижнюю панель инструментов к настройкам по умолчанию.

### Добавить ярлыки (окно)... + {#add-shortcuts-window}
![](/images/interface_add_shortcuts_window.webp)

Нажатие на + добавит плавающую панель инструментов. Она не будет видна, пока вы не нажмёте кнопку bindings и не добавите в неё ярлыки, после чего её можно будет сделать видимой. 

Вы можете создать несколько панелей, у каждой панели в этом меню есть следующие параметры:

* ![](/icons/checked.webp) `Visible` - Переключить видимость панели
* ![](/icons/more.webp)`Bindings` - Показать окно привязок для выбора функций, которые нужно добавить или удалить с панели.
* ![](/icons/list.webp)`Order` - Показать меню для изменения порядка элементов панели.
* ![](/icons/reset.webp) `Reset` - Сбросить панель к состоянию по умолчанию.
* ![](/icons/resize_bottom_right.webp) `Resize corner` - Переключить ручку изменения размера в правом нижнем углу панели.
* ![](/icons/sort_down.webp) `Collapsable` - Переключить ручку сворачивания в правом верхнем углу.
* ![](/icons/trash.webp) `Delete` - Удалить панель.

### Панель инструментов {#toolbox}

Параметры меню инструментов справа в интерфейсе Nomad.

![](/images/interface_toolbox.webp)

#### ![](/icons/resize_bottom_right.webp) Угол изменения UI {#ui-resize-corner}

Переключить ручку изменения размера в нижнем углу панели инструментов.

#### Скрыто {#hidden}
Обычно значок toolbox в верхней панели переключает отображение между длинной одной колонкой и многострочным списком инструментов. Эта опция переключает между многострочным списком и полным скрытием.

#### Цветные {#colored}
Цветовое кодирование значков по категориям, например все инструменты маски коричневые, инструменты разрезания красные, инструменты выравнивания зелёные и т.д.

#### Строки: Авто (>1) {#rows-auto-1}

#### Сбросить порядок {#reset-order}
Сбросить порядок стандартных инструментов в toolbox к значению по умолчанию. Пользовательские значки останутся в toolbox в конце списка.


### Пресеты {#presets}

![](/images/interface_presets.webp)

Набор цветовых пресетов для интерфейса.

#### Контрастная кнопка {#high-contrast-button}
Альтернативный стиль кнопок, делающий их более заметными, когда они включены. При значении Auto Nomad будет использовать этот режим, когда контраст цветов UI между включённым/выключенным состоянием низкий.

#### Виджет цвета/Базовый цвет {#color-widgetcolor-base}
Основные цвета, используемые в интерфейсе.

#### Прозрачная панель, Цвет панели, Сила размытия {#transparent-panel-color-panel-blur-strength}
![](/images/interface_transparent.webp)
При включении появятся дополнительные параметры для управления внешним видом меню и панелей в Nomad. Можно настроить их цвет, прозрачность и силу размытия.

::: tip
На небольших устройствах бывает полезно сделать панель цвета почти белой, прозрачной и с низкой силой размытия, чтобы меню не закрывали сцену.
:::

## Жесты {#gesture}
Меню Gesture управляет тем, как стилус и касания пальцами управляют Nomad.

![](/images/interface_gesture.webp)

### Параметры жестов {#gesture-options}
![](/images/interface_gesture_matrix.webp)

Nomad может ограничивать операции в зависимости от устройства ввода. Например, перетаскивание пальцем может только двигать камеру, а перетаскивание стилусом — только скульптить. Если у вас есть мышь или трекпад, их также можно назначить для управления определёнными операциями.

В настоящее время Nomad позволяет задавать следующие режимы для любого сочетания жестов пальцем, стилусом или мышью:

* Camera
* Sculpt
* Gizmo
* Material picking (через долгое нажатие)
* Select object


`Finger always smooths` - Smooth можно настроить так, чтобы он работал только при перетаскивании пальцем.

### ![](/icons/tool_mask.webp) Маска {#mask}

![](/images/interface_gesture_mask.webp)

`One tap shortcuts` - Включить следующие однократные жесты без необходимости удерживать кнопку mask. Это позволит использовать следующие жесты:
* нажатие на фон — инвертировать маску
* нажатие на замаскированную область — размыть маску
* нажатие на незамаскированную область — сделать маску резче

### Переключить Маска <-> SelMask {#toggle-mask-selmask}
![](/images/interface_gesture_togglemask.webp)
* `Stroke` - Долгое нажатие переключит между Mask и SelMask и начнёт новый штрих. В конце штриха будет снова выбран предыдущий инструмент. 
* `Tool` - Долгое нажатие и отпускание без движения переключит между Mask и SelMask. 

### ![](/icons/tool_hide.webp) Скрыть {#hide}
![](/images/interface_gesture_hide.webp)

`One tap shortcuts` включает следующие ярлыки для инструмента hide:
* Нажмите на face group, чтобы скрыть её
* Нажмите в пустом пространстве, чтобы инвертировать скрытые полигоны

### ![](/icons/finger3.webp) Три пальца {#three-fingers}
![](/images/interface_gesture_threefingers.webp)

Если ваше устройство поддерживает жесты тремя пальцами, настройте ярлыки для этого жеста. 

Матрица опций позволяет задать вертикальные и горизонтальные перетаскивания как отдельные ярлыки. Обратите внимание, что если один и тот же жест выбран для 2 опций, одна из них будет отключена.

* `Rotate lighting` - Вращать окружение, источники света и Matcap.
* `Tool Radius` - Изменять радиус инструмента.
* `Tool Intensity` - Изменять интенсивность инструмента. 

### ![](/icons/fingerprint.webp) История 2/3 {#history-23}
`History shortcuts` - при включении активируются следующие жесты:
* Undo - касание двумя пальцами
* Redo - касание тремя пальцами

`Long press` - при включении удерживание 2/3 пальцев будет быстро выполнять undo/redo.

### Специальные возможности {#accessibility}

![](/images/interface_gesture_accessibility.webp)

`Assistive window` откроет плавающую панель для управления операциями drag, pinch, roll и камерой.

### Камера {#camera}
Ярлык для перехода в меню `Camera` (параметры камеры раньше находились здесь, но были перенесены в меню камеры).

### Двойное касание Pencil -> Привязки {#pencil-tap}

Ярлык для перехода в меню `Bindings` (опции одиночного и двойного нажатия Pencil раньше находились здесь, но были перенесены в меню bindings).


## Привязки {#bindings}
Клавиатурные и кнопочные ярлыки можно задать в меню bindings:

![](/images/interface_bindings.webp)
Почти все функции в Nomad можно привязать к клавиатурным сочетаниям, если у вашего устройства есть клавиатура, или к дополнительным кнопкам стилуса, а также к геймпадам.

Чтобы создать привязку, нажмите на прямоугольник рядом с функцией и нажмите нужную клавишу/кнопку. 

Ищите функции по значкам категорий вверху или через строку поиска по имени. 

Отдельные привязки можно отключить с помощью чекбокса рядом с именем привязки.

### ![](/icons/more.webp) Контекстное меню {#context-menu}
Значок ![](/icons/more.webp) после каждой привязки открывает контекстное меню:

![](/images/interface_bindings_context.webp)

* `Clone` - Клонировать привязку
* `Reset` - Сбросить привязку
* `Delete` - Удалить привязку
* `Toggle shortcut on key tap` - Настроить, будут ли короткое и долгое нажатия обрабатываться по-разному. При включении короткое нажатие активирует инструмент. Долгое нажатие активирует инструмент только пока клавиша удерживается, при отпускании будет возвращён предыдущий инструмент. Иногда это называют «липкими клавишами» (sticky keys) в других 3D-приложениях.

### Расширенные {#advanced}
Внизу меню bindings находится значок шестерёнки для дополнительных параметров:

![](/images/interface_bindings_advanced.webp)

* `Toggle shortcut on key tap` - Короткое нажатие стандартных ярлыков для mask, smooth, gizmo, hide, sub будет переключать в этот режим, но удерживание клавиши будет временно переключать в этот режим, а при отпускании режим вернётся к предыдущему. Иногда это называют «липкими клавишами» (sticky keys) в других 3D-приложениях.
* `Toggle tool shortcuts` - При использовании одного из ярлыков инструмента, если один и тот же ярлык нажать дважды, он переключит на предыдущий инструмент. Таким образом можно постоянно переключаться между двумя инструментами одной и той же горячей клавишей.
* `Invert Y (Zooming)` - Инвертировать направление зума.
* `Reset bindings` - Сбросить все привязки к значениям по умолчанию.

## Отладка {#debug}
Некоторые экспериментальные и отладочные параметры находятся в этом меню. Многие из этих опций лучше оставлять по умолчанию и изменять только после обращения в поддержку Nomad.

![](/images/interface_debug.webp)
### UV {#uv}
* `Normalize Uvs` - Nomad нормализует UV внутри тайла [0-1].

### Quad Remesh {#quad-remesh}
* `Instant Mesh` - Включить алгоритм instant remesh
* `Quadriflow` - Альтернативный метод quad remesh.

### Рендер {#render}
* `Heightmap` - Переключить окно просмотра на рендер глубины сцены. Это можно использовать для создания альфа-карт для кистей.
* `Refraction write depth (back)` - Обратная сторона преломляющих (refraction) мешей будет записываться в буфер глубины.
* `Flip Y (normal map)` - Инвертировать канал Y при запекании normal map, требуется для некоторых игровых и рендер-движков.
* `Angle weighted smooth normals` - Корректировка работы сглаженного шейдинга, позволяющая избежать заломов в некоторых случаях.

### Целевая частота кадров {#target-fps}
При отключении Nomad будет синхронизировать количество кадров в секунду с частотой обновления вашего дисплея.

При включении вы можете задать количество кадров в секунду, которое будет рендерить Nomad.

### Интерфейс {#debug-interface}
* `Top: layout` 
  * Collapse: На небольших устройствах верхняя панель будет сворачиваться в дополнительные подменю. 
  * Scroll: Если вы привыкли к Nomad на больших экранах и предпочитаете обычный макет, включение этой опции вернёт стандартную широкую верхнюю панель, которую можно прокручивать.
  * Multiline: Отображать всё верхнее меню в несколько строк.
* `Bottom: draggable popup` - В нижней панели есть несколько кнопок, открывающих диалог. Если эта опция включена, эти диалоги можно перемещать по экрану.  
* `Toolbox: show all` - Nomad скрывает инструменты, не относящиеся к текущему выбору, например все кисти скульптинга скрыты на примитивах, которые не валидированы. Эта опция будет затемнять неактивные инструменты вместо их скрытия.
* `Toolbox: colored` - Цветовое кодирование значков toolbox в зависимости от их типа.
* `Panel: Drop shadows` - Рисовать тени вокруг меню и панелей. На некоторых старых устройствах это может влиять на производительность.
* `Panel: Blending` - Отладочная опция
* `Pointer: hovering dot` - Для устройств, поддерживающих наведение стилуса, отображать точку при наведении стилуса на меню и панели.

### Gif-вертушка {#gif-turntable}
Nomad может экспортировать анимированный gif-«вертолет» (turntable). Обратите внимание, что из-за ограничений формата gif качество низкое. Обычно лучше использовать запись экрана.

* `Duration` - продолжительность вертела в секундах
* `Rotation center` - центр вращения камеры, то есть часть сцены, вокруг которой будет вращаться камера
* `Transparent background` - Использовать прозрачный фон для gif. Учтите, что gif поддерживает только 1-битную прозрачность, поэтому края могут быть сильно зазубренными.
* `Max frame sampling` - Многие эффекты высококачественного рендера Nomad достигаются путём комбинирования нескольких кадров. Этот ползунок задаёт, сколько кадров комбинировать.
* `Export (GIF)` - начать процесс экспорта gif.

### Постобработка {#post-process}
* `Filtering` - Отладочная опция
* `Format` - Отладочная опция
* `Buffer reuse` - Отладочная опция

### Производительность {#performance}
* `Multicore general` - Отладочная опция
* `Multicore sculpting` - Отладочная опция
* `Partial Drawing` - Экспериментальная функция! Используйте, если вы скульптите относительно небольшую часть высокополигональной сетки. Это должно сделать скульптинг более плавным, но не включайте wireframe! Также это может вызывать визуальные артефакты во время мазков кистью.

### Возможности {#feature}
* `Flip quad split (on tap)` -  Отладочная опция
* `Join: merge radius` - Вершины будут автоматически сварены, если они достаточно близко при объединении мешей. Вы можете настроить радиус этим ползунком.

### Отладка {#dev}
* `Logs` - Открыть окно логов
* `App review popup` - Отладочная опция
* `FPS` - добавить счётчик кадров в секунду в статистику окна просмотра.
