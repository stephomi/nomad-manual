# ![](/icons/history.webp) История
![](/images/history_overview.webp)

Как и в большинстве инструментов для создания контента, в Nomad можно отменять и повторять все действия редактирования.
Существует ограничение на количество операций, которые можно отменить, но вы можете управлять этим поведением.

::: tip
Вы можете использовать быстрые жесты для отмены/повтора:
- касание двумя пальцами — отмена
- касание тремя пальцами — повтор
:::

## История
![](/images/history_history.webp)

Эта панель отображает стек истории, показывая количество шагов, название операции и объём памяти, который использует этот шаг.

## Настройки
![](/images/history_settings.webp)

### Лимит истории (Мб)
Если стек истории превышает это значение, более старые операции будут удалены, чтобы уложиться в заданный лимит памяти.


### Максимум отменяемых операций
Вы можете управлять максимальным количеством операций.

## Восстанавливать камеру
Для каждой операции сохраняется точка обзора камеры.
Если включить эту опцию, при отмене или повторе операции камера будет возвращаться к сохранённой точке обзора.

## Включать действия

* `Lights` — если отключено, операции с источниками света (кроме перемещений с помощью гизма) будут игнорироваться стеком истории
* `Matcaps & HDRIs` — если отключено, изменения маткапов и HDRI будут игнорироваться стеком истории
* `PostProcess` — если отключено, изменения параметров постобработки будут игнорироваться стеком истории
