# ![](/icons/layer.webp) 레이어 {#layers}

이 메뉴에는 레이어 스택이 포함되어 있습니다. 이는 오브젝트에 대한 편집 내용을 비파괴적인 방식으로 저장하는 방법이며, 레이어를 결합하고 재활용하는 다양한 기능을 제공합니다.

![](/images/layers_overview.webp) 

## 개요 {#overview}

Nomad의 레이어는 여러 가지 용도로 사용됩니다.

페인트 정보(색상, 거칠기, 금속성, 불투명도)는 2D 페인트 애플리케이션과 유사한 방식으로 레이어와 함께 작동합니다. 레이어를 생성하고 모델에 페인트를 적용할 수 있습니다. 레이어는 켜거나 끌 수 있고, 불투명도를 조절할 수 있으며, 복제할 수 있고, 레이어 스택에서 순서를 변경할 수 있으며, 블렌딩 모드를 조정할 수 있습니다.

Nomad는 또한 조각(sculpting)을 위해 레이어를 사용합니다. 레이어는 주름과 같은 미세한 디테일을 저장할 수도 있고, 큰 변형을 저장할 수도 있어 다른 3D 애플리케이션의 블렌드셰이프/쉐이프 키/모프 타깃처럼 동작하게 할 수 있습니다. 예를 들어 서로 다른 표정들을 각각 다른 레이어에 조각한 뒤, 레이어 슬라이더를 조절해 새로운 표정으로 조합할 수 있습니다.

이 경우 레이어에 저장되는 변경 사항은 순수하게 가산(additive) 방식이므로, 페인트와 달리 레이어 순서는 중요하지 않습니다.

레이어는 [Delete Layer](tools.md#delete-layer) 도구를 통해 부분적으로 지울 수 있으며, 레이어에 저장된 다양한 정보를 기반으로 마스크를 생성하는 데에도 사용할 수 있습니다.

![](/videos/layer.mp4)

::: tip
대부분의 스컬핑 소프트웨어와 달리, 메시의 토폴로지를 변경해도 레이어가 삭제되지 않습니다. [Voxel Remesher](topology.md#voxel-remesher), [Multiresolution](topology.md#multires) 또는 [Trim](tools.md#trim)/[Split](tools.md#split) 도구를 사용할 수 있지만, [Voxel Remesher](topology.md#voxel-remesher)를 사용할 경우 레이어의 품질에 영향이 있을 수 있다는 점에 유의하세요.
:::

::: tip
레이어를 블렌드셰이프/모프 타깃으로 사용할 경우, [씬 메뉴](scene.md#object-menu)에 추가 레이어 기능이 있습니다. 레이어를 오브젝트로 분리하거나, 일치하는 오브젝트들을 레이어로 결합할 수 있습니다.
:::

## 레이어 메뉴 {#layer-menu}

![](/images/layers_menu.webp)

`Add layer`를 눌러 새 레이어를 생성합니다.

각 레이어에는 이름, 강도/계수를 제어하는 슬라이더, 그리고 옵션 버튼이 있습니다.

### 옵션 {#options}

| Action       | Icon                         | Description                                         |
| :----------: | :--------------------------: | :-------------------------------------------------  |
| Visible      | ![](/icons/eye_open.webp)   | 레이어를 표시/숨기기                               |
| Blend Mode   | ![](/icons/opacity.webp)    | 포토샵 스타일의 블렌딩 모드. 4개의 아이콘은 색상, 거칠기, 금속성, 불투명도에 대한 모드를 표시합니다. |
| Edit Name    | ![](/icons/pencil.webp)     | 레이어 이름 편집                                   |
| Delete       | ![](/icons/trash.webp)      | 레이어 삭제                                        |
| Duplicate    | ![](/icons/clone.webp)      | 레이어 복제                                        |
| Merge Down   | ![](/icons/merge_down.webp) | 레이어를 아래 레이어(또는 기본 메시)와 병합       |

레이어를 레이어 스택의 다른 위치로 이동하려면, 이름을 길게 누른 후 드래그합니다.

### 더 보기... {#more}

'More...' 버튼을 누르면 현재 레이어에 대한 추가 옵션이 표시됩니다:

![](/images/layers_more.webp) 

#### 채널 계수 {#channel-factors}

이 컨트롤을 사용하면 해당 레이어의 조각/색상/거칠기/금속성/불투명도의 양을 스케일링할 수 있습니다. 이 값들은 레이어 강도 슬라이더 값에 곱해지므로, 예를 들어 레이어 강도가 1이고 색상 채널 계수가 0.5라면, 표시되는 색상 강도는 0.5가 됩니다.

`Offset`은 레이어의 조각 강도를 제어합니다. 색상/거칠기/금속성은 0과 1 사이로 제한되지만, Offset은 양수와 음수를 포함해 어떤 값이든 될 수 있습니다. 

::: tip
Offset을 사용하면 범프 레이어를 캐비티 레이어로 바꾸거나, 웃는 표정을 찡그린 표정으로 바꿀 수 있습니다:
![](/videos/layer_happysad.mp4)


대칭적인 레이어는 복제한 뒤 del layer로 좌/우 형태로 분리할 수 있습니다:
![](/videos/layer_leftright.mp4)

음수 Offset 계수를 가진 레이어를 비어 있는 레이어로 베이크한 뒤 새로운 양수 형태를 만들 수 있습니다.

1보다 큰 양수 Offset 계수를 가진 레이어는 더 극단적인 블렌드셰이프를 실험하는 데 사용할 수 있습니다.
:::

::: warning
현재 레이어는 세 채널(색상/금속성/거칠기)에 대해 하나의 불투명도 채널만 공유합니다.  
채널별 강도가 최대값이 아닌 여러 레이어를 병합하면, 최종 결과가 다르게 보일 수 있습니다.

향후에는 각 채널이 자체 알파 채널을 가지도록 하여 이 제한을 제거할 수 있습니다.
:::


#### ![](/icons/tool_mask.webp) 마스크 {#mask}
각 슬라이더 옆의 마스크 버튼은 해당 채널로부터 마스크를 생성합니다. 페인팅 애플리케이션에서 레이어를 선택 영역으로 사용하는 것과 유사하게, 레이어에서 수행한 작업을 다른 작업에 재사용할 수 있습니다.

#### ![](/icons/preview.webp) 미리보기 {#preview}
![](/images/layers_preview.webp) 

활성화하면 이 레이어에 대한 추출 설정을 미리 봅니다(다음 섹션 참조).

X-ray가 활성화되면, 추출된 형태만 실체를 가지며 나머지 형태는 투명해집니다. 이는 음수 추출 높이를 사용할 때 유용합니다.

#### 추출 {#extract}
![](/images/layers_extract.webp) 

![](/videos/layer_shell.mp4)

`Extract` 버튼은 레이어의 내용을 새 오브젝트로 복제하며, 일반적으로 다음 섹션에서 설정한 두께를 사용합니다.

`Closing action`은 복제가 어떻게 이루어지는지를 결정합니다:

* None - 해당 부분만 추출하고, 측면을 생성하거나 구멍을 메우려 하지 않습니다.
* Fill - 구멍을 삼각형으로 채우고 부드럽게 만듭니다. 평평한 표면에는 이 옵션을 사용하지 마세요.
* Shell - 두께 값과 방향 옵션을 사용해 추출된 형태를 닫습니다.
* Layer - 레이어 차이를 추출합니다.

#### ![](/icons/height.webp) 두께 {#thickness}
![](/images/layers_thickness.webp) 

쉘(껍질) 익스트루전의 깊이입니다. 양수 값은 표면 밖으로, 음수 값은 표면 안쪽으로 성장합니다.

이 값 옆의 플러스/마이너스는 익스트루전 방향을 설정합니다:
* Minus ( - )는 현재 표면에서 시작해 아래로 익스트루전합니다. 
* Plus ( + )는 현재 표면에서 시작해 위로 익스트루전합니다.
* PlusMinus ( ± )는 위/아래를 동일한 양만큼 밀어내어, 익스트루전이 원래 표면에 절반 정도 박히도록 합니다.

#### 매끄러움 {#smoothness}
![](/images/layers_smoothness.webp) 

추출할 영역의 가장자리가 들쭉날쭉하다면, 이 슬라이더는 가장자리를 블러 처리해 더 부드러운 형태로 만들려고 시도합니다. 

#### ![](/icons/height.webp) 엣지 루프(측면) {#edge-loop-side}
![](/images/layers_edgeloop.webp) 

이 섹션은 Closing action이 'Shell'일 때 표시됩니다. 

`Auto Edge-loop (side)`는 쉘 측면에 정사각형 폴리곤을 만들기 위한 엣지 루프 수를 자동으로 계산합니다. 

비활성화하면, `Division` 슬라이더가 측면 분할 수를 설정합니다.

_여기까지가 'More...' 서브메뉴의 끝입니다._

### 고급 {#advanced}
![](/images/layers_advanced.webp)

#### 상위 레이어 디테일 유지 {#keep-top-layers-details}

하위 레이어에 큰 변형을 가하더라도 상위 레이어의 작은 디테일이 유지되도록 합니다.

기본적으로 한 레이어에 작은 주름을 조각한 뒤, 기본 레이어에 큰 변형을 가하면 그 주름은 사라집니다. 이 토글을 활성화하면, 작은 디테일이 항상 아래의 큰 변형 위에 떠 있도록 유지됩니다. 다음 영상에서, 기본 설정에서는 주름 디테일이 사라지지만, 'keep top layers details'를 활성화하면 계속 보이는 것을 확인할 수 있습니다:

![](/videos/layers_details.mp4)


#### UI: 목록 확장 {#ui-expand-list}

기본 레이어 메뉴에서는 레이어 가시성과 레이어 불투명도만 토글할 수 있습니다. 이 옵션을 활성화하면 모든 레이어에 대한 전체 컨트롤이 확장되어 표시됩니다.

![](/images/layers_expand.webp)

#### 변형 동기화 {#sync-transform}

활성화하면, 선택되지 않은 모든 레이어가 변형(회전, 스케일, 스큐)에 따라 함께 조정됩니다. 

다른 레이어를 현재 적용 중인 새 변형 없이 사용하려면 이 옵션을 비활성화하세요.

`Auto`로 설정하면, 보이는 레이어만 조정됩니다.
