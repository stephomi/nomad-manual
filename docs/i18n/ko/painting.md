# ![](/icons/paint.webp) 페인팅  

페인트 스트로크의 색상, 거칠기, 금속성을 제어하고, 페인트 속성의 전체 채우기(flood fill), 그리고 페인트 도구가 레이어, 마스크, 숨겨진 선택과 어떻게 상호작용하는지 설정합니다.

![](/images/paint_overview.webp)  

## 개요

Nomad는 PBR 버텍스 페인팅을 사용합니다. 이게 무슨 뜻일까요?

### PBR
PBR(Physically Based Rendering, 물리 기반 렌더링)은 영화, TV, 게임, 모바일에서 널리 쓰이는 컴퓨터 그래픽 기법입니다. 조명을 물리적 특성에 기반해 정의하고, 표면을 색상, 거칠기, 금속성으로 정의함으로써 매우 다양한 포토리얼한 룩을 구현할 수 있습니다.

### 버텍스 페인팅

버텍스 페인팅이란 페인트 정보가 텍스처가 아니라 모델의 버텍스(정점)에 저장된다는 뜻입니다. Nomad는 수십만, 많게는 수백만 개의 버텍스를 가진 모델을 처리할 수 있기 때문에, 매우 디테일한 표면 페인팅이 가능합니다. 조각할 수 있는 디테일이라면, 그 디테일을 그대로 페인팅할 수도 있습니다.

이는 또한 Nomad에서의 페인팅은 다른 3D 애플리케이션에서 종종 느리고 기술적인 과정인 UV 맵핑을 필요로 하지 않는다는 의미이기도 합니다. 많은 다른 3D 애플리케이션은 Nomad만큼 높은 버텍스 수를 지원하지 못하지만, Nomad에는 좋은 텍스처 베이킹 및 디시메이션 도구가 함께 제공됩니다.

### 텍스처링

Nomad는 텍스처를 지원하지만, 이는 가져온 모델에 이미 존재해야 하거나, 버텍스 페인팅을 텍스처로 베이킹해서 만들어야 합니다. 

텍스처는 단순히 이미지이지만, 3D 문맥에서는 보통 오브젝트에 할당된 이미지를 의미합니다.
이미지를 모델에 감싸기 위해서는, 모델에 텍스처 좌표(UV)가 필요합니다.

Nomad는 [자동으로 UV를 계산](topology.md#uv-unwrap)할 수 있지만, 전체적인 품질을 세밀하게 제어할 수는 없습니다.

::: tip
워크플로 예시:
- Nomad에서 조각한 뒤 [UV 언랩](topology.md#uv-unwrap)
- 이미 Nomad에서 페인팅을 시작했다면 [버텍스 페인팅을 텍스처로 전송](topology.md#bake-vertex-colors-to-texture)
- Procreate로 내보내기
- Procreate에서 텍스처 작업
- 렌더링을 위해 다시 Nomad로 가져오기
:::

이제 개요는 끝났으니, 페인팅 메뉴의 각 섹션을 살펴보겠습니다:


## 스트로크 페인팅
![](/images/paint_intensity.webp)  

이 도구에 대해 페인팅을 활성화합니다. 동시에 조각과 페인팅을 해야 할 때 유용합니다.

페인팅이 주 기능인 도구(예: Paint, Smudge, Mask)에는 이 체크박스가 없습니다.

### 페인트 강도

기본 도구 강도와는 다른 강도를 사용할 수 있게 해주는 슬라이더입니다.

`Alpha`, `Falloff`, `Randomize` 체크박스는 해당 기능이 페인팅에 영향을 줄지 여부를 결정합니다. 예를 들어 Clay 도구에 랜덤화를 켜 두었더라도, 색상은 랜덤화되지 않게 할 수 있습니다.


## 머티리얼
![](/images/paint_material.webp) 

첫 번째 아이콘은 머티리얼 미리보기 형태입니다. 3D 머티리얼 미리보기 위를 드래그하면 회전합니다. 

두 번째 아이콘은 선택된 알파와 폴오프 옵션으로 그려질 페인트 스트로크의 미리보기입니다.

Material 제목 옆의 미리보기 버튼을 누르면 none, Material, Triplanar 사이를 전환할 수 있습니다. 이는 페인트 속성을 인터랙티브하게 변경할 때 어떤 일이 일어날지를 결정합니다:

* `None` - 속성을 조정할 때 모델에 미리보기가 표시되지 않습니다.
* `Material` - 속성을 조정할 때 오브젝트에 머티리얼 값이 미리보기로 표시됩니다. 텍스처를 사용하고 모델에 UV가 있다면 UV가 사용됩니다.
* `Triplanar` - 머티리얼이 트라이플래너 프로젝션으로 미리보기됩니다. 

스포이트(eyedropper)는 씬 안의 오브젝트에서 모든 속성을 샘플링하는 데 사용할 수 있습니다.

## 머티리얼 프리셋
3D 미리보기 형태를 탭하면 머티리얼 프리셋 메뉴가 열리며, 이를 복제(clone)해서 자신만의 프리셋을 정의할 수 있습니다.

![](/images/paint_presets.webp) 

`Embed Textures`와 `Alpha` 토글을 켜면, 이 머티리얼에서 사용하는 텍스처가 프리셋 안에 함께 저장됩니다. 이에 대해서는 아래에서 더 설명합니다.

## PBR 슬라이더
![](/images/paint_sliders.webp) 

[PBR](shading.md#pbr) 페인팅은 4개의 채널을 사용합니다:
- `Color` 페인팅될 색상입니다. 스포이트를 사용해 모델의 다른 부분이나 레퍼런스 이미지에서 색상을 선택할 수 있습니다.
- `Roughness` 표면이 얼마나 "거칠거나" "매끄러운지"를 나타냅니다. 거칠기 값이 낮을수록 반사가 날카롭게 보입니다.
- `Metalness` 표면이 금속인지 아닌지를 나타냅니다. 대부분의 경우 값은 0% 또는 100%여야 하며, 그 사이 값은 예외적인 경우에만 사용해야 합니다.
- `Opacity` 머티리얼을 얼마나 투명하게 볼 수 있는지를 나타냅니다. 엄밀히 말하면 PBR 스펙의 일부는 아니지만, 많은 상황에서 유용합니다. 1은 완전 불투명, 0은 완전 투명입니다. 불투명도와 굴절은 다른 개념이며, Nomad에서 굴절은 굴절 머티리얼로 처리됩니다. 

머티리얼 프리셋을 선택하면, 3개의 채널이 동시에 페인팅됩니다(불투명도는 의도적으로 제외되는 경우가 많습니다). 즉, 단순히 ‘빨간색’을 칠하는 대신 ‘거친 빨간 금속’이나 ‘매끄러운 흰색 플라스틱’을 칠할 수 있습니다.

사각형은 텍스처 슬롯으로, 클릭하면 해당 속성에 단색 값 대신 텍스처를 사용할 수 있습니다.

슬라이더 옆의 브러시 아이콘은 그 속성을 오브젝트 전체에 채우기(flood fill)합니다.

체크박스는 해당 속성을 활성/비활성화합니다. 예를 들어 색상만 칠하거나, 거칠기와 불투명도만 칠하도록 설정할 수 있습니다. 

아래는 서로 다른 거칠기와 금속성 조합의 예시입니다:

|                | Metalness 0%                      | Metalness 100%               |
| :------------: | :-------------------------------: | :--------------------------: |
| Roughness 0%   | ![](/images/dielectric_r0.webp)   | ![](/images/metal_r0.webp)   |
| Roughness 50%  | ![](/images/dielectric_r50.webp)  | ![](/images/metal_r50.webp)  |
| Roughness 100% | ![](/images/dielectric_r100.webp) | ![](/images/metal_r100.webp) |

::: warning
[Matcap 렌더링](shading.md#matcap) 모드에서는 색상만 지원되며, 금속성과 거칠기는 무시됩니다.
:::

::: tip
PBR 페인팅에 텍스처를 사용할 때는 `Stamp` 도구로 바꾸거나, 스트로크 메뉴에서 dot 이외의 모드를 사용해 텍스처가 번지지 않도록 하는 것이 유용한 경우가 많습니다.

![](/videos/paint_color_texture.mp4)  
:::

::: tip
저폴리 오브젝트에 금속 표면을 페인팅할 때는, [전역](settings.md#smooth-shading) 또는 [오브젝트별](material.md#smooth-shading)로 `Smooth Shading`을 켜는 것을 고려해 볼 수 있습니다.
:::

## 전체 페인트(Paint all)

![](/images/paint_paint_all.webp)

현재 머티리얼을 오브젝트에 적용합니다. 일반 모드에서는 ‘Paint All’을 사용하고, 트라이플래너 프로젝션으로도 적용할 수 있습니다.

색상/금속성/거칠기/불투명도 슬라이더 옆의 체크박스 설정이 그대로 적용되며, 비활성화된 속성은 채워지지 않습니다.

추가 버튼들은 전체 페인트가 어떻게 더 영향을 받을지 제어합니다:

| Icon                        | Description                                   |
| :-------------------------: | :-------------------------------------------: |
| ![](/icons/tool_mask.webp) | 마스크된 영역은 영향을 받지 않습니다.               |
| ![](/icons/tool_hide.webp) | 숨겨진 영역은 영향을 받지 않습니다.               |
| ![](/icons/opacity.webp)   | 위의 도구 페인팅 강도 값을 사용합니다.           |
| ![](/icons/layer.webp)     | 레이어에서 아직 칠해지지 않은 영역은 영향을 받지 않습니다. |
| ![](/icons/triplanar.webp) | 트라이플래너 설정 표시                         |
| ![](/icons/cog.webp)       | 트라이플래너 설정 열기                         |

### 트라이플래너 설정
![](/images/paint_triplanar_settings.webp)

[머티리얼 메뉴의 트라이플래너 설정](material.md#triplanar)과 비슷하게, 프로젝션의 블렌딩, 타일링, 오프셋을 제어할 수 있습니다. 

이 메뉴 상단의 미리보기 체크박스를 사용해 값을 조정하는 동안 지속적인 미리보기를 활성화할 수 있습니다.

## 전역 머티리얼(Global material)
이 옵션을 활성화하면, 선택된 머티리얼이 다른 도구와 동일하게 공유됩니다. 이때 거칠기, 금속성, 색상 설정만 고려된다는 점에 유의하세요.
