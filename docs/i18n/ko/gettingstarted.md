# 시작하기 {#getting-started}

## Nomad에 오신 것을 환영합니다! {#welcome-to-nomad}

Nomad는 여러 기기에서 동작하는 3D 스컬핑 앱으로, 압력 감지 스타일러스를 사용하는 태블릿(예: Apple iPad + Pencil, Samsung Galaxy Tab + 스타일러스)에서 가장 잘 작동합니다.

ZBrush, Blender 같은 데스크톱 스컬핑 앱에서 영감을 받았으며, 풍부한 기능을 유지하면서도 이해하기 쉬운 UI에 중점을 두고 있습니다. 이미 3D 스컬핑 앱을 사용해 본 적이 있다면, Nomad는 매우 익숙하게 느껴질 것입니다.

3D 스컬핑이 처음이라면, 몇 가지 기본 개념을 알아 두는 것이 좋습니다.

::: tip 동영상이 더 좋나요?
YouTube에는 이제 초보자용 튜토리얼 영상이 아주 많이 있습니다. 다음은 훌륭한 링크들입니다:

* [Nomad Sculpt Crash Course by Dave Reed](https://www.youtube.com/watch?v=69m86ICy2Q4)
* [Nomad Sculpt Beginner tutorial by Small Robot Studio](https://www.youtube.com/watch?v=J644wXoTiww)
* [NOMAD FOR BEGINNERS series by SouthernGFX](https://www.youtube.com/watch?v=bEh0WxRoOmg)

이 제작자들의 메인 채널도 꼭 확인해 보세요. 자주 새로운 튜토리얼을 올립니다.
:::

## 첫 번째 스컬프 {#your-first-sculpt}

Nomad를 처음 실행하면 화면에 구체가 하나 보입니다. 스타일러스를 구체 위에서 드래그하면 바로 스컬핑을 시작할 수 있습니다. 대칭(Symmetry)은 기본적으로 활성화되어 있어 스컬핑이 더 쉬워집니다.

![](/videos/gettingstarted_01.mp4)

브러시 크기를 키우거나 줄이려면, 왼쪽에 있는 반경(radius) 슬라이더를 사용합니다.

![](/videos/gettingstarted_02.mp4)

브러시 강도를 세게 또는 약하게 하려면, 왼쪽에 있는 강도(intensity) 슬라이더를 사용합니다.

![](/videos/gettingstarted_03.mp4)

기본 도구는 `Clay tool`이며, 표면을 더하는(add) 역할을 합니다. 표면을 깎아내고 싶다면, 왼쪽의 `Sub` 버튼을 탭합니다. 다시 표면을 더하는 모드로 돌아가려면 Sub 버튼을 다시 탭합니다.

![](/videos/gettingstarted_04.mp4)

표면을 부드럽게(smooth) 하려면 `Smooth` 버튼을 탭합니다. 일반 스컬핑으로 돌아가려면 Smooth 버튼을 다시 탭합니다.

![](/videos/gettingstarted_05.mp4)

모델 주변을 회전하려면, 모델 바깥의 빈 공간을 드래그합니다.

![](/videos/gettingstarted_06.mp4)

줌(확대/축소)은 두 손가락으로 집기/벌리기 제스처를 사용합니다.

![](/videos/gettingstarted_07.mp4)

카메라를 패닝(pan)하려면, 화면에 두 손가락을 누른 채로 드래그합니다.

![](/videos/gettingstarted_08.mp4)

실수했을 경우, 두 손가락으로 탭하면 실행 취소(undo)가 됩니다. 또는 왼쪽 아래의 실행 취소 버튼을 사용할 수도 있습니다. 

![](/videos/gettingstarted_09.mp4)

::: tip 데스크톱 버전
데스크톱에서는 alt/opt 키를 사용해 카메라를 조작합니다:

* 펜촉으로 빈 공간 드래그 = 카메라 회전
* alt+펜촉 드래그 = 패닝
* alt+펜촉 드래그 후 alt에서 손 떼기 = 줌 (zbrush와 동일)

버튼이 2개 이상 있는 Wacom 펜을 사용하는 경우, Wacom 설정에서 펜촉과 버튼을 다음과 같이 설정하세요:

* 펜촉 = 왼쪽 클릭
* 아래쪽 로커 버튼 = 가운데 클릭
* 위쪽 로커 버튼 = 오른쪽 클릭

이렇게 설정하면, 펜만으로 카메라를 조작할 수 있습니다:

* 위쪽 로커 + 호버 이동 = 카메라 회전
* 아래쪽 로커 + 호버 이동 = 패닝
:::

## 색상 추가 {#add-color}

Nomad에서는 스컬프 표면에 색을 칠할 수 있습니다. 오른쪽 도구 메뉴에서 `Paint` 도구를 찾아 클릭합니다. 그러면 왼쪽 툴바에 색이 있는 구체가 나타납니다. 이것을 클릭하면 색 선택기가 열립니다. 색을 선택한 뒤, 모델 위에 칠해 보세요.

![](/videos/gettingstarted_10.mp4)

지우려면, 왼쪽 툴바의 `Erase` 버튼을 탭한 뒤 표면을 지우면 됩니다. 다시 페인트 모드로 돌아가려면 Erase 버튼을 다시 탭합니다.

![](/videos/gettingstarted_11.mp4)

Clay 브러시의 add/sub 모드, smooth, paint를 사용해서, 간단한 카툰 스타일의 머리를 만들어 보세요:

![](/images/gettingstarted_head1.webp)

## 기타 도구 {#other-tools}

도구 팔레트에는 스컬핑을 도와주는 다양한 도구가 있습니다. 지금까지는 기본 시작 도구인 clay 브러시, smooth, paint를 사용했습니다. smooth는 자주 사용되기 때문에 왼쪽 툴바에 별도의 단축 버튼이 있습니다.

Nomad의 도구들은 move, crease, inflate 같은 스컬핑 관련 도구부터, split, trim처럼 목공이나 금속 가공 도구에 가까운 기능까지 매우 다양한 작업을 할 수 있습니다. 자세한 내용은 [Tools](tools.md) 페이지를 참고하세요.

move, crease, inflate, smooth 도구를 사용해서 머리에 더 많은 디테일을 추가하고, 형태를 바꿔 보세요:

![](/images/gettingstarted_head2.webp)

이제 Nomad의 기본을 익혔으니, 나머지 인터페이스를 살펴보겠습니다.

## 인터페이스 {#interface}

![](/images/interface_overview1.webp)

* `Top menus` - Nomad의 대부분 기능에 접근하는 메뉴입니다. 왼쪽 상단 메뉴는 주로 씬과 오브젝트 관련 기능을, 오른쪽 상단 메뉴는 도구 관련 기능을 다룹니다. 화면이 작은 기기에서는 공간을 절약하기 위해 이 메뉴들이 하나로 접힐 수 있습니다.
* `Stats` - 씬 정보, 현재 오브젝트 정보, 마스크 상태, 메모리 사용량 등을 표시합니다.
* `Nav Cube` - 현재 스컬프를 어느 방향에서 보고 있는지 보여 주는 보조 도구이자, 다양한 뷰로 바로 이동할 수 있는 단축 기능입니다. 큐브를 탭하면 해당 면 방향으로 뷰가 이동합니다. 큐브를 드래그하면 회전합니다. 큐브 옆의 작은 아이콘을 탭하면 현재 오브젝트를 화면에 맞추거나, 기본 홈 뷰로 리셋할 수 있습니다.
* `Toolbox` - Nomad의 도구들이 이 스크롤 가능한 영역에 있습니다.
* `Left toolbar` - 대부분 도구에서 사용하는 반경과 강도 슬라이더, 도구별 상황에 맞는 버튼, 그리고 대칭, 도구의 alt/sub 모드, 마스킹, 스무딩, 기즈모, 페인트 옵션에 대한 단축 버튼이 있습니다.
* `Bottom toolbar` - 자주 사용하는 기능에 대한 단축 버튼들로, 아래에서 설명합니다.

::: tip 왼손잡이인가요?
모든 툴바의 위치와 순서를 좌우 반전할 수 있습니다. [Mirror top bar](interface.md#mirror-top-bar) 및 관련 옵션을 참고하세요.
:::

## 하단 툴바 {#bottom-toolbar}

![](/images/interface_bottom_toolbar.webp)

* `Undo` - 마지막 작업을 되돌립니다.
* `Redo` - 마지막 실행 취소를 다시 적용합니다.
* `History` - [History](history.md) 메뉴에서 설명하는 히스토리 옵션에 접근합니다.
* `Solo` - 현재 오브젝트만 보이게 할지, 모든 오브젝트를 보이게 할지 토글합니다.
* `X-Ray` - 다른 모든 오브젝트를 X-ray 모드로 렌더링하고, 현재 오브젝트는 실체감 있게 표시합니다. 길게 누르거나 위로 스와이프하면 X-ray 모드의 색상과 불투명도를 설정할 수 있습니다.
* `Voxel` - [Voxel Remesher](topology.md#voxel-remesher)의 복셀 리메시 버튼에 대한 단축키입니다. 길게 누르거나 위로 스와이프하면 복셀 리메시 품질을 설정하는 단축 옵션이 나타납니다.
* `Grid` - 그리드 표시를 토글합니다. 길게 누르거나 위로 스와이프하면 그리드 옵션이 나타납니다.
* `Wire` - 와이어프레임 오버레이를 토글합니다. 길게 누르거나 위로 스와이프하면 와이어프레임 옵션이 나타납니다.
* `Inspect` - 현재 메시에 대한 추가 데이터를 표시할지 토글합니다. 기본적으로 UV를 표시하지만, 길게 누르거나 위로 스와이프하면 다른 속성이 존재할 경우 이를 검사할 수 있으며, 배경에 표시할지 메시에 표시할지도 선택할 수 있습니다.

## 다음 단계 {#next-steps}

다음에 무엇을 배울지는 전적으로 당신과, 당신이 흥미를 느끼는 것에 달려 있습니다! 다음은 몇 가지 제안입니다:

스컬핑 도구에 대해 더 배우고 싶나요? [Tools](tools.md) 섹션으로 이동하세요.

모델을 내보내고 싶나요? 또는 스컬핑할 모델을 가져오고 싶나요? 스컬프의 이미지를 만들고 싶나요? [Files](files.md) 섹션으로 이동하세요.

스컬프의 디테일을 제어하는 방법을 더 알고 싶나요? [Topology](topology.md) 섹션으로 이동해 멀티레즈(multires)와 디시메이션(decimation)에 대해 배우세요.

더 많은 오브젝트를 다루고 싶나요? 단순한 오브젝트와 프리미티브를 결합해 더 큰 씬을 만들고 싶나요? [Scene](scene.md) 섹션으로 이동하세요.

Nomad에 대해 *모든 것*을 배우고 싶나요? 좋은 선택입니다! 이 매뉴얼은 Nomad 전반을 다루며, 많은 팁과 트릭을 포함하고 있고, 상단에 훌륭한 검색 기능도 있습니다. 왼쪽의 내비게이션을 사용해 더 알아보세요.

동영상을 선호한다면, Holger Schönischka가 YouTube에 Nomad 관련 팁과 트릭을 대량으로 올려 두었습니다: https://www.youtube.com/@ProcreateFX/videos
