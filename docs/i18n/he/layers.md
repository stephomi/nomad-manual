# ![](/icons/layer.webp) שכבות {#layers}

תפריט זה מכיל את מחסנית השכבות, דרך לאחסן עריכות לאובייקט שלך בצורה לא הרסנית, והרבה דרכים לשלב ולמחזר שכבות.

![](/images/layers_overview.webp) 

## סקירה {#overview}

שכבות ב‑Nomad משמשות למספר מטרות.

מידע צביעה (צבע, מחוספסות, מתכתיות, אטימות) עובד עם שכבות בדומה ליישומי ציור דו־ממדיים. ניתן ליצור שכבה ולצבוע על המודל. אפשר להדליק או לכבות את השכבה, לכוונן את האטימות שלה, לשכפל אותה, לשנות את סדרה במחסנית השכבות, ולשנות את מצב הערבוב שלה.

Nomad משתמשת גם בשכבות לפיסול. שכבה יכולה לאחסן פרטים עדינים כמו קמטים, או שינויים גדולים, מה שמאפשר להן לתפקד כמו blendshapes/shape keys/morph targets ביישומי תלת־ממד אחרים. לדוגמה, אפשר לפסל הבעות פנים שונות על שכבות שונות, ולהתאים את מחווני השכבות כדי לשלב אותן להבעות חדשות.

במקרה זה השינויים המאוחסנים בשכבה הם תוספתיים בלבד, וסדר השכבות לא משנה כמו שהוא משנה בצביעה.

ניתן למחוק שכבות באופן חלקי באמצעות כלי [Delete Layer](tools.md#delete-layer), וניתן גם להשתמש בשכבות כדי ליצור מסכות על בסיס המידע השונה המאוחסן בשכבה.

![](/videos/layer.mp4)

::: tip
בניגוד לרוב תוכנות הפיסול, שינוי הטופולוגיה של רשת לא ימחק את השכבות. ניתן להשתמש ב‑[Voxel Remesher](topology.md#voxel-remesher), ב‑[Multiresolution](topology.md#multires) או בכלי [Trim](tools.md#trim)/[Split](tools.md#split), אך שים לב שכאשר משתמשים ב‑[Voxel Remesher](topology.md#voxel-remesher), איכות השכבה תיפגע.
:::

::: tip
אם משתמשים בשכבות עבור blendshapes/morph targets, קיימת פונקציונליות שכבות נוספת ב‑[תפריט הסצנה](scene.md#object-menu). ניתן להפריד שכבות לאובייקטים, ולשלב אובייקטים תואמים לשכבות.
:::
----

### עוד... {#more}

כפתור 'More...' יציג אפשרויות נוספות עבור השכבה הנוכחית:

![](/images/layers_more.webp) 

#### גורמי ערוץ {#channel-factors}

בקרים אלה מאפשרים לך לסקל את כמות הפיסול/צבע/מחוספסות/מתכתיות/אטימות עבור השכבה. ערכים אלה מוכפלים במחליק פקטור השכבה, כך למשל אם עוצמת השכבה היא 1, אבל מקדם ערוץ הצבע הוא 0.5, אז הצבע שיוצג יהיה בעוצמה 0.5.

`Offset` שולט בעוצמת הפיסול של השכבה. בעוד שצבע/מחוספסות/מתכתיות מוגבלים בין 0 ל‑1, offset יכול להיות כל ערך, חיובי או שלילי. 

::: tip
ניתן להשתמש ב‑Offset כדי להפוך שכבת בליטות לשכבת שקעים, או חיוך לקמיטת מצח:
![](/videos/layer_happysad.mp4)


שכבה סימטרית יכולה להיות משוכפלת ואז מחולקת לצורות שמאל/ימין עם del layer:
![](/videos/layer_leftright.mp4)

שכבות עם מקדמי offset שליליים יכולות להיאפות לשכבות ריקות כדי ליצור צורות חיוביות חדשות.

שכבות עם מקדמי offset חיוביים מעל 1 יכולות לשמש לניסויים ב‑blendshapes קיצוניים יותר.
:::

::: warning
כרגע שכבות חולקות ערוץ אטימות יחיד לכל 3 הערוצים (צבע/מתכתיות/מחוספסות).
אם ממזגים מספר שכבות עם עוצמה פר‑ערוץ שאינן בעוצמה מלאה, ייתכן שהתוצאה הסופית תיראה שונה.

אולי בעתיד לכל ערוץ יהיה ערוץ אלפא משלו כדי להסיר מגבלה זו.
:::


#### ![](/icons/tool_mask.webp) מסיכה {#mask}
כפתור המסכה שליד כל מחוון ייצור מסכה מהערוץ הזה. בדומה לשימוש בשכבות ליצירת בחירות ביישומי ציור, זה מאפשר לך לעשות שימוש חוזר בעבודה שביצעת בשכבה עבור פעולות אחרות.

#### ![](/icons/preview.webp) תצוגה מקדימה {#preview}
![](/images/layers_preview.webp) 

כאשר מופעל, יציג בתצוגה מקדימה את הגדרות ה‑extract עבור שכבה זו (ראה את הסעיף הבא).

כאשר מצב xray מופעל, רק הצורה המופקת תהיה אטומה, שאר הצורה תהפוך לשקופה, שימושי אם אתה משתמש בגבהי החילוץ השליליים.

#### החלק החוצה {#extract}
![](/images/layers_extract.webp) 

![](/videos/layer_shell.mp4)

הכפתור `Extract` ישכפל את התוכן של השכבה לאובייקט חדש, בדרך כלל עם עובי מוגדר על ידי המשתמש שנקבע בסעיף הבא.

`Closing action` קובע כיצד מתבצע השכפול:

* None - פשוט לחלץ את החלק, לא לנסות ליצור צדדים או למלא חורים.
* Fill - החור מתמלא ומוחלק עם משולשים. אין להשתמש באפשרות זו עבור משטחים שטוחים.
* Shell - סגירת הצורה המופקת עם ערך העובי ואפשרויות הכיוון.
* Layer - חילוץ ההפרש של השכבה.

#### ![](/icons/height.webp) עובי {#thickness}
![](/images/layers_thickness.webp) 

עומק האקסטרוזיה של ה‑shell. ערכים חיוביים גדלים מחוץ לפני השטח, ערכים שליליים גדלים לתוך פני השטח.

הפלוס/מינוס שליד ערך זה יקבע את כיוון האקסטרוזיה:
* מינוס ( - ) יתחיל מהמשטח הנוכחי ויחלץ כלפי מטה. 
* פלוס ( + ) יתחיל מהמשטח הנוכחי ויחלץ כלפי מעלה.
* PlusMinus ( ± ) ידחוף את החלק העליון והתחתון של האקסטרוזיה החוצה במידה שווה, כך שהיא תהיה שקועה למחצה במשטח המקורי.

#### החלקה {#smoothness}
![](/images/layers_smoothness.webp) 

אם הקצוות של האזור שיופק משוננים, מחוון זה ינסה לטשטש את הקצה לצורה חלקה יותר. 

#### ![](/icons/height.webp) לולאת קצה (צד) {#edge-loop-side}
![](/images/layers_edgeloop.webp) 

סעיף זה גלוי כאשר פעולת הסגירה היא 'Shell'. 

`Auto Edge-loop (side)` תחשב את מספר לולאות הקצה לאורך צדדי ה‑shell כדי ליצור פוליגונים ריבועיים. 

אם מבוטל, מחוון `Division` יקבע את מספר החלוקות בצדדים.

_זהו סוף תת־תפריט 'More...'._

### מתקדם {#advanced}
![](/images/layers_advanced.webp)

#### שמור על פרטי השכבות העליונות {#keep-top-layers-details}

מבטיח שפרטים קטנים בשכבות גבוהות יישארו נראים כאשר מבצעים שינויים גדולים בשכבות נמוכות.

כברירת מחדל, אם אתה מפסל קמטים קטנים על שכבה, ואז הולך ועושה שינויים גדולים בשכבת הבסיס, הקמטים יאבדו. הפעלת מתג זה תאפשר לפרטים הקטנים הללו תמיד "לצוף" מעל השינויים הגדולים שמתחת. בסרטון הבא, ראה כיצד פרטי הקמטים מוסרים כברירת מחדל, אך נשארים נראים כאשר 'keep top layers details' מופעל:

![](/videos/layers_details.mp4)


#### ממשק: הרחב רשימה {#ui-expand-list}

תפריט השכבות המוגדר כברירת מחדל מאפשר לך להחליף נראות שכבות ואטימות שכבה. הפעלת אפשרות זו מרחיבה את כל הבקרים עבור כל שכבה.

![](/images/layers_expand.webp)

#### סנכרן טרנספורמציה {#sync-transform}

כאשר מופעל, כל השכבות שלא נבחרו יתוקנו בהתאם לסיבוב, קנה המידה וההטיה של הטרנספורם. 

בטל אפשרות זו אם השכבות האחרות מיועדות לשימוש ללא הטרנספורם החדש שאתה מחיל.

כאשר מוגדר על `Auto`, רק השכבות הגלויות יתוקנו.
